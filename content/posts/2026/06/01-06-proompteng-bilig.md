## Action Summary

Bilig is a GitHub Action and library designed to manage and validate Excel workbook logic programmatically. It identifies and resolves stale cached formula values in `.xlsx` files, ensuring accurate calculations and preventing errors when these files are processed in CI pipelines or Node.js applications. Additionally, it provides tools for editing, recalculating, and managing workbook data via JSON, making it a robust solution for automating spreadsheet-related workflows without relying on external tools like Excel or Google Sheets.
