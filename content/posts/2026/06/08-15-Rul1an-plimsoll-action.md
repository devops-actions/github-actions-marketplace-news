## What's Changed

First release of the Plimsoll release-review GitHub Action.

Diff two capability surfaces (previous vs new release), post a sticky PR review, emit SARIF to code scanning, and gate the PR (`fail-on: pending | blocked | never`). Free and complete for one repo, no account, no third-party action dependencies. The hosted layer (cross-repo audit ledger, fleet view, governed approvals, anchored attestations) is behind an optional `license` token and is in preview.

Usage: see README. Pin to a commit SHA in production.
