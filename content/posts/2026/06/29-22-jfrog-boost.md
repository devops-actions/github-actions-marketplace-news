## What's Changed

## Features
- Added filters for more tools — binary/ELF inspection, apt/dpkg, valgrind, and Rscript — so their output is compressed automatically.
- Broadened the pip filter to strip more install noise (output now ~80% smaller).
- Added stdin-pipe filters for ccbuild, ls, and diff.

- Refreshed the Boost website and redesigned the capabilities section as a tmux-style workspace.
