## What's Changed

Generate a disposable email inbox for CI testing.

No Docker. No SMTP config. No signup required.

- Instant inbox generation (no network request)
- Works in any CI environment
- Free sandbox mode or Workspace API key
- Compatible with Playwright, Cypress, Jest

Usage:
  - name: Generate test inbox
    id: inbox
    uses: zerodrop-dev/create-inbox@v1
