## Action Summary

The Soundcheck Security Review GitHub Action automates security checks on your repository by conducting a severity-ranked analysis of code changes for every pull request or scheduled workflow. It identifies vulnerabilities against standards like OWASP Web Top 10, API Security, and LLM Top 10, and optionally applies fixes directly to the codebase. Key capabilities include commenting detailed findings on pull requests, committing autofixes, and gating merges based on critical or high-severity issues.
