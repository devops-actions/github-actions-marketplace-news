## Action Summary

The `otel-genai-fleet-summary-action` GitHub Action automates the analysis of OpenTelemetry GenAI OTLP span exports, identifying governance issues such as unauthorized model usage, missing attributes, and high error rates. It posts a Markdown summary of findings as a pull request comment and can fail builds when high-severity issues are detected, ensuring compliance and improving oversight of large language model (LLM) trace data across fleets. Key capabilities include error rate monitoring, model validation, and integration with other governance tools in the Kinetic Gain Suite.
