## What's Changed

We are thrilled to announce the initial release of **Pull Request AI Reviewer (Gemini)**! 

This GitHub Action acts as your automated pair-programmer, analyzing your pull requests and providing high-quality, actionable feedback directly on your code diffs using Google's Gemini AI.

### Key Features in v1.0.0

* **Intelligent Code Analysis:** Automatically reviews PRs for bugs, performance issues, and security concerns.
* **Inline Comments:** Posts review comments directly on the specific lines of code that need attention, just like a human reviewer.
* **Customizable:** Exclude specific files or lockfiles, choose your preferred language for feedback, and easily select between different Gemini models.
* **Fast & Lightweight:** Built as a pure JavaScript action (no Docker cold-starts) for lightning-fast execution times.

### Getting Started

Check out the [README](https://github.com/leonardosalasd/code-review-bot/blob/main/README.md) for quick setup instructions and workflow examples. All you need is your `GITHUB_TOKEN` and a free Google Gemini API key.
