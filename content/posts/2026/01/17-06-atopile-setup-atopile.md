## Action Summary

This GitHub Action sets up the `atopile` CLI in GitHub Actions workflows, enabling users to run `ato` commands seamlessly. It automates the configuration of the CLI by allowing users to specify a version or derive it from a configuration file and provides support for using pre-built Docker images. This action is designed to simplify the integration and management of `atopile` operations, such as building or syncing multiple packages, within CI/CD pipelines.
