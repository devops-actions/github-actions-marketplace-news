## Action Summary

The GitHub Omni Comment Action consolidates outputs from multiple CI jobs into a single, organized comment on a pull request. It solves the problem of excessive comment noise by grouping test results, deployment previews, performance statistics, and other job outputs into distinct sections within one comment. Key capabilities include customizable section order, support for comments sourced from files, optional collapsing of sections, and compatibility with workflows outside the pull request context.
