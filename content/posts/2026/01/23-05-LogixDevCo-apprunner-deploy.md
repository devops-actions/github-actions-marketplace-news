## Action Summary

The AWS App Runner Deploy Action is a GitHub Action designed to automate application deployments to AWS App Runner from branches, pull requests, or tags. It streamlines the process by automatically building Docker images, integrating with AWS Elastic Container Registry (ECR), and tracking deployments using GitHub Deployments. Additionally, it supports Sentry integration for release tracking and Slack notifications, simplifying deployment workflows for fully managed environments.
