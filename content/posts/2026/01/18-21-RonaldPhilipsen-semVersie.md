## Action Summary

The semVersie GitHub Action automates semantic versioning by calculating version bumps (major/minor/patch) based on conventional-commit style pull requests. It simplifies release workflows by determining the next version number and optionally labeling pull requests with their version impact, streamlining version management in CI/CD processes. Key capabilities include calculating versions, adding PR labels, and supporting customizable metadata for releases.
