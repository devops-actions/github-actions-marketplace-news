## Action Summary

The **Code Periscope GitHub Action** automates the analysis of a repository's commit history to detect risk signals and code quality trends over time. It generates a machine-readable report that can be used in subsequent CI/CD workflow steps, helping teams identify potential issues and maintain code quality. This action is particularly useful for monitoring code health in mono-repos or modular codebases and integrates seamlessly into existing CI pipelines.
