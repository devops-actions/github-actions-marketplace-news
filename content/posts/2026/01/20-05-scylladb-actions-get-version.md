## Action Summary

This GitHub Action and CLI tool, **Get Version**, automates the retrieval and filtering of software versions from sources like DockerHub, Maven Central, and GitHub. It solves the problem of identifying specific versions or patterns of software by providing robust semantic version filtering with support for positional selection and pattern matching. Key capabilities include extracting the latest, stable, or custom-filtered versions and handling API rate-limiting efficiently with retry mechanisms.
