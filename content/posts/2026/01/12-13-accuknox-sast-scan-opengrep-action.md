## Action Summary

Accuknox SAST is a GitHub Action that performs static application security testing (SAST) on repositories to identify security vulnerabilities and uploads the results to AccuKnox's CSPM (Cloud Security Posture Management) panel. It automates security scans, integrates with workflows for continuous monitoring, and supports features like AI-driven analysis and soft failure handling for non-blocking scans.
