## Action Summary

This GitHub Action and CLI tool, **Get Version**, automates the retrieval and filtering of software versions from sources like DockerHub, Maven Central, and GitHub. It offers powerful semantic version filtering with support for pattern matching and positional selection, making it easier to identify specific versions (e.g., latest, stable, or by regex criteria) across repositories. This tool streamlines version management, aiding in tasks such as dependency upgrades or environment consistency checks.
