## Action Summary

The **Qlux Code Reviewer** is a GitHub Action that automates code review processes by leveraging AI-powered analysis to provide detailed feedback directly on pull requests. It identifies issues, generates high-level summaries, makes inline comments with severity levels, and supports interactive responses to user queries. This action streamlines code reviews, improves development workflows, and integrates seamlessly with OpenAI-compatible APIs for intelligent and customizable analysis.
