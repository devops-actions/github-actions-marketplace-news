## Action Summary

This GitHub Action automates the installation of the latest stable or experimental version of the Windows App SDK Runtime on a Windows runner. It is designed to streamline tasks such as running .msix app packages for testing or automation scenarios, eliminating the need for manual runtime installation. This action focuses solely on the runtime, not the SDK.
