## Action Summary

The Tork AI Governance Action automates AI governance checks in CI/CD pipelines by detecting sensitive personal information (PII), validating policies, and enforcing compliance standards. It scans codebases rapidly, provides a governance score, and integrates with GitHub Code Scanning for detailed compliance reporting. This action simplifies the enforcement of data security and regulatory requirements in development workflows.
