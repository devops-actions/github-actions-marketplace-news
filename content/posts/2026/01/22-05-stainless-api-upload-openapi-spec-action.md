## Action Summary

This GitHub Action automates the process of building and previewing updates to Stainless SDKs directly from pull requests, enabling streamlined SDK development and integration workflows. It simplifies authentication using GitHub OIDC or API keys and supports integration with GitHub and GitLab CI pipelines. Key capabilities include generating preview builds for SDKs, commenting build results on pull requests, and triggering merge builds upon pull request completion.
