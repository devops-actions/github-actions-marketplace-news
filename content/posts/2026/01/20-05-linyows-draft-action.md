## Action Summary

The `Draft Action` GitHub Action automates the generation and updating of index files for markdown-based templates, such as ADRs and Design Docs. It simplifies documentation management by keeping index files organized and up-to-date whenever changes are made. Key features include the ability to specify templates, auto-commit updates, and integrate seamlessly into workflows to streamline documentation processes.
