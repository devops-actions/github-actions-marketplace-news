## Action Summary

This GitHub Action, *Review My Code, OpenCode!*, provides LLM-powered, conversational code reviews that mimic the behavior of a real developer. It automates the review process by analyzing code changes, maintaining state across PR discussions, and offering proportional and context-aware feedback. The action aims to solve the shortcomings of traditional tools by being interactive, stateful, and focused, enabling teams to improve code quality while reducing unnecessary noise.
