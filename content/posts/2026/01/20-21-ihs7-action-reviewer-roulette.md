## Action Summary

Reviewer Roulette is a GitHub Action designed to automate the assignment of a specified number of random reviewers to pull requests. It helps streamline the review process by dynamically selecting active users as reviewers, reducing manual effort. Key features include the ability to exclude specific users, set maximum reviewer limits, and integrate seamlessly with workflows like pull request creation.
