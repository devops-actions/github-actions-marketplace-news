## Action Summary

The Build Number (Abacus) GitHub Action automates the generation of **monotonic build numbers** with independent counters for multiple projects or keys. It eliminates manual tracking by ensuring build numbers always increment by 1 and supports advanced operations like setting, updating, resetting, or deleting counters. This action is ideal for managing consistent versioning across complex CI workflows.
