## Action Summary

SARIF Courier is a GitHub Action that processes SARIF (Static Analysis Results Interchange Format) reports and posts a formatted summary as comments on pull requests, providing immediate visibility into security and static analysis findings without requiring GitHub Advanced Security (GHAS). It enables seamless integration with any SARIF-compliant tools, ensuring developers can review issues directly in their PR workflow, promoting faster feedback and resolution. This action simplifies security reporting and eliminates the need for proprietary or paid GitHub features.
