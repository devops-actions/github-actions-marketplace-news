## Action Summary

Repo Policy Gate is a GitHub Action designed to enforce repository and dependency policies on pull requests by performing fast, local checks within CI workflows. It automates the validation of pull request title conventions, the presence of required repository files, and the restriction of disallowed dependencies (including semver ranges), providing clear explanations for any violations. This action helps streamline compliance and code quality enforcement without requiring external dashboards or accounts.
