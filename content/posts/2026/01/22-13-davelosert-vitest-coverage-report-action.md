## Action Summary

The `vitest-coverage-report-action` is a GitHub Action that automates the generation and reporting of test coverage results from Vitest. It provides a high-level coverage summary and detailed, file-specific reports, which can be displayed as part of the GitHub Actions step-summary and as comments on pull requests. This action streamlines the process of reviewing code coverage, helping developers identify uncovered lines and maintain test quality efficiently.
