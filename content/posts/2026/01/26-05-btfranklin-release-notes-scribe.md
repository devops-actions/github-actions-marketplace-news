## Action Summary

The "Release Notes Scribe" GitHub Action automates the generation of draft release notes for GitHub using commit messages and code diffs between tags, enhanced by AI-powered summarization via the OpenAI Responses API. It streamlines the process of creating structured and detailed release notes, reducing manual effort and improving consistency. Key capabilities include customizable AI models, integration with GitHub releases, and support for handling large or complex change logs efficiently.
