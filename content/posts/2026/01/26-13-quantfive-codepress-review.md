## Action Summary

CodePress Review is a GitHub Action that automates inline code reviews on pull requests using AI-powered large language models (LLMs) from various providers. It provides intelligent, context-aware feedback, generates structured pull request descriptions, and can approve, request changes, or comment on PRs based on customizable review guidelines. This tool streamlines the code review process, reduces manual effort, and ensures high-quality, actionable feedback for developers.
