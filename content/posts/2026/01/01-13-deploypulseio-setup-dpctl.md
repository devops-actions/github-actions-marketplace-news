## Action Summary

This GitHub Action simplifies the deployment and management of DeployPulse applications by automating the installation and usage of the `dpctl` tool within GitHub workflows. It allows developers to release updates for iOS and Android apps, manage deployment environments, and streamline CI/CD processes. By integrating DeployPulse directly into GitHub Actions, it reduces manual effort and ensures consistent application delivery.

## Release notes

Initial release v1. We are proud to announce the `dpctl` GitHub action step which will automatically download and install the `dpctl` CLI tool for easy CodePush update deployments to the DeployPulse platform.
