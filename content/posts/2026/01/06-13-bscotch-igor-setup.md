## Action Summary

The `igor-setup` GitHub Action automates the setup of the Igor executable, enabling users to build GameMaker projects in CI/CD pipelines. It simplifies and streamlines the configuration of runtime environments and platform-specific settings for supported targets (Windows, Android, and iOS). The action provides key capabilities such as managing GameMaker runtime versions, configuring platform preferences, and caching runtime files for efficient project builds.
