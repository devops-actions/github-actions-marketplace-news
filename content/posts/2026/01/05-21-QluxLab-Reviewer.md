## Action Summary

The Qlux Code Reviewer is a GitHub Action that automates code reviews by leveraging AI-powered analysis integrated directly into pull requests. It provides detailed feedback, such as high-level summaries, inline comments, and severity-based issue identification, while allowing interactive discussions through contextual replies to user comments. This action streamlines the code review process, ensures consistent quality checks, and supports multiple AI models and customizable configurations.
