This GitHub Action automates the installation of plugins for the Fish shell using supported plugin managers like Fisher or Oh My Fish. It simplifies plugin management in CI/CD workflows by ensuring the appropriate plugin manager is installed (if not already present) and securely handling plugin installation. This action provides a streamlined way to enhance Fish shell functionality in automated environments.
