## Action Summary

This GitHub Action generates dynamic, animated representations of a user's GitHub contribution graph in five distinct styles, including various shapes, themes, and automated month labels. It automates the creation of visually engaging SVG and GIF assets and organizes them into an output branch, simplifying the process of showcasing and customizing contribution activity. Key features include auto-detection of repository owners, high-quality graphics, and a fully automated gallery for easy preview and sharing.
