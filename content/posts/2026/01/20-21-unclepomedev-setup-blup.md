## Action Summary

This GitHub Action, `setup-blup`, automates the installation of [blup](https://github.com/unclepomedev/blup), a Blender Version Manager, and sets up Blender environments across Linux, Windows, and macOS with caching support. It simplifies the process of managing Blender versions, including stable, daily, and experimental builds, enabling streamlined workflows for developers and CI pipelines. Key capabilities include automated version selection, support for caching downloaded binaries, and compatibility with multiple operating systems.
