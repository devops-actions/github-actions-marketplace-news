## Action Summary

This GitHub Action transforms your GitHub contribution graph into a dynamic and visually appealing animation with five distinct styles, customizable shapes, and automated month labels. It automates the generation of multiple snake-themed SVG and GIF animations, along with a gallery preview in a dedicated branch, eliminating manual effort and enhancing profile customization. Key features include auto-detection of the repository owner, support for various themes and shapes, and integration-ready assets for showcasing on personal profiles or sharing on social media.
