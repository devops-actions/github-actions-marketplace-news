## Action Summary

The **gentesty-evaluation-action** is a GitHub Action designed to automate the process of triggering evaluation runs on the Gentesty platform. It streams real-time logs from the evaluation process into GitHub Actions and provides the final success or failure result directly within the workflow logs. This action simplifies and secures the integration of Gentesty evaluations into CI/CD pipelines, enabling seamless testing and feedback loops.
