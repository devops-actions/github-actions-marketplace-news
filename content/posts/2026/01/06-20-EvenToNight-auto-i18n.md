## Action Summary

The "Auto i18n Translator" GitHub Action automates the translation of JavaScript/TypeScript localization files into multiple languages using Google Translate. It streamlines the process by detecting changes in the source files, translating only modified keys, and preserving custom translations via special markers. Additionally, it auto-commits and pushes updated translations, saving time and ensuring consistent localization across projects.
