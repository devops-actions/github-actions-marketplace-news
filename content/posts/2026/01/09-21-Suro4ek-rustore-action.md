## Action Summary

The RuStore Publish Action is a GitHub Action designed to streamline the process of uploading APK and AAB files to RuStore. It automates tasks such as reusing existing drafts, inheriting metadata (e.g., screenshots and descriptions) from previous releases, and handling upload errors by saving drafts for retry attempts. This action simplifies and accelerates the release management workflow for Android applications on RuStore.
