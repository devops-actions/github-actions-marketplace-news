## Action Summary

The `kusari-ingest` GitHub Action automates the ingestion of software artifacts such as SBOMs, SLSA attestations, and other related files into the Kusari Platform. It simplifies the process of integrating artifact data into a user's Kusari tenant with minimal configuration, enabling efficient artifact tracking and compliance management. Key capabilities include authentication, support for various artifact types, and optional checks for blocked dependencies, streamlining workflows for software supply chain security.
