## Action Summary

The **GV Sanity Check** GitHub Action detects stability drift by comparing metric snapshots in CI pipelines. It automates the process of identifying deviations between current metrics and a defined baseline, helping maintain consistent performance and catch regressions early. This action is particularly useful for ensuring stability in software projects by monitoring key performance indicators over time.
