## Action Summary

The **aqua-installer-cache** GitHub Action is designed to cache the installation directory of [aqua](https://aquaproj.github.io/), a declarative CLI version manager, to improve CI/CD workflow efficiency. By caching aqua packages and configuration files, it reduces redundant downloads, mitigates GitHub API rate limits, and accelerates execution times. The action supports multiple operating systems, automatically detects configuration files, and optimizes caching with unique keys based on OS, architecture, and configuration changes.
