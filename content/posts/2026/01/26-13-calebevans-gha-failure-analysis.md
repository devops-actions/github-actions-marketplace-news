## Action Summary

This GitHub Action automates the analysis of failed workflows by processing logs, correlating failures with code changes, and generating actionable root cause reports. It leverages AI-powered tools for semantic log processing and intelligent failure analysis, offering insights into causes of failures, their relation to PR changes, and providing structured recommendations. This action streamlines troubleshooting, enhances CI/CD efficiency, and reduces manual effort in diagnosing workflow issues.
