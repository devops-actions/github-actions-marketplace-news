## Action Summary

The "Medcrypt Helm SBOM Upload" GitHub Action automates the process of uploading a Software Bill of Materials (SBOM) file to Medcrypt's Helm system for tracking and vulnerability analysis. It streamlines product and version management within Helm by enabling automatic creation of products and versions if they do not already exist. This action simplifies SBOM integration into CI/CD pipelines, enhancing software transparency and security compliance.
