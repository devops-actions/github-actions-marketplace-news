## Action Summary

The Qlux Code Reviewer is a GitHub Action that automates code reviews using AI-powered analysis, integrating directly into pull requests. It provides detailed feedback, including high-level summaries and inline comments, while supporting interactive responses to user queries. This action streamlines code quality checks, reduces manual review effort, and offers configurable features like customizable prompts, file filtering, and severity-based comment filtering.
