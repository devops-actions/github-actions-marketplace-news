## Action Summary

This GitHub Action automatically expands AWS IAM wildcard actions in pull request diffs and posts inline comments detailing each specific action the wildcard matches, with links to AWS documentation. It simplifies code reviews by providing clear visibility into the security implications of wildcard usage, helping reviewers quickly understand changes to IAM policies. This action supports various file types, operates with minimal permissions, and ensures secure, auditable behavior.
