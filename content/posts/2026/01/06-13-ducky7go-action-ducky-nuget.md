## Action Summary

The **Ducky NuGet Mod Publisher** GitHub Action automates the process of packaging and publishing game mods to NuGet servers in compliance with the NuGet Mod Packaging Specification. It simplifies tasks such as generating `.nuspec` files, validating mod metadata, handling file inclusions, and supporting multiple authentication methods (e.g., OIDC or API keys) for secure publishing. This action is designed to streamline mod distribution workflows, eliminate manual packaging efforts, and ensure compliance with NuGet standards.
