## Action Summary

This GitHub Action automates the process of building and previewing Stainless SDKs directly from pull requests, streamlining updates to OpenAPI specifications or Stainless configurations. It supports authentication via GitHub OIDC or API keys, enabling secure integration and reducing manual effort. The action simplifies SDK builds by automatically generating results for pull requests and managing workflows for preview and merge stages.
