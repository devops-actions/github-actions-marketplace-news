## Action Summary

This GitHub Action automates the analysis of failed workflows by processing logs, identifying root causes using AI-powered tools, and correlating failures with recent code changes. It generates structured, actionable reports that include evidence and recommendations, helping developers quickly diagnose and address issues. Key features include semantic log processing, PR context-aware analysis, secret redaction, and support for multiple LLM providers.
