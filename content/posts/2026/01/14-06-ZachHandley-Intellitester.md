## Action Summary

IntelliTester is a GitHub Action that leverages AI, Playwright, and mobile/web emulators to automate the creation and execution of test suites using simple YAML instructions. It simplifies test automation by offering features like debugging with Playwright Inspector, JSON schema-based configuration validation, and automated resource cleanup for databases and backends. This action is designed to streamline testing workflows, reduce manual effort, and ensure a clean testing environment.
