## Action Summary

The "Ducky NuGet Mod Publisher" GitHub Action automates the packaging and publishing of game mods to NuGet servers by parsing mod metadata, validating file structures, and generating compliant `.nuspec` files. It simplifies mod distribution by supporting multi-server publishing, automating file inclusion, and handling authentication via API keys or Trusted Publisher (OIDC). This action is ideal for developers who want to streamline the process of preparing and deploying mods to NuGet repositories while ensuring compliance with the NuGet Mod Packaging Specification.
