## Action Summary

The `copilot-external-doc-updater` is a GitHub Action that automates the process of updating documentation and changelogs in Notion by leveraging GitHub Copilot SDK and Notion MCP Server integration. It simplifies tasks such as creating and appending changelog entries upon pull request merges and optionally synchronizing README.md contents to Notion pages. This action is ideal for maintaining up-to-date documentation and changelogs directly from GitHub workflows, with support for AI-driven updates and manual triggers.
