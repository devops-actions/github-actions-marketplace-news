## Action Summary

This GitHub Action automates the installation of the QHYCCD SDK on Linux, Windows, and macOS systems, supporting both Intel and Apple Silicon architectures. It simplifies the setup process for using the QHYCCD SDK in workflows, with optional caching to optimize performance and minimize redundant downloads. This action is particularly useful for automating CI/CD pipelines involving projects that rely on QHYCCD's SDK.
