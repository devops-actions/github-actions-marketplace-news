## Action Summary

The `distr-create-version-action` GitHub Action automates the creation of new application versions in the Distr platform, supporting Docker and Helm-based applications. It streamlines CI/CD workflows by enabling seamless version management and optional automatic updates to deployments, reducing manual effort and ensuring consistent application versioning. This action is particularly useful for teams leveraging Distr to manage and deploy containerized or Helm applications.
