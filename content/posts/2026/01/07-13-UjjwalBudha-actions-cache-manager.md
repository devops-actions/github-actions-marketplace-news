## Action Summary

The **GitHub Actions Cache Manager** is a tool designed to automate the cleanup of outdated GitHub Actions caches, helping users stay within the 10GB cache storage limit. It selectively removes older caches based on criteria such as age or count while preserving recent ones. This action simplifies cache management, ensuring efficient storage usage and maintaining optimal workflow performance.
