## Action Summary

Anno is a GitHub Action that automates the creation of concise summaries for release notes and pull requests, leveraging large language models (LLMs) like ChatGPT. It generates release summaries of code changes, integrates with Jira for issue tracking, and shares updates via Slack. Additionally, it enhances pull request descriptions by summarizing changes and linking relevant Jira tickets, streamlining communication and improving project management workflows.
