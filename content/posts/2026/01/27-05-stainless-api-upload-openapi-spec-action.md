## Action Summary

This GitHub Action automates the building and previewing of Stainless SDKs directly from GitHub pull requests. It streamlines the process of generating SDKs by triggering builds and providing feedback on updates to OpenAPI specs or Stainless configurations. The action supports authentication via GitHub OIDC or API keys, enhancing security and ease of integration, and also includes limited support for GitLab CI workflows.
