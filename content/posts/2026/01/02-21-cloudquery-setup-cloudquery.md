## Action Summary

The `setup-cloudquery` GitHub Action automates the installation of the CloudQuery CLI on Linux or macOS runners, enabling its use in subsequent workflow steps. It simplifies the process of integrating CloudQuery into CI/CD pipelines by handling the download and setup of the CLI, allowing users to efficiently manage and synchronize cloud infrastructure data. This action is ideal for workflows involving CloudQuery operations, such as syncing data from cloud environments.
