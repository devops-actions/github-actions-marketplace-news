## Action Summary

This GitHub Action facilitates automated deployment of WordPress sites to Ubuntu servers using rsync over SSH. It simplifies the deployment process by providing default configurations tailored for WordPress, including pre-defined file exclusions to protect live installations, and allows customization of source directories, rsync flags, and exclude rules. This action streamlines manual deployment tasks, ensuring efficient and secure updates to remote servers.
