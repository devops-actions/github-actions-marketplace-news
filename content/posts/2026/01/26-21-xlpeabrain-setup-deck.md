## Action Summary

The `setup-deck` GitHub Action automates the installation of `decK`, a tool used for managing Kong declarative configurations, within GitHub Actions workflows. It simplifies the setup process by allowing users to install the latest or a specific version of `decK`, and optionally capture command outputs for use in subsequent workflow steps. This action streamlines tasks related to Kong configuration management in CI/CD pipelines.
