## Action Summary

The **test-summary-action** GitHub Action is designed to summarize JUnit test results from Gradle builds. It automates the collection and reporting of test outcomes (e.g., passed, failed, skipped) across multiple subprojects, presenting the results in a clear table format within the `GITHUB_STEP_SUMMARY`. This action helps developers quickly identify and analyze test statuses, enhancing build transparency and debugging efficiency.
