## Action Summary

**Lite Release** is a GitHub Action that automates key aspects of the software release process, including version bumping, file updates (e.g., release vs. snapshot), Git tagging, and GitHub Release creation. It simplifies and streamlines release management by offering features like customizable version tracking, commit messages, and release notes templating. With its smart bumping capabilities and integration with GitHub CLI, this action reduces manual effort and ensures consistency in versioning and release workflows.
