## Action Summary

Code Sentinel is an AI-powered GitHub Action designed to automate code reviews by focusing on security, architecture, performance, and bug detection. It learns team-specific coding patterns and conventions to provide context-aware, consistent feedback through inline comments and summaries. Supporting multiple LLM providers and offering privacy-first options, it streamlines the code review process and enhances code quality while reducing manual effort.
