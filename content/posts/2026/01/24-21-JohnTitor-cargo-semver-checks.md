## Action Summary

The Cargo Semver Labeler Action automates semantic versioning checks for Rust projects by running `cargo-semver-checks` on pull requests. It determines the required version bump (major, minor, or patch) by comparing the PR changes against the base branch and applies corresponding labels to the pull request. This action streamlines the versioning process in PR workflows, ensuring consistent and accurate semver updates.
