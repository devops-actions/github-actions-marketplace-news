## Action Summary

The Weekly Changelog Action automates the generation of a weekly changelog by summarizing recent repository commits using an AI model. It creates both technical and user-facing summaries, supports multiple languages, and optionally performs deeper analysis of file changes. The action simplifies changelog management by automatically committing updates to a `CHANGELOG.md` file, ensuring consistent and efficient documentation of project changes.
