## Action Summary

LazyPR is an AI-powered GitHub Action designed to automate the creation of high-context pull request summaries, risk assessments, and changelogs based on code changes rather than commit messages. It streamlines code review processes by detecting mismatched commit descriptions, scoring PR risks, and generating precise, developer-friendly insights. This tool helps teams save time, improve documentation quality, and enhance transparency across repositories.
