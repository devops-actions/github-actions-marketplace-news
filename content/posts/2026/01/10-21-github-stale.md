## Action Summary

This GitHub Action, **Close Stale Issues and PRs**, automates the process of managing inactive issues and pull requests by warning about inactivity and then closing them after a specified timeframe. It helps maintainers keep repositories organized and free of outdated or unresolved items by applying customizable rules for identifying and handling stale items. Key features include configurable timeouts, customizable messages, and label-based exemptions for fine-tuned control.
