## Action Summary

This GitHub Action automates the process of identifying dormant developer users in an organization by analyzing activity such as commits, issues, and pull requests across all repositories. It classifies users based on their activity status (active, inactive, or never-active) and generates a downloadable CSV report summarizing the results. This action helps organizations monitor developer engagement and identify inactive contributors efficiently.
