## Action Summary

The **Simple Discord Notify** GitHub Action is a lightweight, zero-dependency tool designed to send customizable notifications to Discord using webhooks. It automates the process of alerting teams about events in a GitHub workflow, such as build statuses or releases, with support for dynamic layouts, status-based color coding, and detailed context. By leveraging `curl` and bash, it provides a fast and reliable alternative to more complex or resource-heavy solutions, ensuring proper error handling without silent failures.
