## Action Summary

The **Trivy PR Decorator** GitHub Action integrates with Trivy vulnerability scans to automate the process of posting clear, formatted comments on pull requests that summarize detected security vulnerabilities. It enhances team collaboration by providing detailed, sortable vulnerability tables, visual severity indicators, and smart comment management to avoid duplicates. This action streamlines security workflows by keeping pull request discussions informed and organized.
