## Action Summary

The **Lint Autofix (Community)** GitHub Action automates the process of running ESLint and Prettier on pull requests to identify and suggest code formatting or linting fixes. It generates a single comment on the pull request containing a diff of suggested changes, without making any commits to the repository. This action helps streamline code review by highlighting formatting issues and potential errors, ensuring code consistency without requiring manual intervention.
