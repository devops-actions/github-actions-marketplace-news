## Action Summary

The GitHub Omni Comment Action consolidates outputs from multiple CI jobs into a single, organized comment on a pull request. It helps reduce comment noise by combining various sections, such as test results or deployment previews, into one structured comment, improving clarity and collaboration in PR discussions. Key capabilities include support for custom section ordering, adding comments from files, and specifying target pull requests.
