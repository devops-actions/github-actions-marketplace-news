## Action Summary

The Works AI GitHub Action automates the synchronization of repository activity with the Works work tracking system. It simplifies work tracking by automatically detecting work IDs from commits or PRs, marking work as complete when PRs are merged, and reporting progress from CI/CD events. Additionally, it can generate AGENTS.md files for automation, streamlining task management and integration with development workflows.
