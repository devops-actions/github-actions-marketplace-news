## Action Summary

The **Workflow Performance Monitor** GitHub Action tracks and analyzes workflow runs by monitoring jobs, steps, and process activities, as well as collecting resource usage metrics like CPU load, memory, network, and disk I/O. It provides detailed insights with trace charts and tables, including step execution times, process details, and success/failure statuses. This action helps automate performance monitoring, optimize workflows, and identify bottlenecks by summarizing results in pull request comments or job summaries.
