## Action Summary

This GitHub Action facilitates integration between GitHub and the OpenText Core Software Delivery Platform (SDP) or Software Delivery Management (SDM) solutions, formerly known as ALM Octane/ValueEdge. It automates the triggering of Model-Based Testing (MBT) test runs from a GitHub repository and synchronizes the results with the OpenText products, streamlining CI/CD workflows. The action simplifies test execution and result management by enabling seamless communication between GitHub and OpenText's testing tools.
