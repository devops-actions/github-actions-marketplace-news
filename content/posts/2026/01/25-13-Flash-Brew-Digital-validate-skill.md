## Action Summary

The "Validate Skill" GitHub Action is designed to validate `SKILL.md` files against the Agent Skills specification to ensure they are compliant, error-free, and compatible with coding agents and tools like the Vercel Skills CLI. It automates the detection of issues such as missing file references, incorrect directory structures, and rule violations, providing detailed outputs of errors and warnings in a CI/CD-friendly format. This action helps developers catch problems early, maintain standardization, and ensure cross-agent compatibility of their skills.
