## Action Summary

The "AllTrue Security Testing for AI Systems" GitHub Action automates security testing for AI models and LLM endpoints by integrating with the AllTrue platform. It identifies vulnerabilities such as prompt injection, data leakage, and harmful content generation, while also scanning models for malicious code and policy violations. Key capabilities include flexible testing scopes, HuggingFace integration, parallel execution, and comprehensive reporting, enabling organizations to enhance the security of their AI infrastructure efficiently.
