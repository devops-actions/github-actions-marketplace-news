## Action Summary

This GitHub Action automates security testing for AI systems by integrating with the AllTrue platform to identify, test, and report vulnerabilities in large language model (LLM) endpoints and AI models. It streamlines tasks such as prompt injection testing, model scanning for malicious code and policy violations, and onboarding models from platforms like HuggingFace. Key capabilities include flexible scoping, parallel execution, automated issue creation, and comprehensive reporting, helping users efficiently secure AI infrastructure.
