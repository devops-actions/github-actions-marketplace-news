## Action Summary

The `action-crosspost` GitHub Action automates the process of sharing content from a sitemap or RSS/Atom feed to multiple social media platforms (e.g., Twitter/X, Mastodon, LinkedIn). It simplifies crossposting by filtering URLs based on criteria like age and patterns, supports message customization with metadata, and offers a dry-run mode for testing before publishing. This action helps streamline content distribution and ensures efficient social media management directly from GitHub workflows.
