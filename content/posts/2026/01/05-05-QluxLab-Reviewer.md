## Action Summary

The **AI Code Reviewer Action** is a GitHub Action designed to automate code reviews for pull requests by leveraging AI models like OpenAI's GPT. It analyzes code changes to provide a high-level summary, inline comments with improvement suggestions, and supports customizable prompts and patterns to tailor reviews. This action streamlines the code review process, enhances code quality, and saves time by automating detailed feedback and identifying potential issues efficiently.
