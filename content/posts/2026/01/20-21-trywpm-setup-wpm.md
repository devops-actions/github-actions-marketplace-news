## Action Summary

The `setup-wpm` GitHub Action automates the installation and setup of the `wpm` package manager in GitHub Actions workflows. It simplifies the process of ensuring the correct version of `wpm` is installed, with support for specifying a version or defaulting to the latest. This action also supports integration with caching to optimize build times and reduce redundant downloads, making workflows more efficient.
