## Action Summary

Qlux Code Reviewer is a GitHub Action that automates code reviews using AI, integrating directly into pull requests to analyze code changes and provide feedback. It streamlines the review process by identifying issues, posting inline comments with severity levels, and supporting interactive discussions through contextual replies. The action is highly configurable, supports multiple AI models, and helps maintain clean, efficient pull request workflows.
