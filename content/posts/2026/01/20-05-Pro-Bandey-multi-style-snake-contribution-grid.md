## Action Summary

This GitHub Action generates dynamic, animated representations of your GitHub contribution graph in five unique styles, incorporating various shapes, themes, and automated month labels. It automates the process of creating high-quality SVG and GIF animations, storing the output in a designated branch, and providing an auto-generated gallery for easy integration into profile READMEs. This action simplifies the task of personalizing and showcasing contribution activity with visually appealing designs.
