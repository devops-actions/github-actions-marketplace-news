## Action Summary

The `ai-eval` GitHub Action uses Claude AI to evaluate input content against a customizable prompt, returning a structured decision with an approval status (`approved`) and reasoning (`reason`). It automates content evaluation tasks, such as reviewing code, pull requests, or configuration files, by providing consistent and detailed feedback based on user-defined criteria. Key features include secure API handling, support for text or file-based input, and automatic input truncation to comply with API limits.
