## Action Summary

The **Telegram GitHub Action** enables users to send text messages to a specified Telegram chat directly from GitHub Actions workflows. It automates notifications for events such as commits, releases, or other repository activities, allowing seamless communication and updates via Telegram. Key features include support for message customization, parsing modes (Markdown/HTML), silent notifications, and the ability to target specific Telegram chats or threads.
