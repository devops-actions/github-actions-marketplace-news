## Action Summary

This GitHub Action leverages OpenCode's language model to provide intelligent, conversational, and context-aware code reviews. It automates the review process by analyzing pull requests, offering proportional suggestions, tracking review state within PR comments, and engaging in interactive discussions to resolve findings. Designed to function like a real developer, it helps streamline code reviews by eliminating shallow or noisy feedback and focusing on meaningful issues such as security vulnerabilities, logic errors, or best practices.
