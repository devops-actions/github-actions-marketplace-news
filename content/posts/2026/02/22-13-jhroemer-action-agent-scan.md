## Action Summary

The **action-agent-scan** GitHub Action is designed to analyze contributors to a pull request and determine if they are likely to be AI agents. It automates the process of running a test (based on the "voight-kampff-test" library) and provides a summary of the contributor, helping repository maintainers identify potential automated contributors. This streamlines monitoring and decision-making for PR reviews.
