## Action Summary

The **GreenIT Analysis GitHub Action** automates the evaluation of web application sustainability by running the GreenIT-Analysis CLI against specified URLs. It generates EcoIndex-based HTML reports, enforces a configurable quality gate to ensure environmental performance standards are met, and uploads the reports as workflow artifacts. This action streamlines the integration of eco-efficiency checks into CI/CD pipelines, ensuring applications adhere to sustainability benchmarks.
