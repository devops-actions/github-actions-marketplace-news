## Action Summary

Anno is a GitHub Action that automates the creation of concise summaries for release changes and pull requests using large language models (LLMs). It generates release summaries that highlight code changes between workflow runs and posts them to Slack, with optional Jira integration for issue tracking. Additionally, it provides brief pull request descriptions, improving collaboration and streamlining communication within development teams.
