## Action Summary

Anno is a GitHub Action that uses large language models (LLMs) to automate the generation of release and pull request summaries. It provides concise release summaries by analyzing code changes and integrates with Slack and Jira for enhanced reporting. Additionally, it simplifies pull request management by automatically adding brief, context-aware summaries, reducing manual effort and improving communication.
