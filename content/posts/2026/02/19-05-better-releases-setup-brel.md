## Action Summary

The `setup-brel` GitHub Action automates the installation of the `brel` CLI tool from GitHub Releases and makes it accessible in the `PATH` for subsequent workflow steps. This action simplifies the setup process for using `brel` by handling version selection, installation, and configuration, enabling seamless integration into CI/CD pipelines. It also provides outputs such as the installed version and binary path for further use in workflows.
