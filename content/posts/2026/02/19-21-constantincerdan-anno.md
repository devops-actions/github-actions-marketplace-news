## Action Summary

Anno is a GitHub Action that leverages large language models (LLMs) to automate the creation of release and pull request summaries. It generates concise release notes summarizing code changes and posts them to Slack, with optional Jira integration for linking related issues. Additionally, it enhances pull requests by generating brief summaries and linking relevant Jira tickets, streamlining release communication and code review processes.
