## Action Summary

This GitHub Action generates and updates pull request comments with a visual summary of Terraform Plan changes, including both plain text and JSON outputs. It automates the process of reviewing infrastructure changes by providing clear, detailed reports directly in pull requests, helping teams identify and assess potential impacts efficiently. Key capabilities include customizable report headers, optional inclusion of plan and diff views, and the ability to remove stale comments from outdated commits.
