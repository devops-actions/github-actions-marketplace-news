## Action Summary

The `@nyaomaru/changelog-bot` GitHub Action automates the generation of polished changelog entries by combining Git history, PR titles, and release notes. It streamlines the release process by creating human-readable changelogs, optionally enhanced with AI-generated summaries, and can automatically open pull requests with updated changelogs and release notes. This tool eliminates manual effort in maintaining changelogs, integrates seamlessly into CI workflows, and provides fallback mechanisms for reliability.
