## Action Summary

The `vercel-seo-audit` GitHub Action is a CLI tool designed to identify and explain SEO issues that may prevent Google from properly indexing Next.js sites hosted on Vercel. It automates the detection of common misconfigurations, such as redirect loops, missing `robots.txt` or `sitemap.xml` files, canonical URL mismatches, and Vercel/Next.js-specific quirks, providing actionable insights to improve site indexing and search engine visibility. Its key capabilities include generating detailed SEO audit reports, customizable crawling options, and support for integration with CI workflows.
