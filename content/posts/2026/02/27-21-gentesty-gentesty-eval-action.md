## Action Summary

The **gentesty-evaluation-action** GitHub Action automates the process of triggering evaluation runs on the Gentesty platform directly from a GitHub workflow. It streams real-time logs from Gentesty into your workflow logs using WebSocket and provides the final test result (success or failure) upon completion. This action simplifies integration with Gentesty, enhancing CI/CD pipelines by enabling secure, automated testing and real-time feedback.
