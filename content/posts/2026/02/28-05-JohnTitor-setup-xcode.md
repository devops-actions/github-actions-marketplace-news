## Action Summary

The `setup-xcode` GitHub Action automates the process of selecting and switching between preinstalled Xcode versions on macOS runners. It supports both GitHub-hosted and self-hosted environments, resolving and configuring the desired Xcode version based on user input. This action simplifies workflows that require specific Xcode versions, ensuring compatibility and reducing manual setup effort.
