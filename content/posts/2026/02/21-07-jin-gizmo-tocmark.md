## Action Summary

TOCmark is a lightweight tool and GitHub Action for generating customizable tables of contents (TOCs) in Markdown files, particularly for GitHub-hosted documentation such as README.md files. It automates the creation and updating of TOCs using AWK, allowing users to control heading selection, display styles, and formatting without requiring additional dependencies. This action simplifies documentation management by ensuring clear, organized navigation within Markdown files.
