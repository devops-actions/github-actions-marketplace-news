## Action Summary

The **Copilot GitHub Action** streamlines project and workflow management by integrating Git-Flow practices with GitHub Projects. It automates tasks such as branch creation from issue labels, linking issues and pull requests to project boards, updating progress status, and generating AI-powered pull request descriptions. This action simplifies collaboration and keeps repositories, tasks, and project boards synchronized, while also providing optional AI assistance for task automation and code fixes.
