## Action Summary

The `setup-beam` GitHub Action automates the setup of an Erlang/OTP environment in GitHub workflows, with optional installations for Elixir, Gleam, rebar3, local.hex, and local.rebar. It enables precise or flexible version management, supports multiple operating systems, and includes problem matchers for enhanced debugging. This action simplifies and accelerates the configuration of development environments for projects using Beam-based languages.
