## Action Summary

AI Review is an AI-powered code review tool designed to automate and enhance the code review process within CI/CD pipelines. It integrates seamlessly with popular version control systems (e.g., GitHub, GitLab, Bitbucket) to provide inline comments, summary reviews, and conversational replies powered by multiple large language model (LLM) providers. This tool helps developers improve code quality, ensure consistency, and expedite reviews while maintaining full human control over the process.
