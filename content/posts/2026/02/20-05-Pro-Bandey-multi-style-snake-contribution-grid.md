## Action Summary

This GitHub Action transforms your GitHub contribution graph into a dynamic, animated visualization with five unique styles, geometric shapes, and bold month labels for enhanced visual appeal and context. It automates the generation of multiple snake-themed SVG and GIF assets, organizes them in a dedicated branch, and provides an auto-generated gallery for easy sharing and integration into profile READMEs. The action simplifies customization with auto-detection of repository owners and offers high-quality outputs for profile decoration or social sharing.
