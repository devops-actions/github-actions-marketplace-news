## Action Summary

The **BundleCheck Size Audit Action** automates the auditing of npm package gzip bundle sizes in CI workflows. It identifies the gzip sizes of specific package versions, evaluates them against defined per-package or total size budgets, and reports results via PR comments. This action helps ensure package size constraints are met, provides outputs for further automation, and can enforce policies by failing workflows if budgets are exceeded.
