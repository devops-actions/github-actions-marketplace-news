## Action Summary

The `docs-build-action` GitHub Action automates the process of building project documentation using the [@diplodoc/cli](https://github.com/diplodoc-platform/cli) tool and uploads the generated build files, logs, and inputs as artifacts. It streamlines documentation workflows by ensuring consistent builds and artifact storage, which is particularly useful for managing documentation updates during pull requests. The action supports customizable build configurations, making it adaptable to different project setups.
