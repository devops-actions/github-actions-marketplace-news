## Action Summary

The `fish-shop/install-plugin-manager` GitHub Action automates the installation of plugin managers for the fish shell, such as Fisher and Oh My Fish, within CI/CD workflows. It simplifies the setup process for managing fish shell plugins, enabling users to efficiently install, remove, and manage plugins as part of their workflows. This action reduces manual setup tasks and ensures compatibility with supported plugin managers.
