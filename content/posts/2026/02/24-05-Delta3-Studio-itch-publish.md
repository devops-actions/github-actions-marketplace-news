## Action Summary

This GitHub Action automates the process of publishing game releases to itch.io by leveraging the official `butler` CLI tool. It streamlines game deployment within GitHub Actions workflows, allowing developers to upload game files, manage build versions, and specify deployment channels efficiently. This action simplifies and automates the otherwise manual process of deploying games to itch.io, saving time and reducing potential errors.
