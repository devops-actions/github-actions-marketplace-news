## Action Summary

The `idea-check-action` is a GitHub Action that evaluates the viability of a project idea by scanning platforms like GitHub, Hacker News, npm, PyPI, and Product Hunt for existing solutions. It calculates a "reality signal" score (0-100) to indicate competition levels and identifies the top competitor, helping users assess the originality and market saturation of their ideas before investing time in development. This action automates the competitive analysis process, providing actionable insights directly within GitHub workflows.
