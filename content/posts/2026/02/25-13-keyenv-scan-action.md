## Action Summary

The KeyEnv Secret Scanner GitHub Action automates the detection of hardcoded secrets, such as API keys, tokens, and passwords, in your codebase during CI/CD workflows. It helps prevent sensitive information from reaching production by providing configurable severity thresholds, JSON output for integration with downstream processes, and optional result uploads to the KeyEnv dashboard for monitoring and reporting. This action streamlines secret scanning and enhances security within software development pipelines.
