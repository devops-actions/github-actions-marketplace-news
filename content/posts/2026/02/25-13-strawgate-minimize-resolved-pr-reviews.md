## Action Summary

The "Minimize Resolved PR Reviews" GitHub Action automatically minimizes outdated pull request review comments once all threads in the review have been resolved. This helps streamline PR discussions by keeping the focus on unresolved issues, reducing visual clutter, and improving collaboration efficiency. It also supports optional features like manual triggers and user-specific filtering.
