## Action Summary

The `openspec-badge-action` is a GitHub Action that automates the generation of SVG badges displaying key metrics from OpenSpec projects, such as the number of specs, requirements, task statuses, and open changes. It simplifies project tracking by providing visual indicators of project health and progress directly in the repository. This action is particularly useful for teams seeking to monitor and communicate project status effectively.
