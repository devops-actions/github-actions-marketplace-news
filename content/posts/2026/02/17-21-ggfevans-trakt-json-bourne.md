## Action Summary

The `trakt-json-bourne` GitHub Action automates the process of fetching and normalizing user watch history data from the Trakt API, optionally enriching it with poster metadata from TMDB. It generates a clean JSON file containing recent watch entries and statistics, making it ideal for scheduled updates or manual triggers within workflows. This action simplifies data collection, reduces dependencies, and provides a streamlined, shell-based implementation for tracking and visualizing media consumption.
