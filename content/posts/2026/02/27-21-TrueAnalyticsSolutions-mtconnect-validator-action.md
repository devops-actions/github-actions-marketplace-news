## Action Summary

The MTConnect Validator GitHub Action automates the discovery and validation of MTConnect XML files in a repository by uploading them to the MTConnect Validator API. It supports file discovery via explicit lists or regex patterns, handles optional API key authentication, and provides detailed compliance and validation results. This action streamlines the process of ensuring MTConnect file compliance, reducing manual effort and enabling integration into CI/CD workflows.
