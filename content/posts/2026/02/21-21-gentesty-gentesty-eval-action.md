## Action Summary

The **Gentesty Evaluation Action** is a GitHub Action designed to automate the process of triggering evaluation runs on the Gentesty platform. It provides real-time log streaming from Gentesty to GitHub Actions and delivers the final test results (success/failure) directly into the workflow logs. This action simplifies integration with Gentesty, enabling secure and efficient testing workflows within CI/CD pipelines.
