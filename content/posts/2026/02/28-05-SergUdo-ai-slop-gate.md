## Action Summary

The **ai-slop-gate** GitHub Action is a CI/CD tool designed to detect and manage low-quality AI-generated code by combining static analysis with multi-LLM code review. It automates the enforcement of quality gates and compliance checks, such as supply chain security, GDPR/DSGVO compliance, and enterprise policy adherence, ensuring audit-ready outputs and alignment with regulatory requirements. Key capabilities include automated code review, risk-based policy enforcement, vulnerability scanning, and integration with popular CI/CD platforms.
