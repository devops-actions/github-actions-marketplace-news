## Action Summary

This GitHub Action transforms your GitHub contribution graph into a dynamic, customizable animation with five unique themes, various geometric shapes, and automated month labels for improved visualization. It automates the generation of animated SVG and GIF assets and organizes them in an "output" branch for easy access and sharing, eliminating the need for manual setup or hardcoding. This tool is ideal for enhancing GitHub profile aesthetics and showcasing contributions creatively.
