## Action Summary

This GitHub Action integrates the Radar CLI into CI/CD pipelines, automating comprehensive application security (AppSec) scans by orchestrating multiple security scanners. It identifies vulnerabilities such as hardcoded secrets, insecure code patterns, dependency issues, and misconfigurations, consolidating findings into a single SARIF report. Additionally, it offers optional integration with platforms like Eureka ASPM or GitHub Advanced Security for centralized vulnerability management and tracking.
