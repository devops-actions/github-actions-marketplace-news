## Action Summary

The **Gentesty Evaluation Action** is a GitHub Action designed to automate the process of triggering evaluation runs on the Gentesty platform. It streams real-time logs from the evaluation process directly into GitHub Actions workflow logs and provides the final success or failure result upon completion. This action simplifies and secures the integration of Gentesty's testing capabilities into CI/CD workflows, supporting both staging and production environments.
