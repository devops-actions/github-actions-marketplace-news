## Action Summary

The `fish-shop/install-plugin` GitHub Action automates the installation of plugins for the fish shell using supported plugin managers like Fisher or Oh My Fish. It simplifies and streamlines the process of managing shell plugins in CI/CD workflows by handling plugin manager installation (if necessary) and securely managing plugin inputs to mitigate security risks. This action is particularly useful for developers looking to set up or enhance their fish shell environment as part of their automated workflows.
