## Action Summary

The `bunnycdn-storage-deploy` GitHub Action automates the management of files in BunnyCDN storage. It provides functionality to upload files and folders to a storage zone, remove existing files from the storage, and purge the BunnyCDN pull zone to ensure updated files are served. This action simplifies deployment workflows by streamlining file uploads, cleanups, and cache invalidation for BunnyCDN.
