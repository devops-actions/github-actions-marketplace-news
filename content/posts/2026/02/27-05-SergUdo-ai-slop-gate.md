## Action Summary

**ai-slop-gate** is an open-source CI/CD tool that combines static analysis and multi-LLM code review to detect low-quality, AI-generated code and enforce automated quality gates. It enhances software supply chain security with features like SBOM generation, license audits, and CVE scanning, while supporting compliance with regulations such as GDPR and the EU Cyber Resilience Act. The action streamlines DevSecOps workflows by enabling deterministic, audit-friendly policy enforcement, centralized risk management, and seamless integration with CI/CD pipelines.
