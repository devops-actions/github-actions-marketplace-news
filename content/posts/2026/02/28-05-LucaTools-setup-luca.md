## Action Summary

The `setup-luca` GitHub Action automates the installation of the Luca CLI, a lightweight tool manager, and optionally installs tools specified in a configuration file (spec file). This action simplifies environment setup by ensuring required tools are installed and available in the workflow's `PATH`, streamlining build and CI/CD processes. It also supports pinning specific Luca versions for consistent tool management.
