## Action Summary

Hwaro is a lightweight and fast static site generator written in Crystal, designed to simplify the process of building high-performance websites. It automates tasks like Markdown content processing, template rendering with Jinja2-like syntax, and generating essential web files (e.g., sitemaps and RSS feeds), while also offering features such as SEO optimization, build hooks, and caching for improved performance. This tool is ideal for developers managing blogs, documentation, or personal sites who seek an efficient and flexible solution without unnecessary complexity.
