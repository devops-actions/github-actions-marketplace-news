## Action Summary

The `csda-version` GitHub Action calculates the next CSDA version for a repository based on a specific versioning format tied to program increments, sprints, and release numbers. It automates version determination to streamline release workflows, particularly when used in conjunction with tools like `release-please`. This action simplifies version management and ensures consistent versioning for repositories.
