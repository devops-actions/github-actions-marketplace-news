## Action Summary

This GitHub Action is designed to build and preview updates to [Stainless](https://stainless.com/) SDKs directly from GitHub workflows, automating tasks such as generating SDKs based on changes to OpenAPI specifications or Stainless configurations in pull requests. It supports both GitHub and GitLab CI environments, with authentication options like GitHub OIDC or API keys, and provides seamless integration to streamline the SDK development and deployment process. This action simplifies and accelerates SDK updates, ensuring that changes are previewed and validated during the CI/CD lifecycle.
