## Action Summary

The `andrewaylett/pre-commit-action` GitHub Action automates the execution of [pre-commit](https://pre-commit.com) hooks in CI workflows. It simplifies tasks such as code linting and formatting by running pre-configured hooks against a repository's codebase during pull requests or pushes. This action ensures code quality and consistency by integrating `pre-commit` checks seamlessly into a GitHub workflow.
