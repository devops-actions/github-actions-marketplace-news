## Action Summary

The "Gemini i18n Translator" GitHub Action automates the process of translating i18n JSON files using Google Gemini models. It detects changes in source files, generates translations in the specified target languages, and creates a Pull Request with the results. This action streamlines translation workflows by leveraging high-quality AI-driven translations, managing branch creation, and ensuring seamless integration with GitHub repositories.
