## Action Summary

The `fish-shop/install-plugin` GitHub Action automates the installation of plugins for the fish shell using supported plugin managers, such as Fisher and Oh My Fish. It simplifies the process of setting up and managing fish shell plugins in CI workflows by handling plugin manager installation (if necessary) and securely installing specified plugins. This action streamlines development workflows by reducing manual intervention and ensuring consistent plugin setups.
