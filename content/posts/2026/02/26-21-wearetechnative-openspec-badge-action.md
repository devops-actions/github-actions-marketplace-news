## Action Summary

The `openspec-badge-action` is a GitHub Action that automates the generation and deployment of SVG badges displaying key metrics from OpenSpec, such as the number of specification files, defined requirements, open changes, and task completion status. It provides a visual summary of a project's health directly in the repository's README, helping teams monitor progress and maintain project transparency.
