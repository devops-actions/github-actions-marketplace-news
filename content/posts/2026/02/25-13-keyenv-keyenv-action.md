## Action Summary

The KeyEnv GitHub Action automates the retrieval of secrets from KeyEnv and integrates them into GitHub Actions workflows as environment variables or `.env` files. It simplifies secret management by securely fetching and masking sensitive values, supporting both project-scoped and non-scoped service tokens. This action eliminates the need for manual secret handling, enhancing workflow security and efficiency.
