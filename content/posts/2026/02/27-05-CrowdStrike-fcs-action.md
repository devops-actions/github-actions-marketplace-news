## Action Summary

The FCS CLI GitHub Action integrates the CrowdStrike Falcon Cloud Security CLI into CI/CD pipelines to automate security scanning tasks. It enables Infrastructure as Code (IaC) scanning for misconfigurations and vulnerabilities, container image scanning for security issues, and the generation of Software Bill of Materials (SBOMs) in CycloneDX format. This action streamlines security assessments by providing flexible configuration options, customizable scan parameters, and automatic result uploads to the CrowdStrike Falcon Console.
