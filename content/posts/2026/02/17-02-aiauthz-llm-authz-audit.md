## Action Summary

The `llm-authz-audit` GitHub Action is a security auditing tool designed to scan LLM-powered applications for vulnerabilities such as authorization gaps, leaked credentials, missing rate limits, and prompt injection risks. It automates the detection of security issues using 13 analyzers and 27 rules aligned with the OWASP Top 10 for LLM Applications, ensuring code is secure before deployment. Its key capabilities include cross-file authorization analysis, rule-based vulnerability detection, and severity-based reporting to streamline security reviews.
