## Action Summary

The **Contributor Report** GitHub Action analyzes pull request (PR) contributors and generates detailed reports using objective metrics to help maintainers identify and mitigate low-quality, AI-generated spam contributions. It automates the evaluation of a contributor's activity history, providing pass/fail metric results, recommendations, and insights to improve contribution quality while enabling informed decision-making for repository maintainers. This action addresses the growing challenge of "AI slop" in open-source projects by promoting transparency and reducing maintainer workload.
