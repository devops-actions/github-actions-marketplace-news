## Action Summary

This GitHub Action, **Cairn**, automates the generation and deployment of historical reports to GitHub Pages by collecting and ingesting CI outputs. It simplifies tasks such as gathering check results, managing run records, building static report assets, publishing them, and optionally posting PR summary comments. Cairn is designed to streamline CI/CD workflows by maintaining a historical record and providing clear, accessible reporting directly within a repository.
