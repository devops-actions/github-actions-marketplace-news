## Action Summary

The **InariWatch Risk Assessment GitHub Action** automates AI-powered pre-deployment risk analysis for pull requests by analyzing code changes (diffs) and posting a risk score with detailed findings and recommendations directly on the PR. This action helps developers identify potential risks early, ensuring safer and more informed code merges. It supports multiple AI providers and updates comments dynamically with each commit to avoid unnecessary clutter.
