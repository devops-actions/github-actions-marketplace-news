## Action Summary

The `vcopy` GitHub Action and CLI tool enables seamless copying of GitHub repositories between organizations, whether on GitHub Cloud, Enterprise, or both, with comprehensive support for mirroring branches, tags, commits, releases, and optional metadata like issues and wikis. It ensures data integrity with a robust 5-layer cryptographic verification process, solving challenges related to accurate and secure repository migration. Key features include batch copying, customizable path exclusions, metadata migration, and detailed verification reports, making it a powerful tool for automated, reliable repository replication and auditing.
