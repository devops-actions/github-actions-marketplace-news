## Action Summary

The "Gemini i18n Translator" GitHub Action automates the translation of i18n JSON files using Google Gemini models and creates a Pull Request with the translated content. It simplifies localization workflows by detecting file changes, translating them, and managing version control tasks like branch creation and Pull Request submission. This action ensures high-quality translations with customizable source paths, output destinations, and language targets.
