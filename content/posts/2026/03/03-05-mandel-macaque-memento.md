## Action Summary

`git-memento` is a Git extension designed to automatically attach AI-generated coding session traces as human-readable markdown notes to Git commits. It enables developers to preserve the context of their AI-assisted coding by linking session transcripts to commits using Git Notes, supporting extensible AI providers. This action streamlines collaboration and auditability by automating the storage, synchronization, and sharing of commit metadata across repositories and teams.
