## Action Summary

`ci-rootcause` is a deterministic CI failure analysis engine designed to identify and rank the root causes of failed CI runs. It generates structured failure graphs, confidence scores, evidence-backed fix plans, and optional guarded fix pull requests to automate the process of diagnosing and addressing issues in CI pipelines. This tool simplifies root cause analysis, provides actionable insights, and integrates seamlessly with GitHub Actions and GitLab CI workflows.
