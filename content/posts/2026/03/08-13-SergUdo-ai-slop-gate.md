## Action Summary

**ai-slop-gate** is an open-source CI/CD tool that combines static analysis and multi-LLM (Large Language Model) code review to detect and mitigate low-quality or AI-generated code. It automates tasks such as supply chain security checks (e.g., SBOM generation, license audits, CVE scanning), ensures compliance with regulations like GDPR and the EU Cyber Resilience Act, and enforces policy-as-code for consistent governance. Its key capabilities include multi-LLM support, audit-ready reporting, and integration with popular CI/CD platforms for streamlined, proactive code quality and security management.
