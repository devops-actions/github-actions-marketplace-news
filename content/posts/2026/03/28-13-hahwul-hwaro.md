## Action Summary

Hwaro is a lightweight and fast static site generator written in Crystal that processes Markdown with TOML front matter and Jinja2-compatible templates to build high-performance static websites. It automates tasks like content management, SEO optimization, and build performance enhancements, including parallel processing, live reload, and incremental caching. Key capabilities include advanced templating, multilingual support, image processing, and seamless deployment to multiple platforms such as GitHub Pages, Netlify, and Vercel.
