## Action Summary

The `pkl-swift-action` GitHub Action automates the generation of Swift source code from Pkl model files. It streamlines the process of converting Pkl data into Swift bindings, saving developers time and reducing manual coding effort when working with Pkl models in Swift projects. Key capabilities include processing multiple Pkl files and specifying output locations for the generated code.
