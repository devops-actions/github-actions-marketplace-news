## Action Summary

The "spit-the-diff" GitHub Action uses AI to transform pull request diffs into creative summaries, such as rap verses, haikus, or humorous code roasts. It automates the process of generating engaging and easily digestible change summaries, helping teams review code changes in a fun and innovative way. The action supports multiple AI providers and updates a single persistent comment on pull requests.
