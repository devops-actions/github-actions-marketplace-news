## Action Summary

The **WP Composer Changelog Action** is a GitHub Action that automatically comments WordPress plugin changelogs on pull requests whenever WP Composer dependencies in `composer.lock` or `composer.json` files are updated. It fetches changelogs from the WordPress.org API, highlights potential issues with unstable versions, and helps streamline dependency change reviews. This action simplifies tracking plugin updates and ensures better version management for WordPress projects.
