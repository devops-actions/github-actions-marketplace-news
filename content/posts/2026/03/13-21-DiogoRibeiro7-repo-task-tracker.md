## Action Summary

The `repo-task-tracker` GitHub Action automates the process of converting a `tracker.json` file in a repository into GitHub Issues and syncing those issues to a centralized GitHub Projects (v2) board. It streamlines task tracking across multiple repositories, ensuring issue statuses, labels, and project board fields are consistently updated. This action is ideal for managing tasks across diverse codebases from a single, unified location.
