## Action Summary

The **github-screenshot-action** is a GitHub Action designed to automate the capture, monitoring, and versioning of website screenshots using a JSON-based configuration. It simplifies website monitoring by automating screenshot generation with support for parallel execution, retry logic, cron scheduling, and automatic pull request creation. Key features include Puppeteer-based rendering for accurate visual results, customizable wait strategies, and the ability to isolate screenshots in a dedicated branch for streamlined version control.
