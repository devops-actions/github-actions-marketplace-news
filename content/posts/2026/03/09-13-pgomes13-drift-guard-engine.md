## Action Summary

The **drift-guard** GitHub Action automates the detection and classification of breaking and non-breaking API contract changes across OpenAPI, GraphQL, and gRPC schemas. It helps developers ensure API compatibility and stability by identifying changes that could impact system integrations or client usage. Key capabilities include automated schema comparison and impact analysis to streamline API version management.
