## Action Summary

The `action-bump-cli` GitHub Action automates version management by updating semantic version strings in any file using regex patterns, leveraging the `bump` CLI. It simplifies workflows by allowing users to specify the file, version pattern, and desired version increment (major, minor, or patch) or set an explicit version, without requiring manual edits. This action is versatile, supporting various file formats, and provides flexibility for seamless integration into versioning and release processes.
