## Action Summary

Drift is a GitHub Action and analysis tool designed to detect architecture erosion in AI-accelerated Python codebases. It identifies structural issues such as pattern fragmentation, boundary violations, and hotspots that may go unnoticed but degrade code quality over time. By providing fast, deterministic feedback without relying on large language models, Drift enables teams to maintain robust code structures in AI-assisted development workflows.
