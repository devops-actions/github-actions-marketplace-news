## Action Summary

This GitHub Action facilitates the deployment of Kubernetes manifests to clusters, automating tasks such as artifact substitution, resource stability checks, and secret management. It supports advanced deployment strategies, including canary and blue-green deployments, with options for service mesh integration or standard Kubernetes routing. This action streamlines Kubernetes workflows, ensuring efficient and reliable application deployments.
