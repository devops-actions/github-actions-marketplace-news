## Action Summary

**Changelog.Action** is a GitHub Action designed to automate the process of updating changelog files in repositories adhering to the [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) format. It streamlines the management of changelog entries by updating version numbers and extracting release notes from the `[Unreleased]` section, which can be used for tasks like creating GitHub releases. This action simplifies release workflows by ensuring accurate and consistent changelog updates.
