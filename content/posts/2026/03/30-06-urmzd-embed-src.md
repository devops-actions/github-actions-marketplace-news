## Action Summary

The `embed-src` GitHub Action automates the process of embedding external source files into text-based documents (e.g., Markdown, YAML, code files) by replacing special comment placeholders with the referenced file content. It supports various file types and allows for raw or code-fenced insertion, making it easier to maintain up-to-date, accurate documentation or configuration files. This action streamlines workflows by eliminating manual copying and ensures consistency across embedded content.
