## Action Summary

Basilisk is an open-source AI red teaming and LLM security testing framework designed for security researchers, penetration testers, and AI safety engineers. It automates the discovery of vulnerabilities in language models, such as GPT-4, Claude, and Gemini, using genetic prompt evolution to generate adversarial inputs that expose weaknesses. With features like comprehensive attack modules, multi-model scanning, and forensic logging, it streamlines the process of identifying and mitigating AI system vulnerabilities.
