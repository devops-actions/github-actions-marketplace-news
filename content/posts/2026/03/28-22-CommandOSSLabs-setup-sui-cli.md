## Action Summary

The **setup-sui-cli** GitHub Action automates the installation of a specified version of the Sui CLI, a command-line tool for interacting with the Sui blockchain, and optionally configures a deployer wallet for a supported Sui network. It simplifies tasks such as managing blockchain accounts, switching networks, and preparing environments for deploying or testing on Sui. The action provides key outputs like the resolved RPC URL and active wallet address, streamlining blockchain development workflows.
