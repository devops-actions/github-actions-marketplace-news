## Action Summary

The `graph-sync` GitHub Action automates the execution of the `worai` command-line tool to synchronize knowledge graphs using a specified profile and configuration. It streamlines tasks such as installing the required `worai` version, setting up Python Playwright with browser dependencies, and running the graph synchronization process. This action simplifies the setup and execution of graph synchronization workflows, supports caching for dependencies, and enables integration with multiple input sources like URLs, sitemaps, and Google Sheets.
