## Action Summary

This GitHub Action, "Maximize GitHub Runner Space," reclaims disk space on GitHub-hosted `ubuntu-latest` runners by removing preinstalled tools, SDKs, caches, and optional components. It automates cleanup tasks, runs them in parallel, and provides disk space before-and-after reporting, solving the problem of limited storage for large builds or tests. Its key capabilities include customizable cleanup profiles, selective removal of components, and support for maximizing available disk space efficiently.
