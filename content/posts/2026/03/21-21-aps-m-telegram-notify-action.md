## Action Summary

The `telegram-notify-action` GitHub Action automates the process of sending text messages and files to Telegram through a bot. It is designed to facilitate streamlined communication by enabling notifications directly from workflows, supporting text messages with optional parsing modes, file attachments, and messages sourced from external files. This action is ideal for integrating Telegram notifications into CI/CD pipelines or automated processes.
