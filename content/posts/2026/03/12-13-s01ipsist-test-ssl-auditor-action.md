## Action Summary

The **test-ssl-auditor-action** is a GitHub Action designed to audit the JSON output from [test-ssl-action](https://github.com/s01ipsist/test-ssl-action) against configurable security rules. It automates the process of validating SSL/TLS configurations by checking for compliance with user-defined rules, such as minimum TLS version, blocked ciphers, and certificate expiry limits. The action provides detailed violation reports and pass/fail control, streamlining SSL/TLS security assessments in CI/CD workflows.
