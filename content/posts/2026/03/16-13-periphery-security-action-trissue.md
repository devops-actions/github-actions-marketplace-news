## Action Summary

TrIssue is a GitHub Action designed to automate the process of managing vulnerabilities detected by Trivy scans. It converts Trivy scan results into GitHub issues, enabling repositories to track and prioritize vulnerabilities efficiently. Key capabilities include adding customizable labels, identifying fixable vulnerabilities, and supporting dry-run mode for testing without issue creation.
