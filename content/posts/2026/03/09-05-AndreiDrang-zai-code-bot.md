## Action Summary

The Zai Code Bot GitHub Action automates pull request reviews and enables context-aware, interactive commands powered by Z.ai models. It streamlines code reviews by providing automatic PR assessments, context-rich explanations, and impact analyses, while allowing developers to use commands like `/zai review` and `/zai explain` directly in PR comments. Its key capabilities include inline review comments, support for large files, and adaptive command execution based on PR context, improving review efficiency and collaboration.
