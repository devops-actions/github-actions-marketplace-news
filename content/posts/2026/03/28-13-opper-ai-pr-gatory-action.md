## Action Summary

**pr-gatory** is a GitHub Action that implements a merge queue for pull requests, enabling serialized, automated merging without requiring external services or GitHub Enterprise. It automates tasks such as rebasing PRs onto the latest base branch, verifying status checks, performing squash merges, and handling failures with actionable feedback. This ensures that each PR is tested against the up-to-date codebase, mitigating risks of integration conflicts and streamlining the merge process.
