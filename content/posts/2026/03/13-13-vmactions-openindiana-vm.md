## Action Summary

This GitHub Action enables users to run CI workflows on OpenIndiana, an operating system not natively supported by GitHub Actions, by creating and managing virtual machines. It automates the process of environment setup, including syncing files, transferring environment variables, and providing a consistent working directory between the host and VM. The action supports customizable VM configurations, such as memory, CPU, architecture, networking, and file synchronization methods, offering a flexible solution for testing and building across OpenIndiana environments.
