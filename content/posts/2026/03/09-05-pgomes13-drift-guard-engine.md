## Action Summary

Drift Guard is a GitHub Action that detects and classifies breaking and non-breaking changes in API contracts for OpenAPI, GraphQL, and gRPC schemas. It automates the process of comparing schema changes across branches, helping developers ensure API stability and compatibility. This tool is particularly useful for maintaining consistent and reliable API versions during development and deployment workflows.
