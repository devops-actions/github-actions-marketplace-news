## Action Summary

The **oxidized-skills** GitHub Action is a security auditing tool designed to analyze AI agent skills and associated directories for vulnerabilities, risks, and compliance issues. It automates the detection of dangerous patterns, prompt injection risks, insecure package installations, and other security flaws across various file types and scripting languages. Key capabilities include static analysis, secret scanning, configurable ignore rules, and output in multiple formats (e.g., JSON, SARIF), providing a comprehensive and efficient solution for enhancing the security and reliability of AI-driven projects.
