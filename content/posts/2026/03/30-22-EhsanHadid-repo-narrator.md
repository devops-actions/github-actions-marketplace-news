## Action Summary

Repo Narrator is a GitHub Action that automates repository documentation management by generating and updating annotated file trees, `structure.md` (a guided overview of the codebase), and markdown files like `README.md` and `CHANGELOG.md` after every merged pull request. It streamlines the process of keeping repository documentation synchronized with code changes, ensuring clarity and consistency. The action supports AI-driven capabilities using Claude, OpenAI, or Gemini to enhance its functionality.
