## Action Summary

The **Skill Eval Action** is a GitHub Action designed to automate the evaluation of Claude Code skills against YAML-defined test cases. It provides automated grading and generates detailed reports directly within pull requests, streamlining the testing process for skill development. Its key capabilities include support for single or batch skill evaluations, dynamic discovery of skills, and the ability to evaluate only modified skills to optimize CI workflows.
