## Action Summary

This GitHub Action, formerly known as Digger (now OpenTaco), provides CI/CD automation for Terraform and OpenTofu directly within existing CI systems. It eliminates the need for separate infrastructure by running Terraform securely and cost-efficiently in your CI environment, solving issues like sensitive data exposure, race conditions, and scalability. Key capabilities include PR-level locks, drift detection, Open Policy Agent (OPA) support, and native Terraform plan/apply workflows, making it a robust alternative to platforms like Atlantis and Terraform Cloud.
