## Action Summary

The `iserter/php-obfuscator` GitHub Action is a tool designed to automatically obfuscate PHP source code in CI/CD pipelines, providing copyright protection and enhanced deobfuscation resistance. It supports modern PHP versions (7.4–8.5) and utilizes techniques like control flow flattening, string encoding, and symbol scrambling to secure code against reverse engineering. Key capabilities include incremental processing, multi-pass analysis for consistent symbol scrambling, and extensive configurability via YAML files.
