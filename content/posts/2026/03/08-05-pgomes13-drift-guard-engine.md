## Action Summary

Drift-Guard is a GitHub Action designed to detect and classify breaking and non-breaking changes in API contracts for OpenAPI, GraphQL, and gRPC schemas. It automates schema comparison tasks, helping developers ensure API compatibility and prevent disruptions caused by unintended breaking changes. This action simplifies the process of validating schema consistency across different code branches.
