## Action Summary

The **Drawio Export Action** automates the process of exporting Draw.io diagrams to various file formats, such as PNG, PDF, SVG, and more, directly within a GitHub repository. It simplifies tasks like keeping exported diagram files synchronized with changes to the source `.drawio` files, ensuring consistent outputs across workflows. Key capabilities include customizable export formats, transparent backgrounds, scalable diagrams, and support for exporting all pages or recent changes.
