## Action Summary

DriftaBot Agent is a GitHub Action designed to detect breaking API changes in pull requests for provider repositories by comparing OpenAPI schemas. It automates the identification of impacted consumer repositories, scans them for affected files, and creates or updates GitHub Issues in those repositories. This action streamlines the process of managing API changes, ensuring consumers are promptly informed without requiring additional configuration on their end.
