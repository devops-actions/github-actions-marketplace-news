## Action Summary

The **TrueNAS Scale API Client Action** is a GitHub Action designed to interact with TrueNAS Scale via its JSON-RPC 2.0 WebSocket API. It automates tasks such as authenticating with the TrueNAS API and executing specified API methods, returning the results as a JSON string for further use in workflows. This action simplifies integration with TrueNAS systems, enabling efficient querying and management of resources.
