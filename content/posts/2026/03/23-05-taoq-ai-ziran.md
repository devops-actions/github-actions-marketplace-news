## Action Summary

ZIRAN is a GitHub Action and security testing tool designed to identify vulnerabilities in AI agents, including those with tools, memory, and multi-step reasoning capabilities. It automates the discovery of attack paths arising from dangerous tool combinations, detects side effects from agent actions, and executes multi-phase penetration testing campaigns. Key features include tool chain discovery, autonomous pentesting, multi-agent coordination, and protocol-specific security testing, making it ideal for comprehensive AI agent security analysis.
