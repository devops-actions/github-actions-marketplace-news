## Action Summary

The **Quant Agent Dispatch Action** is a GitHub Action designed to automate the testing of AI agents on the Quant platform by running them in asynchronous test mode with deterministic responses. It facilitates sending user messages to agents, simulating interactions using tool fixtures, and automatically polling for agent completion. The action outputs the agent's results and full conversation context, making it ideal for validating agent behavior and enabling reproducible test workflows.
