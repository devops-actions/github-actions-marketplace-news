## Action Summary

This GitHub Action retrieves the latest semantic version number from tagged images in a specified Docker Hub repository. It automates the process of identifying the most recent version based on customizable version constraints, helping developers ensure they are using or referencing up-to-date Docker images. Key capabilities include filtering tags by minimum or maximum version criteria and returning the latest tag and version.
