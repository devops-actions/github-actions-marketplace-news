## Action Summary

The `build-docker` GitHub Action streamlines the process of building, pushing, and securing Docker images. It automates multi-platform image builds, integrates with container registries (e.g., GitHub Container Registry, Docker Hub), and performs security tasks like Docker Scout vulnerability scanning and Kubescape analysis. Additionally, it supports metadata handling, caching optimization, and SARIF report generation, making container management more efficient and secure.
