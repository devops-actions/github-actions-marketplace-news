## Action Summary

The `agent-bom` GitHub Action is a security scanner designed for AI infrastructure, automating the identification of vulnerabilities (CVEs), credential exposures, and potential attack vectors across various MCP agents, containers, Kubernetes clusters, cloud environments, and GPU workloads. It provides detailed insights into compromised AI agents, exposed credentials, affected tools, and the business impact of security risks, offering actionable recommendations to mitigate threats. Key capabilities include vulnerability enrichment, blast radius mapping, container and cloud scanning, and enforcement of security gates in CI/CD pipelines.
