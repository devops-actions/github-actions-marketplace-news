## Action Summary

The `Setup-Cangjie` GitHub Action automates the installation of the Cangjie build environment within workflows, supporting LTS, STS, and Nightly versions. It simplifies the setup process by handling version selection, configuration, and optional token integration for authenticated access to the Cangjie repository. This action is ideal for streamlining build and deployment processes that require the Cangjie environment.
