## Action Summary

The **Sentinel AI Compliance Scan** GitHub Action automates the process of verifying AI projects for compliance with the **EU AI Act** regulations directly within GitHub workflows. It securely performs high-precision audits locally on your GitHub runner without exposing code or metadata, ensuring privacy. Key capabilities include automatic PR gating for non-compliant projects, a compliance badge for transparency, and fast, secure audits powered by a Rust/WASM architecture.
