## Action Summary

This GitHub Action automates the process of obtaining a Yandex Cloud IAM token for service accounts using Workload Identity Federation. It eliminates the need for storing and managing authorized keys by securely exchanging a GitHub OpenID Connect ID Token for a Yandex Cloud IAM token. This action facilitates seamless integration with Yandex Cloud resources in workflows, enabling secure and efficient service account authentication.
