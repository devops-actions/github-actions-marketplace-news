## Action Summary

The `codeowner-gen` GitHub Action automates the generation and formatting of a `CODEOWNERS` file from a structured YAML configuration (`codeowners.yaml`). It simplifies the management of code ownership by ensuring the output is well-formatted, column-aligned, alphabetized, and optionally grouped for better readability. This action reduces the complexity of reviewing `CODEOWNERS` file changes in pull requests by shifting focus to the YAML configuration, making it easier to maintain accurate and organized ownership rules.
