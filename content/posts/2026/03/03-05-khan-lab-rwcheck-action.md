## Action Summary

This GitHub Action, RWCheck, automates the screening of `.bib` files in a repository against the Retraction Watch database to identify retracted references based on DOI or PMID. It helps ensure the integrity of bibliographic data by flagging or failing workflows when retracted references are detected, and generates a detailed Markdown summary of the findings. This tool streamlines the process of verifying references, making it particularly useful for researchers and developers working with academic publications.
