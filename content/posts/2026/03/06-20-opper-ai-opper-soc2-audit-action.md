## Action Summary

The "SOC2 Compliance Audit" GitHub Action automates the process of auditing a repository for SOC2 compliance by leveraging AI-powered agents. It evaluates the repository against key SOC2 Trust Service Criteria (e.g., Security, Availability, Processing Integrity, Confidentiality, Privacy) and generates a detailed findings report. This action helps organizations identify potential compliance gaps, manage risks, and optionally automate issue creation or fixes, streamlining the path to SOC2 certification.
