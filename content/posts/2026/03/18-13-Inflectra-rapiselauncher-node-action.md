## Action Summary

The `Inflectra/rapiselauncher-node-action` GitHub Action automates the execution of Rapise test sets stored in SpiraTest (or related Spira products) by installing the Rapise engine and running tests via RapiseLauncher. It simplifies continuous integration workflows by enabling seamless test execution on both Linux and Windows runners, with support for passing custom parameters and leveraging existing Spira configurations. This action is ideal for teams using SpiraTest to manage automated testing, as it eliminates the need for manual test execution and ensures consistent integration within CI/CD pipelines.
