## Action Summary

**AI Review** is an AI-powered code review tool designed to automate and enhance the code review process within CI/CD workflows. It integrates with popular version control systems like GitHub, GitLab, and Bitbucket, providing inline comments, summary reviews, and AI-generated replies to pull requests. By enabling features like customizable prompts, iterative repository exploration (agent mode), and support for multiple language models, it helps improve code quality, enforce consistency, and streamline team collaboration in code reviews.
