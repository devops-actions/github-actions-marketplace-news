## Action Summary

The `setup-tq` GitHub Action automates the installation and configuration of `tq` (tomlq), a CLI tool for querying and manipulating TOML files. It simplifies workflows by enabling users to quickly set up `tq` within their GitHub Actions, making it easier to extract and process data from TOML files during CI/CD pipelines. Key capabilities include specifying the `tq` version and caching the binary for optimized performance.
