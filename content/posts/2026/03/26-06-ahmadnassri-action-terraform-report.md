## Action Summary

The **Terraform Pull Request Report Generator** GitHub Action automates the process of updating pull requests with a visual summary of changes from a Terraform Plan. It provides detailed insights into infrastructure modifications by generating and posting plan and diff reports directly in pull request comments. This helps streamline collaboration, improve visibility into proposed changes, and ensures better tracking of infrastructure updates during code reviews.
