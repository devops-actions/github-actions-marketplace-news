## Action Summary

**Agent-BOM** is a security scanner designed to analyze AI infrastructure and supply chains by discovering, scanning, analyzing, and enforcing security measures. It generates AI Bills of Materials (BOMs), identifies vulnerabilities (CVEs), evaluates the blast radius of compromises, and ensures compliance with 14 frameworks. The tool automates critical security tasks such as MCP agent detection, runtime traffic enforcement, misconfiguration scanning, and cloud posture assessment, offering comprehensive insights and real-time mitigation to secure AI ecosystems.
