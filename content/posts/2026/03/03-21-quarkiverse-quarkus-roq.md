## Action Summary

Quarkus Roq is a static site generator built on Java and Quarkus, designed to simplify the creation of websites and blogs using Markdown, Asciidoc, or HTML content with FrontMatter metadata. It automates tasks like generating static HTML and asset files from dynamic Quarkus applications and supports integration with GitHub Actions for streamlined deployment. The tool provides features like type-safe content management with JSON/YAML and flexible customization of website layouts and SEO.
