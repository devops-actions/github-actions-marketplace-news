## Action Summary

The **Hosho Prompt Reviewer** GitHub Action evaluates AI agent prompts against six research-backed prompt engineering quality factors, such as scope, structure, context, and model-specific alignment. It identifies weaknesses in prompts, provides actionable improvement suggestions with before/after snippets, and automates quality reviews by posting feedback directly on pull requests or in the Actions tab. This streamlines prompt optimization and ensures adherence to best practices, saving time and improving prompt effectiveness.
