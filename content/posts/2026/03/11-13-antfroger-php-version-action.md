## Action Summary

The **PHP Version Action** is a GitHub Action that automates the detection of PHP versions required for your workflows by analyzing the `composer.json` file. It generates a version matrix ranging from the minimum to the latest supported PHP versions, eliminates the need for hardcoding PHP versions, and ensures workflows stay up-to-date with real-time version data from [PHP.watch](https://php.watch). This action simplifies managing PHP version constraints, provides flexible outputs (minimum, latest, and matrix versions), and supports semantic versioning.
