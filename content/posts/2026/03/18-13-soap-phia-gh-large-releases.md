## Action Summary

This GitHub Action enables uploading release files larger than 2GB to GitHub by splitting them into smaller parts and using Cloudflare Workers for seamless concatenation during download. It solves the problem of GitHub’s file size limit for releases, making it possible to handle large artifacts efficiently. Key capabilities include automated file splitting, upload, and public or private download support via custom Cloudflare Worker URLs.
