## Action Summary

**Pully** is a GitHub Action that automates the tracking and reporting of pull request (PR) statuses directly to a designated Slack channel. It provides real-time updates through a single, dynamically edited Slack message per PR, summarizing key details like PR status, title, author, review requests, approvals, and change requests. This action streamlines collaboration in fast-paced environments by eliminating the need for manual status updates and enabling teams to monitor PR progress efficiently in Slack.
