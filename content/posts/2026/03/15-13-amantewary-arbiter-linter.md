## Action Summary

Arbiter is a GitHub Action and CLI tool designed to lint and analyze system prompts for large language models (LLMs). It detects issues such as logical conflicts, redundancies, ambiguities, and interference patterns that can cause LLMs to behave unpredictably or fail. By treating prompts as code, Arbiter automates the process of identifying and resolving these issues, ensuring more reliable and efficient autonomous agent performance.
