## Action Summary

Python Nurse is a GitHub Action designed to perform static analysis on Python monorepo applications, focusing on linting, type checking, security scanning, dead code detection, code complexity, and more. It automates the evaluation of changed files in pull requests, providing a 0–100 health score with detailed diagnostics as a PR comment. The action streamlines code quality checks, enhancing development workflows by consolidating multiple tools into a lightweight and efficient solution.
