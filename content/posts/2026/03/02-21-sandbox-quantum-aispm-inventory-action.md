## Action Summary

This GitHub Action integrates with AQtive Guard by SandboxAQ to perform static analysis on code repositories, identifying and inventorying AI-related assets such as models, agents, and MCP servers. It automates the process of detecting AI components, sending the inventory to an AQtive Guard instance for enrichment and analysis, and allowing users to review results in the AQtive Guard web interface. This action simplifies AI asset management and enhances security oversight for projects leveraging AI.
