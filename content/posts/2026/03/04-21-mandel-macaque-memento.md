## Action Summary

The `git-memento` Git extension enhances AI-assisted development by attaching AI conversation transcripts as Git notes to commits, creating a transparent audit trail. It solves the problem of losing context around AI-generated code changes by documenting the "why" behind modifications, aiding code reviews, compliance, debugging, and onboarding. Key capabilities include seamless integration with Git workflows, provider extensibility (e.g., Codex, Claude Code), and the generation of human-readable markdown notes.
