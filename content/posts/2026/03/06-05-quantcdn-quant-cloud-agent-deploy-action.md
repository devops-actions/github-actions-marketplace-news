## Action Summary

The Quant Agent Deploy Action is a GitHub Action designed to automate the creation, updating, and management of AI agents on the Quant platform. It reads agent definitions from a specified directory, interacts with the Quant API to create or update agents, and ensures idempotency by maintaining a consistent state for agent configurations. This action streamlines the deployment process, supports version-controlled prompts, and simplifies agent lifecycle management, reducing manual effort and minimizing the risk of configuration errors.
