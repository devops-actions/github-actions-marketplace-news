## Action Summary

This GitHub Action enables the automated publishing of [Allure reports](https://github.com/allure-framework/allure2) to [NeoFS](https://fs.neo.org), a decentralized distributed object storage integrated with the NEO Blockchain. It simplifies the process of securely storing and sharing test results by leveraging NeoFS infrastructure, making it particularly useful for projects with similar reporting workflows. The action offers seamless integration with NeoFS and serves as a reference for utilizing the `publish-to-neofs` tool for flexible object storage.
