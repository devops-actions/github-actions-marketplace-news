## Action Summary

The Canyon GitHub Action automates the process of uploading code coverage reports in JSON format to a Canyon server. It simplifies integration with CI/CD pipelines by extracting relevant repository, commit, and environment details from GitHub Actions workflows, while offering flexible configuration and optional authentication. This action is particularly useful for teams looking to streamline coverage tracking and manage test scenarios more effectively.
