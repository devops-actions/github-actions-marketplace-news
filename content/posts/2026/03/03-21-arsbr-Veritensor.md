## Action Summary

Veritensor is a GitHub Action designed to enhance security across the AI supply chain by scanning and detecting threats in AI artifacts such as models, datasets, RAG (retrieval-augmented generation) documents, and notebooks. It automates the identification and mitigation of vulnerabilities like data poisoning, prompt injections, compromised dependencies, and leaked secrets, which are often missed by traditional static analysis tools. The action provides advanced capabilities including binary format analysis, runtime threat blocking, cryptographic data provenance, and high-performance parallel scanning to ensure robust protection for machine learning workflows.
