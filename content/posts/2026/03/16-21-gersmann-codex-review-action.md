## Action Summary

The Codex Review Action is a GitHub Action designed to automate the review and editing of pull requests using OpenAI's Codex model. It provides inline review comments, a summary of findings, and enables trusted users to trigger autonomous edits through `/codex` comments. This action streamlines code review processes, reduces manual effort, and ensures consistent application of review standards.
