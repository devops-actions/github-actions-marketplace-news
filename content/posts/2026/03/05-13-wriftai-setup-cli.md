## Action Summary

The WriftAI CLI Action automates the installation of the WriftAI Command Line Interface (CLI) on GitHub Actions runners across multiple operating systems (`ubuntu-latest`, `windows-latest`, and `macos-latest`). It simplifies setting up the CLI in workflows, enabling users to run WriftAI commands directly within their CI/CD pipelines. This action streamlines environment preparation and ensures consistent access to the specified or latest version of the WriftAI CLI.
