## Action Summary

The `firenza/get-organization-repositories` GitHub Action retrieves a comprehensive list of repositories within a specified GitHub organization and outputs the data in JSON format. It automates the process of fetching repository information, eliminating the need for manual API queries or scripting. This action is particularly useful for workflows requiring repository data for auditing, analysis, or further automated processing.
