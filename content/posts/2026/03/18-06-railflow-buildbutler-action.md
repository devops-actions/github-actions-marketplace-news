## Action Summary

The **Build Butler GitHub Action** automates the collection and reporting of build status, test results, and self-hosted runner metrics to the Build Butler CI analytics platform. It provides insights into build trends, test pass rates, and runner fleet health by integrating seamlessly into your GitHub workflows. This action helps teams monitor CI performance, identify bottlenecks, and maintain robust testing and deployment pipelines.
