## Action Summary

Aguara is a security scanning tool designed to analyze AI agent skills and MCP server configurations for vulnerabilities such as prompt injections, data exfiltration, and supply-chain attacks. It automates static code analysis using over 173 rules to detect potential threats before deployment, eliminating the need for external APIs or cloud services. Key features include confidence scoring for prioritizing findings, advanced detection of obfuscated attacks, CI integration, and extensibility through custom rules.
