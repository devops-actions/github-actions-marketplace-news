## Action Summary

Iara is an AI-powered code review tool designed for seamless integration into CI/CD pipelines or local environments. It automates static code analysis, linting, and security checks (SAST) using configurable rules and large language models (LLMs) to detect issues like security vulnerabilities, code inefficiencies, and poor practices across multiple tech stacks and languages. With native GitHub/GitLab support, multilingual output, and smart fallback to free LLM models, Iara simplifies and enhances code review processes, reducing manual effort and improving code quality.
