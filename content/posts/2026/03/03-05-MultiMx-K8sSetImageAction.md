## Action Summary

The `K8sSetImageAction` GitHub Action automates the process of updating the container image for a Kubernetes workload, such as a deployment, in a specified cluster. It simplifies managing Kubernetes deployments by allowing users to patch workloads with a new image and optionally wait for the workload to be ready, reducing manual intervention and ensuring smoother updates. Key features include support for multiple authentication methods (e.g., server/token or kubeconfig), workload readiness checks, and customizable patching options.
