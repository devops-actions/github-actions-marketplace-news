## Action Summary

Prly is a GitHub Action and command-line tool designed to automate the creation of AI-powered summaries for merged pull requests in any GitHub repository. It fetches PR data, optionally filters it by modules you own, and generates structured summaries (Problem/Change/Result) using OpenAI, which can be delivered via email, Microsoft Teams, or webhooks. This tool streamlines status updates and team communication, reducing manual effort in summarizing changes.
