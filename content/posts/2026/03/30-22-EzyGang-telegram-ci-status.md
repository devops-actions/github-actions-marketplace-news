## Action Summary

The **Telegram CI Status** GitHub Action automates sending real-time notifications to Telegram for CI/CD workflows, providing updates at the start and completion of jobs using pre- and post-hooks. It simplifies workflow monitoring by automatically detecting deployments, handling job statuses (success, failure, or canceled), and optionally pinning successful deployment messages. This action is designed for seamless integration with minimal configuration, offering rich HTML-formatted notifications for enhanced visibility.
