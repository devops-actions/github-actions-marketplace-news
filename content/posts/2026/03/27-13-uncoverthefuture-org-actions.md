## Action Summary

This GitHub Actions collection provides tools for automating the deployment of containerized applications to remote Linux hosts over SSH, leveraging Podman. Its primary functionality includes building and pushing container images, extracting and validating version information, and performing SSH-based container deployments with Traefik or host port configurations as defaults. The actions simplify and streamline container deployment workflows, particularly for teams managing remote infrastructure.
