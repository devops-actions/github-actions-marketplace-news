## Action Summary

The **Agent-BOM GitHub Action** is a security scanner designed for AI infrastructure and supply chains. It automates the discovery, scanning, and analysis of AI agents, dependencies, and MCP server configurations, identifying vulnerabilities (CVEs), credentials exposure, and security risks while providing real-time enforcement through behavioral detection and compliance frameworks. This tool helps organizations proactively secure their AI systems by mapping potential attack blast radii and implementing runtime protections.
