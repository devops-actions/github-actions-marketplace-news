## Action Summary

The **HTML Link Processor** GitHub Action automates the post-processing of static HTML sites to enhance link security, improve privacy, and ensure link integrity. It adds attributes like `nofollow`, `noreferrer`, and `noopener` to external links, removes tracking parameters, and verifies the functionality of external links by checking for broken or blocked URLs. This action simplifies the process of ensuring safer, cleaner, and more reliable links in static websites.
