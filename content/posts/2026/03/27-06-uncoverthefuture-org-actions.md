## Action Summary

This GitHub Action repository provides a collection of reusable actions for automating the deployment of containerized applications to remote Linux hosts over SSH, with a focus on using Podman. It simplifies tasks such as container deployment (defaulting to Traefik or host ports), building and pushing container images, and extracting versions for workflows, making it an efficient solution for managing SSH-based container deployments and related workflows.
