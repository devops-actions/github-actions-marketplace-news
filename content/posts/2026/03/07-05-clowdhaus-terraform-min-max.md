## Action Summary

The `terraform-min-max` GitHub Action extracts the minimum and maximum versions of Terraform permitted in a project. It automates the process of identifying these version constraints, which can then be used in workflows such as conditional pipelines or version-specific validations. This action simplifies managing Terraform version compatibility and ensures consistent version enforcement across projects.
