## Action Summary

ZIRAN is a GitHub Action designed to automate security testing for AI agents, focusing on identifying vulnerabilities in agents that utilize tools, memory, and multi-step reasoning. It solves the problem of testing complex tool combinations and multi-agent interactions, which traditional security tools often overlook, by leveraging capabilities such as graph-based tool chain discovery, side-effect detection, and autonomous pentesting. Key features include multi-phase attack campaigns, knowledge graph tracking, and protocol support for advanced agent coordination.
