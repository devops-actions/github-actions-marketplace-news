## Action Summary

Totem is a GitHub Action and CLI tool that automates the enforcement of architectural rules and coding standards across projects, ensuring consistency in code quality and governance. It provides a persistent, model-agnostic context layer that integrates with AI coding agents (e.g., Copilot, Claude) and operates independently of any single AI session. Key features include deterministic CI guardrails, cross-repository shared knowledge, and offline operation, solving the challenge of maintaining consistent development practices and reducing repeated instructions to AI agents.
