## Action Summary

Drift Guard is a GitHub Action designed to detect and classify breaking and non-breaking changes in API contracts for OpenAPI, GraphQL, and gRPC schemas. It automates the process of comparing API specifications across branches, helping developers maintain compatibility and ensure stability during API updates. This tool streamlines API change management by providing clear insights into the impact of schema modifications.
