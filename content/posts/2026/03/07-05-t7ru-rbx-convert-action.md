## Action Summary

The `rbx-convert-action` GitHub Action automates the conversion of Roblox place and model files between XML (`.rbxlx`, `.rbxmx`) and binary (`.rbxl`, `.rbxm`) formats using the `rbx-util` tool. It simplifies workflows by enabling seamless format transformations directly within a CI/CD pipeline, saving time and effort for developers managing Roblox assets. The action supports customizable verbosity and versioning of the `rbx-util` dependency for flexibility.
