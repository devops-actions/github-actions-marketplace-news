## Action Summary

Hwaro is a lightweight and fast static site generator built with Crystal, designed to simplify the creation of high-performance websites using Markdown and Jinja2-compatible templates. It automates tasks such as content processing, template management, and the generation of essential web files like sitemaps and RSS feeds, while supporting features like SEO metadata and build hooks for custom workflows. With a focus on speed, simplicity, and modern web standards, Hwaro is ideal for developers creating blogs, documentation, or personal project sites.
