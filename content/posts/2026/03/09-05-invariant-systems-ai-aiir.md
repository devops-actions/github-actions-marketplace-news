## Action Summary

AIIR (AI Integrity Receipts) is a GitHub Action that provides cryptographically tamper-evident receipts for tracking AI involvement in code commits. It helps organizations meet transparency, compliance, and auditing requirements, such as those under the EU AI Act, by recording whether commits were AI-generated and ensuring the integrity and provenance of code changes. Key capabilities include automated receipt generation per commit, cryptographic hashing for tamper detection, and optional integration with Sigstore signing for authenticity.
