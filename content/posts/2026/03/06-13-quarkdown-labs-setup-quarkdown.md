## Action Summary

The `setup-quarkdown` GitHub Action automates the installation of the Quarkdown CLI, enabling users to easily integrate Quarkdown into their workflows for tasks like document generation and conversion. It simplifies setup by supporting automatic installation of the latest stable release or a specified version, and is compatible across Linux, macOS, and Windows platforms.

## Release notes
