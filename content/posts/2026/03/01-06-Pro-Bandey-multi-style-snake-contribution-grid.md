## Action Summary

This GitHub Action generates a dynamic, animated visualization of your GitHub contribution graph in multiple styles, including different themes, shapes, and formats (GIF and SVG). It automates the process of creating visually appealing contribution animations, complete with bold month labels and a gallery of assets for easy preview, making it ideal for enhancing personal profile READMEs or sharing on social media. The action also includes features like auto-detection of the repository owner and streamlined output generation in a dedicated branch.
