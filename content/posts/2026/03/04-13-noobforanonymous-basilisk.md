## Action Summary

Basilisk is an advanced security framework designed for red teaming AI and LLM applications, offering comprehensive OWASP LLM Top 10 attack coverage. It uses a genetic prompt evolution engine to automate the discovery of novel security bypasses, enabling offensive security testing beyond static tools. The action streamlines tasks like model vulnerability assessments, response divergence detection, and guardrail posture scanning, helping teams identify weaknesses and improve AI system defenses efficiently.
