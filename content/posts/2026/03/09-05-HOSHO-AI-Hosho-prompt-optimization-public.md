## Action Summary

The Hosho Prompt Reviewer GitHub Action evaluates AI agent prompts against research-backed prompt engineering best practices, ensuring they are well-structured, goal-oriented, and optimized for models like GPT, Claude, and Gemini. It automates the review process by identifying weaknesses, providing detailed feedback with suggested fixes, and offering ratings across six key quality factors. This action streamlines prompt optimization, improving clarity, effectiveness, and adherence to model-specific guidelines.
