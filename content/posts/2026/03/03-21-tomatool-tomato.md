## Action Summary

The "Tomato" GitHub Action is a language-agnostic behavioral testing framework designed to automate the management of test infrastructure. It simplifies behavioral-driven development (BDD) by supporting Gherkin syntax for writing tests, automatically orchestrating test containers, ensuring clean state testing, and providing support for various resource types like HTTP, databases, and messaging systems. This tool streamlines the testing process by automating infrastructure setup, maintaining test isolation, and generating synchronized documentation.
