## Action Summary

ZIRAN is a GitHub Action designed for security testing of AI agents, including those with tools, memory, and multi-step reasoning capabilities. It identifies vulnerabilities by analyzing complex attack paths, such as dangerous tool combinations, side effects, and multi-phase campaigns, which are often overlooked by traditional security tools. Key features include graph-based toolchain discovery, autonomous pentesting, multi-agent coordination, and support for advanced protocols like A2A and MCP, making it a comprehensive solution for AI agent security assessment.
