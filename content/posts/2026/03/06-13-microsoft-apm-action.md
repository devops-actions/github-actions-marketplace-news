## Action Summary

The APM Action is a GitHub Action designed to automate the installation and deployment of agent primitives (such as instructions, prompts, skills, and agents) using the APM (Agent Package Manager) CLI within CI workflows. It simplifies the process by reading configuration from an `apm.yml` file or supporting isolated mode for inline dependencies, enabling streamlined management of agent-related resources with minimal setup. Key capabilities include version pinning, custom working directories, and optional generation of documentation (e.g., `AGENTS.md`) after installation.
