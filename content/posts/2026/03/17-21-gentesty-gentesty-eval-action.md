## Action Summary

The **gentesty-evaluation-action** GitHub Action automates the process of triggering evaluation runs on the Gentesty platform, streaming real-time logs via WebSocket, and capturing the final test results (success/failure) directly within GitHub Actions workflows. It simplifies integration with Gentesty by handling secure communication and providing immediate feedback on test execution, enabling efficient test automation and monitoring.
