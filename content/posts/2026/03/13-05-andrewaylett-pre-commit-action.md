## Action Summary

The `andrewaylett/pre-commit-action` GitHub Action automates the execution of [pre-commit](https://pre-commit.com) hooks as part of your CI/CD workflows. It simplifies code quality checks by running pre-configured hooks against your codebase during events like pushes or pull requests. This action supports custom hook invocations and ensures consistent code validation, reducing manual effort and improving code quality control.
