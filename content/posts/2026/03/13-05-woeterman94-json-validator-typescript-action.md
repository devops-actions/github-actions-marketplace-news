## Action Summary

The JSON Validator TypeScript Action is a GitHub Action designed to automate the validation of JSON files in a repository, ensuring correct syntax and optional compliance with JSON Schema. It simplifies the process of detecting invalid JSON files, supports automatic schema detection via the `$schema` property, and provides configurable options for folder scanning, ignore patterns, and error handling. This action is ideal for maintaining data integrity in projects that rely heavily on JSON files.
