## Action Summary

**agent-bom** is a GitHub Action and CLI tool designed to analyze and map the security posture of AI agents, MCP servers, and their dependencies, focusing on identifying vulnerabilities (e.g., CVEs), exposed credentials, and at-risk tools. It automates the discovery of AI-related components, detects potential security issues across software supply chains, container images, cloud infrastructure, and IaC files, and provides actionable insights to mitigate risks. Its key capability lies in linking vulnerabilities to their real-world impact by tracing the blast radius from dependencies to credentials and tools.
