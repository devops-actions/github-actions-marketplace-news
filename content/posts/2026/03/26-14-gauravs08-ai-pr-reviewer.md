## Action Summary

The **AI PR Reviewer** is a GitHub Action that automates pull request and merge request code reviews using AI (Claude). It works with GitHub, GitLab, and Bitbucket to analyze code changes, identify issues related to code quality, security vulnerabilities, and cross-file impacts, and automatically posts inline comments and summary feedback on PRs/MRs. This action streamlines the code review process by providing intelligent, configurable insights, reducing manual effort, and ensuring consistent review quality.
