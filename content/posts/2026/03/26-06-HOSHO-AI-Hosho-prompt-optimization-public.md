## Action Summary

Hosho Prompt Reviewer is a GitHub Action that evaluates AI agent prompts against research-backed best practices to improve their effectiveness. It automates prompt reviews by analyzing six quality factors, providing actionable feedback, and suggesting specific fixes with examples. The tool streamlines the prompt refinement process, ensuring alignment with guidelines from models like GPT, Claude, and Gemini.
