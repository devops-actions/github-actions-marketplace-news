## Action Summary

AI Code Reviewer is a GitHub Action that automates the code review process by leveraging AI models from OpenAI, Anthropic, or Google Gemini to provide intelligent feedback and improvement suggestions on pull requests. It helps developers save time and enhance code quality by generating actionable comments and filtering out irrelevant files based on specified patterns. The action is customizable, supporting different AI providers, models, and optional configurations like custom API endpoints.
