## Action Summary

The "GitHub Spam Account Blocker" is a tool and GitHub Action designed to detect and manage suspicious accounts in a user's followers/following list. It automates spam detection by analyzing account behaviors based on customizable sensitivity levels and provides a review-first workflow for manually approving or rejecting blocking actions, ensuring transparency and control. Key capabilities include explainable detection reasons, progress tracking, and optional scheduled or manual execution of blocking operations via a reusable GitHub Action.
