## Action Summary

This GitHub Action, **Lint i18n**, automates the validation of flat-key JSON files used for internationalization (i18n). It identifies issues such as namespace conflicts, invalid value types, and malformed JSON structures while providing precise line-number error annotations directly in pull request reviews. By recursively scanning locale directories with zero configuration, it ensures the structural integrity of translation files and helps prevent runtime errors in i18n libraries.
