## Action Summary

The Janitor is a structural firewall for AI-generated and human-written code, designed to enforce code integrity and quality at the pull request stage. It automates the detection and removal of unused code, zombie dependencies, duplicate logic, and structural vulnerabilities using a fast, memory-efficient analysis pipeline. By running on local hardware without external network dependencies, it provides a secure, zero-copy enforcement layer to prevent risky code from being merged, enhancing software security and maintainability.
