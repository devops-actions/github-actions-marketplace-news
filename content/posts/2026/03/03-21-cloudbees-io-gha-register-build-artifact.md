## Action Summary

This GitHub Action enables artifact traceability by registering build artifact details, such as name, version, and associated metadata, in the CloudBees platform. It automates the process of informing the platform about newly created artifact versions and their related data, facilitating improved tracking and integration of build results with the CloudBees ecosystem. Key capabilities include support for OpenID Connect (OIDC) authentication, artifact metadata reporting (e.g., digest, labels, commit information), and seamless integration with GitHub workflows.

## Release notes

Inform the CloudBees platform that an artifact version has been created, and report build artifact data generated by a GHA workflow run.
