## Action Summary

SSG is a Go-based static site generator optimized for converting WordPress exports (Markdown with YAML frontmatter) into fast, static websites. It automates tasks like site generation, asset bundling (e.g., minifying HTML/CSS/JS), image optimization (e.g., WebP conversion), and live reloading during development. Key features include support for multiple template engines, Hugo themes, Docker compatibility, and seamless integration with CI/CD workflows, making it an efficient tool for building blogs, documentation, portfolios, and other static sites.
