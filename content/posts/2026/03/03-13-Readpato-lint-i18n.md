## Action Summary

This GitHub Action, **Lint i18n**, provides structural linting for flat-key JSON files used in internationalization (i18n). It automates the detection of common errors such as namespace conflicts, invalid value types, and malformed JSON, ensuring consistency and compatibility in locale files. Key capabilities include recursive file scanning, inline error annotations in pull requests, and precise line-number identification of issues, streamlining the validation process for i18n workflows.
