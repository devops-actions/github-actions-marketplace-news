## Action Summary

This GitHub Action, **Upload TestFlight Release Notes**, automates the process of uploading release notes to a specific build on Apple's TestFlight platform. It simplifies the management and updating of release notes for iOS, macOS, tvOS, and visionOS builds, enhancing workflow efficiency by integrating directly into CI/CD pipelines. Key capabilities include support for multiple platforms, configurable retry attempts, and seamless integration with App Store Connect API credentials.
