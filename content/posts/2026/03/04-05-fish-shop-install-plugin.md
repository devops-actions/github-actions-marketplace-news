## Action Summary

The `fish-shop/install-plugin` GitHub Action automates the installation of plugins for the Fish shell using supported plugin managers, such as Fisher and Oh My Fish. It simplifies workflows by ensuring the required plugin manager is installed (if not already present) and securely handles plugin installation, mitigating risks like script injections. This action is ideal for streamlining plugin setup in CI/CD pipelines using Fish shell.
