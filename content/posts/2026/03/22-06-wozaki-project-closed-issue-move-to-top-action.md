## Action Summary

This GitHub Action automatically moves closed issues to the top of a specified column in GitHub Project V2 boards, ensuring that recently closed issues are prominently displayed. It streamlines project management by updating the issue's status and position upon closure, saving time and maintaining visibility for completed tasks. Key capabilities include support for multiple projects and customizable column targets.
