## Action Summary

This GitHub Action automates the restoration of outdated pull request (PR) review threads by re-anchoring unresolved comments to the current diff while preserving the thread structure, original author attribution, and contextual information. It solves the problem of losing visibility on unresolved feedback when GitHub marks review threads as outdated due to changes in the reviewed lines. Key capabilities include recreating the root comment and replies, linking to the original comment, and optionally including the original diff hunk.
