## Action Summary

Agent-BOM is an open-source security scanner designed to automate the discovery, assessment, and compliance enforcement of AI agent infrastructure. It identifies vulnerabilities, scans dependencies, maps potential blast radius, detects configuration issues, and enforces security standards across environments such as Docker, Kubernetes, multi-cloud setups, and AI models. This tool simplifies complex security tasks by providing enriched vulnerability data, compliance checks, and detection capabilities for tool poisoning and infrastructure risks.
