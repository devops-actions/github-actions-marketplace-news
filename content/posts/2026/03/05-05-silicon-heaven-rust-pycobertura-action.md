## Action Summary

The **Rust pycobertura GitHub Action** automates the process of generating and comparing code coverage reports for Rust projects. It runs tests on the current and previous versions of the repository, produces HTML reports, and adds sticky comments to provide insights into code coverage changes. This action helps teams track and improve test coverage efficiently by offering a clear comparison of coverage metrics over time.
