## Action Summary

The RunWatch GitHub Reporter is a GitHub Action designed to track and report the total cost of CI pipeline failures, including runner minutes, developer time, and shipping delays. It automates the collection of metrics on compute costs, failure patterns, and pipeline efficiency, sending the data to RunWatch dashboards for insights, trends, and alerts. This helps teams identify inefficiencies, reduce wasted resources, and improve deployment reliability.
