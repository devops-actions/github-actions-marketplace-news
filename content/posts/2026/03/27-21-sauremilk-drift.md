## Action Summary

Drift is a deterministic static analysis tool designed to detect architectural drift in AI-accelerated Python codebases. It identifies cross-file coherence issues such as pattern fragmentation, architecture violations, and structural hotspots, providing actionable insights to prevent long-term erosion of code quality. Drift is particularly useful for fast-growing teams leveraging AI coding tools, offering structural feedback to maintain architectural consistency and improve maintainability.
