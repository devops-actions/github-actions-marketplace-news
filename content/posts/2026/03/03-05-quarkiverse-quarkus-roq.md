## Action Summary

Quarkus Roq is a Java-based static site generator powered by the Quarkus framework, designed to simplify the creation of websites and blogs. It automates tasks like transforming content from JSON/YAML files and Markdown/Asciidoc/HTML pages into static websites with type safety and SEO-friendly metadata. Key features include tools for managing structured content, leveraging FrontMatter headers, and generating static files from Quarkus web applications.
