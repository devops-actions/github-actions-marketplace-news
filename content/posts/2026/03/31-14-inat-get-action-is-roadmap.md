## Action Summary

This GitHub Action automates the creation of interactive visual roadmaps using Mermaid flowcharts based on GitHub project data, including milestones, issues, and dependencies. It helps teams track project timelines, visualize issue status and blocking relationships, and provides flexible output options (e.g., repository files or GitHub Wiki). Key capabilities include customizable styling, issue filtering, and milestone grouping for enhanced project management and collaboration.
