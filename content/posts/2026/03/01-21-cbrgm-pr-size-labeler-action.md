## Action Summary

The PR Size Labeler GitHub Action automatically labels pull requests based on the size of changes, such as the number of lines of code or files modified. It streamlines code review processes by providing clear size-based labels, helping teams prioritize and manage pull requests more effectively. This action supports configurable thresholds and integrates with both GitHub and GitHub Enterprise.
