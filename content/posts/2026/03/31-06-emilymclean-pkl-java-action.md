## Action Summary

The **pkl-java-action** automates the generation of Java source code from Pkl model files, streamlining the process of creating Java bindings for Pkl-based data models. It solves the problem of manual code generation, ensuring consistency and reducing development effort when working with Pkl files in Java projects. This action provides capabilities for specifying input files, output destinations, and the desired Pkl version for code generation.
