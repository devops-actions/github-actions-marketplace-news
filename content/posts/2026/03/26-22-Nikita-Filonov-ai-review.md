## Action Summary

**AI Review** is an AI-powered code review tool designed to automate and enhance the code review process by providing inline comments, summary reviews, and AI-generated replies within pull requests. It integrates seamlessly with popular version control systems like GitHub, GitLab, and Bitbucket, and supports multiple large language model (LLM) providers for flexibility. Key features include customizable prompts, an agent mode for deeper repository exploration, and fully client-side operation to ensure privacy, making it ideal for improving code quality, enforcing consistency, and accelerating reviews.
