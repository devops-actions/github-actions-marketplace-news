## Action Summary

Jira Action Man is a GitHub Action designed to automatically extract Jira issue keys from GitHub events (e.g., branch names, PR titles, commit messages, or PR body) and post pull request descriptions as comments on associated Jira tickets. This action streamlines collaboration by linking GitHub activity with Jira issues, offering features like configurable project filters, blocklists, and customizable regex patterns for key extraction. It also supports deduplication of comments and various comment update modes to maintain Jira ticket updates effectively.
