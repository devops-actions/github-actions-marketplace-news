## Action Summary

The TMAS Scan Action is a GitHub Action that automates the process of scanning workspace artifacts, such as repositories, files, directories, or container images, for open-source vulnerabilities, malware, and secrets using the TrendAI™ Artifact Scanner (TMAS). It streamlines security checks by installing the TMAS CLI on the runner, performing scans, and providing detailed findings in action logs and summary reports as comments on related pull requests. This action helps developers improve security and compliance during the CI/CD pipeline.
