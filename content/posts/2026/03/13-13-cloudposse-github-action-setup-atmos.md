## Action Summary

This GitHub Action sets up and installs a specified version of [atmos](https://github.com/cloudposse/atmos), a tool for managing infrastructure and configurations, within a CI/CD workflow. It simplifies the process by adding atmos to the `PATH` and optionally provides a wrapper script to capture command outputs (`stdout`, `stderr`, `exitcode`), making them available for subsequent steps. This action automates the integration of atmos into GitHub Actions, streamlining infrastructure automation and management tasks.
