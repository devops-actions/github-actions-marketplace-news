## Action Summary

`skill-publish` is a GitHub Action and CLI tool designed to automate the publication of immutable, on-chain skill packages via the HOL Registry Broker using Hedera Consensus Service (HCS). By ensuring tamper-evident and reproducible skill releases, it eliminates the reliance on mutable URLs and provides precise version pinning, audit trails, and canonical references. This action streamlines tasks like validation, quoting, publishing, and output generation, ensuring secure and trustless artifact distribution.
