## Action Summary

The `gen-pr` GitHub Action and CLI tool automates the creation of pull requests by leveraging AI and Large Language Models (LLMs) to generate code directly from GitHub issues and pull requests. It simplifies the development workflow by allowing developers to focus on writing issue descriptions, while the AI handles implementation planning and coding tasks. Key capabilities include integration with multiple AI coding tools, flexible usage as a CLI or GitHub Action, and planning functionality to enhance code generation accuracy.
