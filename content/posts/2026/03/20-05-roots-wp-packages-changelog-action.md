## Action Summary

The **WP Packages Changelog Action** is a GitHub Action designed to automatically comment on pull requests with WordPress plugin changelogs when changes are detected in `composer.lock` or `composer.json` files. It simplifies tracking plugin updates by fetching changelogs from the WordPress.org API and providing warnings about unstable versions. This action streamlines the process of monitoring WordPress dependency changes and improves visibility into potential issues during development.
