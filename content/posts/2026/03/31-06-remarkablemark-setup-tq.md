## Action Summary

The `setup-tq` GitHub Action automates the installation and configuration of the `tq` command-line tool, which enables querying and extracting data from TOML files. It simplifies workflows by ensuring `tq` is readily available for tasks such as parsing or manipulating TOML-based configuration files. Key capabilities include version management and optional caching for improved performance.
