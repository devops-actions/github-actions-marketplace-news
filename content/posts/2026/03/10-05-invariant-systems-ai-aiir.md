## Action Summary

AIIR is a GitHub Action designed to generate tamper-evident receipts for Git commits, documenting AI involvement in code creation. It automates compliance with transparency requirements such as the EU AI Act and provides cryptographic audit trails for engineering leadership, auditors, and supply-chain security. Key capabilities include receipt generation, release verification against policy, and machine-readable attestations to ensure integrity and compliance in software development workflows.
