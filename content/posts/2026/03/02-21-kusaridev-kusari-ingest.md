## Action Summary

The `kusari-ingest` GitHub Action automates the ingestion of software artifacts, such as SBOMs and attestations, into the Kusari Platform as part of a CI/CD workflow. It simplifies artifact integration, enhances traceability by capturing repository metadata, and supports security checks, like blocking dependencies flagged in the Kusari Platform. This action streamlines artifact management and improves software supply chain security.
