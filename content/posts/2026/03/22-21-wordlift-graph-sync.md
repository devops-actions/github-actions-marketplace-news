## Action Summary

The **graph-sync GitHub Action** automates the process of synchronizing knowledge graphs using the `worai` CLI tool. It simplifies tasks such as installing `worai`, setting up Python Playwright with browser binaries, and executing `graph sync run` commands with specified profiles and configurations. This action streamlines graph synchronization workflows by handling dependencies, caching, and configuration management, making it ideal for managing structured data in knowledge graphs.
