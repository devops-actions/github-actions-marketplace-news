## Action Summary

The **AI Pull Request Describer** GitHub Action automates the generation of concise descriptions for pull requests by leveraging OpenAI-compatible AI models. Triggered when a pull request is labeled with **"ai-describe"**, it analyzes the changes and updates the pull request description or adds a comment, saving time and improving documentation consistency. Key capabilities include multi-model support, high-performance execution using pre-built binaries, and compatibility with various AI providers, ensuring flexibility and speed.
