## Action Summary

This GitHub Action transforms your GitHub contribution graph into a dynamic, animated visualization with five distinct styles, customizable shapes, and automated month labels. It automates the generation of high-quality SVG and GIF assets, providing a visually engaging way to showcase contribution activity. The action simplifies the process by detecting the repository owner automatically and creates an organized gallery of animations for easy sharing and integration into profile READMEs.
