## Action Summary

The Secure SBOM GitHub Action automates the signing and verification of Software Bill of Materials (SBOMs) and cryptographic digests using the SecureSBOM API. It ensures the integrity and authenticity of SBOMs in CI/CD pipelines by providing features like signing SBOMs with managed keys, verifying signed SBOMs, and signing digests when full SBOMs cannot be transmitted. The action supports CycloneDX SBOMs and is compatible with both v1 and v2 SecureSBOM APIs, offering a secure and efficient way to manage software supply chain transparency.
