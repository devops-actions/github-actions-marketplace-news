## Action Summary

The "Expand AWS IAM Wildcards" GitHub Action automatically identifies and expands IAM wildcard actions (e.g., `s3:Get*`) in pull request diffs, generating inline comments that list the specific actions matched by the wildcard along with links to relevant AWS documentation. This helps reviewers quickly understand changes to IAM policies and their security implications, streamlining the review process and improving clarity. Key features include scanning for wildcard patterns, grouping related expansions, and providing detailed, actionable insights directly in pull request comments.
