## Action Summary

This GitHub Action, "PR to Task (Agent Prompt) Action," automates the process of converting ambiguous pull requests into clear, structured task statements tailored for autonomous coding agents. It generates a detailed task outline—including problem statements, constraints, and acceptance criteria—making it easier for coding agents to understand and execute tasks with precision. This action reduces ambiguity, improves agent performance, and streamlines the collaboration between human developers and AI coding tools.
