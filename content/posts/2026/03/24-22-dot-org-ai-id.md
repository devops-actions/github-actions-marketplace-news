## Action Summary

This GitHub Action facilitates the claiming of anonymous tenants created via [headless.ly](https://headless.ly) by using GitHub identity for authentication, eliminating the need for personal access tokens (PATs) or secrets. It automates the process of tenant provisioning, claiming, and configuration management by leveraging OIDC tokens, ensuring secure and streamlined integration with GitHub workflows. Key capabilities include tenant claim validation, writing tenant configurations, and optional synchronization of agent public keys.
