## Action Summary

The **MCIX DataStage Overlay Apply GitHub Action** automates the process of applying one or more overlay directories to modify DataStage export assets. It enables users to update existing DataStage exports (zip or directory) by applying sequential overlays that can introduce modified JSON, additional assets, or value substitutions based on a properties file, producing updated assets in a specified output format. This action streamlines asset customization workflows, making it efficient to manage layered changes in a controlled and repeatable manner.
