## Action Summary

The **Veracode Connect GitHub Action** automates the integration of Veracode security scanning into your CI/CD pipeline. It supports various types of scans, including Software Composition Analysis (SCA), Infrastructure as Code (IaC), static code analysis, and pipeline scans, optionally integrating with the Bantuu baseline framework. This action simplifies security assessments by automating tasks like packaging, scanning, and reporting, while ensuring build quality by providing configurable build failure thresholds based on scan results.
