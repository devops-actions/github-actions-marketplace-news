## Action Summary

This GitHub Action automates the process of publishing Terraform or OpenTofu `plan` command outputs by displaying them as comments on pull requests, workflow summaries, and uploaded artifacts. It streamlines collaboration by centralizing critical infrastructure changes for review, ensuring visibility and traceability. Key capabilities include customizable output formatting, artifact management, and support for creating aggregated comments in pull requests.
