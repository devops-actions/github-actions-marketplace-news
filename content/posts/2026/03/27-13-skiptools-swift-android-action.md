## Action Summary

The **Swift Android Native Cross-Compiler and Test Runner Action** enables developers to build and test Swift packages on Android emulators using GitHub Actions. It automates the cross-compilation of Swift code for Android and executes Swift Package Manager (SwiftPM) tests in an Android environment, streamlining the CI pipeline for projects targeting both iOS and Android platforms. Key capabilities include support for different Swift and Android NDK versions, customizable build configurations, and integration with official Swift SDKs for Android.
