## Action Summary

Iara is an AI-powered, configurable code review tool designed to automate code analysis in CI/CD pipelines or locally via CLI. It integrates with multiple LLM providers to perform static analysis, linting, and security-focused reviews, detecting issues like dangerous coding patterns, vulnerabilities, and code quality problems. Its key capabilities include multi-platform integration (GitHub/GitLab), multilingual output, and support for rules-based and intelligent AI-driven analysis.
