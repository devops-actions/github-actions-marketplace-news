## Action Summary

The **Baidu Netdisk Upload Action** is a cross-platform GitHub Action designed to automate the process of uploading files to Baidu Netdisk using the BaiduPCS-Go tool. It simplifies managing file uploads by supporting glob patterns, specifying target directories, and offering configurable upload policies (e.g., skip, overwrite, rsync). This action streamlines workflows that require storing artifacts or backups in Baidu Netdisk, saving time and effort.
