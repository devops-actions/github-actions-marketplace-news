## Action Summary

The `mzap` GitHub Action automates multi-target web application security scanning using OWASP ZAP, distributing scan tasks across one or more ZAP API hosts with round-robin scheduling. It supports various scan types, provides optional wait-and-poll functionality for scan completion, and generates reports in HTML or PDF formats. This action simplifies the process of managing concurrent security scans and streamlines report generation for enhanced efficiency in vulnerability assessment workflows.
