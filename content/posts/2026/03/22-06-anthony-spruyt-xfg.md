## Action Summary

The **xfg** GitHub Action enables users to manage files, repository settings, and rulesets across GitHub, Azure DevOps, and GitLab in a declarative manner using a single YAML configuration file. It automates tasks such as syncing configuration files, applying repository standards, enforcing security policies, and creating, forking, or migrating repositories. This action streamlines the process of standardizing and maintaining organizational repository configurations at scale.
