## Action Summary

This GitHub Action, **Azure Kubernetes Service set context**, is designed to configure the Kubernetes cluster context for Azure Kubernetes Service (AKS) in GitHub workflows. It automates the setup of credentials and access to an AKS cluster, enabling subsequent actions such as deployments, secret management, or custom `kubectl` commands. Key features include support for both admin and non-admin credentials, integration with `kubelogin` for non-admin access, and compatibility with private and public clusters.
