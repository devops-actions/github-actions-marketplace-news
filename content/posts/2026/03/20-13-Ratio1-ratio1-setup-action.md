## Action Summary

The `ratio1-setup-action` GitHub Action automates the setup of the Ratio1 SDK runtime in CI environments by installing Python, configuring the SDK, and resetting its configuration. It simplifies tasks such as selecting the appropriate network, setting aliases, managing private keys, and executing post-setup commands, reducing manual configuration overhead. This action is particularly useful for streamlining workflows that depend on the Ratio1 SDK in continuous integration pipelines.
