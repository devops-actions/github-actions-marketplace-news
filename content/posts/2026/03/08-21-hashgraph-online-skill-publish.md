## Action Summary

`skill-publish` is a GitHub Action and CLI tool designed to automate the process of publishing immutable, on-chain skill releases via the HOL Registry Broker. It ensures trustless and reproducible distribution by recording releases on Hedera (HCS) and providing tamper-evident, version-pinned references. This action streamlines validation, packaging, and publishing workflows while offering an audit trail and secure artifact management.
