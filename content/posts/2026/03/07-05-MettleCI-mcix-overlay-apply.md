## Action Summary

This GitHub Action, **MCIX DataStage Overlay Apply**, automates the process of applying one or more overlay directories to DataStage export assets. It modifies an existing DataStage export (zip or directory) by incorporating overlays that can include updated JSON, additional assets, or properties-based transformations, and outputs the updated assets to a new zip or directory. This action streamlines workflows by enabling automated customization and transformation of DataStage assets within CI/CD pipelines.
