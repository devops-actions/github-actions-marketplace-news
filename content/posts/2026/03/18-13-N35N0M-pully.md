## Action Summary

Pully is a GitHub Action designed to streamline communication about pull request (PR) statuses by automatically posting and updating concise Slack messages in a dedicated channel. It provides real-time updates on PR details (e.g., title, size, author, approvals, and review statuses) in a single, editable message, eliminating the need for manual tracking or external services. This action enhances visibility and collaboration in high-paced development environments by offering a clean, centralized overview of PR progress.
