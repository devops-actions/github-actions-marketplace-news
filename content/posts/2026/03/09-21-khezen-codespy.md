CodeSpy is an open-source AI-powered code review tool designed to enhance code quality, detect bugs, and identify security vulnerabilities. It integrates seamlessly into GitHub and GitLab PR workflows, providing transparent reasoning, self-hosting capabilities, and configurable review rules to maintain control and security over sensitive codebases. Key features include intelligent scope detection, cost tracking for LLM usage, local review capabilities, and extensibility for integration with IDEs and various AI models.
