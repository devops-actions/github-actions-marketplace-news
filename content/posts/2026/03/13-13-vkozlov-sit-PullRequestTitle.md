## Action Summary

This GitHub Action automates the normalization of pull request titles to match a Jira key format and appends a corresponding Jira issue link to the PR description. It ensures consistency in PR naming conventions, simplifies the tracking of Jira issues, and provides key outputs (e.g., normalized title and Jira URL) for use in downstream workflows. This action streamlines collaboration between GitHub and Jira by automating manual tasks.
