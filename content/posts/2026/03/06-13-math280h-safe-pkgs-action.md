## Action Summary

The `safe-pkgs-action` GitHub Action automates dependency auditing in CI workflows by analyzing a project's lockfile to identify denied packages based on specified severity thresholds. It provides actionable feedback through GitHub annotations, a job summary, outputs for downstream steps, and can fail workflows when high-risk packages are detected. This action streamlines dependency security auditing, ensuring projects adhere to risk policies.
