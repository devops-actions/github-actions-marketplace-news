## Action Summary

The `load-used-actions` GitHub Action automates the process of retrieving all GitHub Actions used across workflows in an organization's repositories by utilizing the GitHub REST API. It outputs the collected data as a JSON file, which can be used for tasks such as populating an internal actions marketplace or auditing. This action streamlines the management and visibility of actions used within an organization, supporting both Personal Access Tokens (PAT) and GitHub App tokens for authentication.
