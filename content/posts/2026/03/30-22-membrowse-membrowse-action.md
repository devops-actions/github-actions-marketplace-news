## Action Summary

MemBrowse is a GitHub Action designed to analyze the binary size and memory footprint of embedded firmware by extracting detailed memory information from ELF files and linker scripts, mapping symbols to source files, and supporting multiple architectures. It automates memory usage analysis in CI/CD pipelines, providing symbol-level insights, region utilization changes, and historical tracking via integration with the MemBrowse cloud platform. Key capabilities include PR-based memory utilization reports, budget alerts, and onboarding historical build data for comprehensive memory monitoring.
