## Action Summary

The **xfg GitHub Action** enables users to declaratively manage files, repository settings, rulesets, and workflows across GitHub, GitLab, and Azure DevOps using a single YAML configuration file. It automates tasks such as syncing configuration files, applying repository standards, creating or migrating repositories, and setting branch protection rules, making it easier to enforce organizational consistency and streamline DevOps workflows. Key capabilities include generating pull requests for configuration updates, managing repository settings via APIs, and supporting cross-platform repository management.
