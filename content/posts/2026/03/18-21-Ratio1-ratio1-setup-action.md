## Action Summary

The **`ratio1-setup-action`** GitHub Action automates the setup of the Ratio1 SDK runtime in CI environments. It installs Python and the Ratio1 SDK (latest or specified version), configures the network (e.g., testnet, mainnet), and provides options to set aliases, configure private keys, and run custom commands post-setup. This action streamlines the initialization process for developers using the Ratio1 SDK, ensuring consistent and secure configurations across CI workflows.
