## Action Summary

The `fish-shop/indent-check` GitHub Action automates the process of verifying indentation in `.fish` shell files within a repository. It checks files against specified patterns and provides a summary of results, including the number of files checked, passed, and failed. This action helps ensure consistent code formatting and eliminates manual checks for indentation issues in fish shell scripts.
