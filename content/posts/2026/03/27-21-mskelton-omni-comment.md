## Action Summary

The GitHub Omni Comment Action automates the consolidation of multiple CI outputs into a single structured comment on a pull request. It allows users to organize information such as test results, deployment previews, and performance statistics into predefined sections within one comment, reducing noise and improving clarity in PR discussions. Key features include support for dynamic content from files, custom section definitions, and flexible targeting of specific pull requests.
