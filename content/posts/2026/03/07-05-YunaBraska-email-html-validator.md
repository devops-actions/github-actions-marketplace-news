## Action Summary

The "Email HTML Validator" GitHub Action automates the validation of email newsletter templates by analyzing inline HTML, files, URLs, or stdin. It identifies supported, partially supported, or unsupported features using the "Can I Email" dataset and optionally performs accessibility audits via Playwright and axe-core. This tool streamlines email template testing by providing detailed reports in multiple formats (JSON, XML, HTML, Markdown), and supports integration into various workflows, including CLI, Java DSL, CI systems, or containerized environments.
