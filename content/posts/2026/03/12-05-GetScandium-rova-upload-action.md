## Action Summary

The **Rova Mobile Application Test Action** is a GitHub Action designed to automate the upload of Android (`.apk`) and iOS (`.ipa`) builds to the Rova Platform as part of a CI/CD pipeline. It streamlines the testing process by automatically generating AI-driven testing goals, executing tests on the uploaded builds, and reporting results directly to Pull Requests and Slack. This action eliminates the need for manual testing setup, ensuring efficient and autonomous mobile application testing.
