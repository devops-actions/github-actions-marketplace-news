## Action Summary

The **Pervaziv AI Code Review** GitHub Action automates security vulnerability detection in your repository by performing scans on every push or a defined schedule. It integrates with GitHub's Security and Actions tabs to surface findings, provides detailed reports via the Pervaziv AI console, and optionally raises pull requests with suggested fixes for identified issues. This action streamlines vulnerability management during development, helping teams proactively address security risks.
