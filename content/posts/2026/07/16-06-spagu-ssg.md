## Action Summary

This GitHub Action is a static site generator written in Go. It converts Markdown files with YAML frontmatter into a complete website, including features like sitemap, search index, and responsive images. It supports various deployment options such as Cloudflare Pages, GitHub Pages, and Netlify. The action automates the process of building websites from markdown content efficiently.
