## Action Summary

The cibuild action automates the setup and configuration of CI/CD pipelines for iOS and Android projects using YAML files. It provides an interactive wizard or auto-creation feature to generate a ready-to-use GitHub Actions workflow, which is fully non-interactive and works with AI agents and scripts. The action also supports importing existing pipeline YAML files and allows customization by editing the generated pipeline files.
