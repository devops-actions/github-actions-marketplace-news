## What's Changed

Turn the Playwright `trace.zip` your tests already produce into a watchable, shareable demo — and regenerate it on every green build, so the demo can't go stale.

- Single-file, stdlib-only CLI: `specreel.py`
- Trace → narrated HTML player, gallery, single-file bundle, optional MP4
- `recommend` — crawl a running app and scaffold Playwright flows
- GitHub Action for the freshness loop; MCP server + Claude Code skill
- Opt-in, BYO-key AI narration
