## Action Summary

The GitHub Action `skaphos/oiax@v1` automates the reconciliation of Git branches in a GitOps repository, ensuring that changes are promoted through a defined graph of environments. It validates the promotion graph, plans the actions required to reconcile it, and applies those actions by creating, updating, and closing pull requests between branches. The Action is designed for Linux runners on x64 and ARM64 platforms, with release binaries available for other platforms.
