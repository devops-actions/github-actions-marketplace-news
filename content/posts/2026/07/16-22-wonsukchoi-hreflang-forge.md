## Action Summary

**Summary:**

This GitHub Action, `hreflang-forge`, generates a sitemap with `hreflang` alternate links for every page in a Next.js project. It scans the project to identify routes and locales, then creates an XML file detailing each page's URL structure and `hreflang` tags. The action is zero-dependency and can be run using Node.js directly or through npm packages.
