## Action Summary

This GitHub Action automatically updates a specified proxy cache whenever new releases are tagged in your Go module repository. It supports both standard and submodule version tags and allows customization of the proxy configuration, import path, and Go version. The action ensures that your module is immediately available on platforms like pkg.go.dev, making it simple to integrate into continuous integration workflows.
