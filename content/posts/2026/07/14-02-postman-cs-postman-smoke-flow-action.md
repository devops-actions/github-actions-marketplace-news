## Action Summary

This GitHub Action reshapes a generated Postman Smoke collection into a curated `flow.yaml`, injecting runtime authentication for OAuth2 and API keys. It automates the process of preparing a smoke test collection for deployment, ensuring it matches a specified flow specification and handles credentials securely through Postman's gateway token. The action is part of the larger Postman API Onboarding suite and integrates with other actions like `postman-bootstrap-action` and `postman-repo-sync-action`.
