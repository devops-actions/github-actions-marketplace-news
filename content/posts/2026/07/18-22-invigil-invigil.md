## Action Summary

Invigil is a CI quality gate that evaluates an open-source project against product-quality guidelines, not just code style. It checks whether the published artifact boots quickly and legibly, ensuring cold-start users can easily use and contribute to the project. Invigil provides detailed feedback on what's wrong, why it matters, and how to fix it in CI. The tool covers gaps left by existing tools like linters and dependabot, focusing on legibility and error hygiene.
