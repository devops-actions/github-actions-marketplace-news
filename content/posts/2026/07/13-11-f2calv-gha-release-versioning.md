## Action Summary

This GitHub Action calculates and releases a semantic version of a repository using GitVersion, with options to customize the version and tagging behavior. It automatically detects the required GitVersion configuration file version and handles both v5 and v6 configurations. The action supports generating release notes from merged PRs since the last release.
