## Action Summary

This GitHub Action automates the security, publishability, runtime readiness, and trust signals of AI plugin repositories across Codex, Claude, Gemini, and OpenCode ecosystems. It emits structured reports, SARIF, policy results, and submission metadata while staying aligned with the main scanner release train. The action is designed to handle both local repository content and live network probing for `verify` mode, providing flexibility for different integration needs.
