## Action Summary

This action runs multiple linters, including Actionlint, EditorConfig Checker, Hadolint, Shellcheck, YAMLLint, and Zizmor, to analyze various files such as GitHub CI/CD workflows, EditorConfig configurations, Dockerfiles, shell scripts, YAML files, and GitHub workflows. It allows users to disable specific linters or provide custom configuration files and arguments for each linter.
