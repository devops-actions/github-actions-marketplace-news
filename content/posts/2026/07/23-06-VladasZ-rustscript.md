## Action Summary

This GitHub Action automates Rust script execution by interpreting a practical subset of the language without compiling the entire project. It supports running scripts directly as binaries or executing snippets in compiled form, validating without running, and providing a list of supported methods per receiver and engine. The action is particularly useful for small scripts that do not require compilation to run efficiently.
