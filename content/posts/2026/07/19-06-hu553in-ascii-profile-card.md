## Action Summary

This GitHub Action generates Neofetch-style SVG profile cards with daily ASCII art and live GitHub stats. It automates the creation of a dedicated branch for generated files, which can be embedded in a profile README. The action supports various card configurations, including header, section, key/value, and blank rows, allowing users to customize the appearance of their profile cards.
