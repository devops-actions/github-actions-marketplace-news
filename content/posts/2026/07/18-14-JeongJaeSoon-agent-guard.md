## Action Summary

Agent Guard is a real-time guardrail that blocks AI coding agents from accidentally exposing secrets, such as reading `.env` files or writing secret-like values. It uses gitleaks for detection and provides shell scripts for integration into the agent's tool boundary. The action helps prevent leaks before they occur by blocking sensitive file access and providing defense in depth with commit/CI scanning.
