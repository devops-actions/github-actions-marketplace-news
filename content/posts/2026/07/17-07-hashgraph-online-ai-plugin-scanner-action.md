## Action Summary

The action automates the security, publishability, runtime readiness, and trust signal verification of AI plugin repositories across Codex, Claude, Gemini, and OpenCode ecosystems. It emits structured reports, SARIF, policy results, and submission metadata while staying aligned with the main scanner release train. The default install path uses an exact PyPI release, verifies its provenance against `hol-guard`, and only then installs it. Advanced distribution paths are available when needed.
