## Action Summary

**Sentinel** is a high-performance, zero-dependency Git pre-commit secret scanner and credentials detector written in Go. It automatically blocks accidental commits of sensitive information such as API keys, SSH private keys, and cloud credentials before they enter version control. Sentinel uses a three-tier detection pipeline to efficiently scan for secrets, with support for pattern matching, entropy analysis, and context classification.
