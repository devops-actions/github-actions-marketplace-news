## Action Summary

GitHub Star Tracker is a GitHub Action that tracks the star counts of all repositories across your organization on a schedule. It generates visual reports with animated SVG charts, badges, and markdown/HTML reports to help you monitor repository popularity and growth trends. Key capabilities include automatic dark/light mode support, configurable retention, smart filtering options, stargazer tracking, email notifications, and CSV export for machine-readable data.
