## Action Summary

The GitHub Action automates the process of releasing React Native over-the-air updates using the Aether platform. It wraps the @aetherpush/cli release commands, providing features like deployment to different platforms and targeting specific binary versions. The action supports both `release` and `release-react` commands, handling various inputs such as app name, command, rollout percentage, and more, while outputting relevant metadata for further integration into CI workflows.
