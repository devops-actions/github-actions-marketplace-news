## Action Summary

The action scans Android APKs for malware and threats, using the Droidwatch platform. It uploads the build artifact to Droidwatch for analysis and fails the build if a malicious verdict is detected. The action provides outputs like verdict, risk score, and report URL. Users can configure fail-on-verbs and set a timeout for analysis.

## What's Changed

Initial Marketplace release - scan Android APKs (and iOS IPAs, static) with Droidwatch and gate builds on the verdict.
