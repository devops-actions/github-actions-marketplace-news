## Action Summary

The Agent Lint GitHub Action is a lint tool for validating Claude Code, Cursor, and Codex configurations. It implements a robust suite of 286 lint rules across 20 categories to ensure consistent and high-quality setup files. The action supports both Basic and Plugin modes and can be easily integrated into CI pipelines or used with pre-commit hooks.
