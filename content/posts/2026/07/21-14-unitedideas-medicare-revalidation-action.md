## Action Summary

The Medicare revalidation checks action in GitHub Actions automates the process of validating public enrollment evidence and revalidation due dates for a list of NPIs against the CMS Medicare Revalidation List. It supports both demo and real rosters, with options to set hard charge caps. For larger datasets, it uses Apify to handle the lookup operations and can be used as an email reminder service or scheduled comparison tool. The action provides a free two-NPI demo without a token but requires an API token for real use cases.

## What's Changed
