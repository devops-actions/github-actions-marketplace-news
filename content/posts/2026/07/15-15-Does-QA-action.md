## Action Summary

The GitHub Action triggers a test run in DoesQA and waits for it to complete. It automates CI/CD testing by providing input parameters for key data such as API keys, account IDs, and tags, and outputs the final status and report URL of the test run. The action also supports creating Check Runs with automatic or custom labels for better integration with GitHub's workflow system.
