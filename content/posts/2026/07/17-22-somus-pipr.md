## Action Summary

Pipr is a code-owned AI review action that automates structured, provider-neutral reviews across popular code hosts (GitHub, GitLab.com, Azure DevOps Services, and Bitbucket Cloud). It uses a TypeScript configuration file to define review workflows and integrates with various code hosting providers through adapters. The tool runs AI-driven reviews, validates findings, and publishes comments in GitHub pull requests.
