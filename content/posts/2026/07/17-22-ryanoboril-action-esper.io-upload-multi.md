## Action Summary

This GitHub Action automates the process of uploading one or more APK files to esper.io using their API. It requires specifying the Esper.io Enterprise ID, API Key, and tenant name, along with the folder containing APK files to upload. The action also allows for optional release comments to be applied to each uploaded APK version. The result data from the upload operation is available as an output.
