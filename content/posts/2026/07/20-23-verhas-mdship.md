## Action Summary

This GitHub Action, `mdship`, is a command-line and MCP tool for manipulating markdown files. It automates tasks such as fixing heading levels, shifting headings up or down, inserting checksums, reflowing paragraphs, breaking lines at sentence boundaries, numbering headings, adding variables, generating tables of contents, including files, rendering Mermaid diagrams, and using template placeholders. The action ensures consistent content formatting and improves readability by maintaining a hierarchical structure and variable management throughout the markdown document.

## What's Changed

Check the integrity of a markdown document, missing heading, being up to date with the checksums.
