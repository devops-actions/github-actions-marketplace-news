## Action Summary

The ODS Validate Action automates the AI code quality gate by attributing AI-generated code using `Co-Authored-By` trailers, analyzing code quality, scoring technical debt, and enforcing policy through OPA Rego. It runs on every pull request to prevent low-quality AI code from reaching production.
