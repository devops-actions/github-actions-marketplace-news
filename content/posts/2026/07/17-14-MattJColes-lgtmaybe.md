## Action Summary

The lgtmaybe GitHub Action reviews pull requests by analyzing code changes and surrounding lines from the file, detecting logic and correctness bugs, security vulnerabilities, missing or weak tests, outdated code, performance regressions, unnecessary complexity, intent, and ponytail. It uses a language model to generate inline review comments and summaries, reducing the need for static keys and improving safety.
