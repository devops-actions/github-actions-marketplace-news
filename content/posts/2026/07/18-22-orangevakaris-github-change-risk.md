## Action Summary

The GitHub Action checks for risks in changes between two commits without running the repository's code, providing deterministic risk signals and aggregate reports. It supports free and paid per-file reports, is opt-in for commenting on pull requests, and requires API access to function.

## What's Changed

Extends GitHub-hosted runner retries for transient API connectivity interruptions.
