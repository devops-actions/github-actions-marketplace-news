## Action Summary

This GitHub Action compares the versions in package.json and package-lock.json with those of Cargo.toml and pyproject.toml, ensuring they are higher than the latest published tags for each manager. It also provides a way to set custom paths for these files and handle failed checks by setting a fail_on_same_version flag.
