## Action Summary

This GitHub Action automatically translates i18n locale files in a repository using the Shipi18n platform. It supports multiple languages, placeholder preservation, and incremental translation, automating the process of maintaining internationalized content. The action also generates pull requests for review and self-corrects failed translations.
