## Action Summary

The GitHub Action automates the scanning of AI plugin repositories across Codex, Claude, Gemini, and OpenCode ecosystems. It checks for security, publishability, runtime readiness, and trust signals, emitting structured reports, SARIF, policy results, and submission metadata. The action stays aligned to the main scanner release train and can be used in workflows by specifying paths and execution modes.
