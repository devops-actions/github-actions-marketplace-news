## Action Summary

Agent Lint is a versatile linter for Claude Code, Cursor, and Codex configurations. It checks for various issues related to manifest files, hook paths, skill frontmatter, agent fields, prompt content, Claude rules, output styles, settings, hygiene, email format, user configuration, MCP server setup, Codex configuration, Cursor rules, and Cursor skills. The tool supports both Basic and Plugin modes based on the presence of `.claude-plugin/` in the repository, and it provides a GitHub Action for easy integration into CI pipelines.
