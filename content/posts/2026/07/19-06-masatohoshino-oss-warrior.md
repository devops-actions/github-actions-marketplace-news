## Action Summary

The OSS Warrior GitHub Action measures an account's OSS power level across three domains: CONTRIBUTOR, MAINTAINER, and SOLO, and displays it as a living warrior card. The action automates the process of tracking public GitHub events to generate a reproducible card without requiring sign-up or additional servers.

## What's Changed

**Respect to every OSS warrior. Every contribution counts.**

First public release. Measure any GitHub account's OSS power level across three combat domains and wear it as a living warrior card on your profile.

## Highlights
- **Warrior card** in your profile README — metal-framed (BRONZE → MYTHIC), tap to post to X with the card image unfurled via your own GitHub Pages
- **Three leagues**: CONTRIBUTOR / MAINTAINER / SOLO, one unified PR-equivalent formula
- **Capacity-anchored ranks**: Rookie → Weekend Warrior → Mainstay → Pro (*the pre-AI ceiling*) → Super Warrior → **AI Sorcerer (it's over 9000)**
- **Scouter**: scan any account from your Pages console; results reply on an issue, never touch your README
- **Honest by construction**: public events only, reproducible numbers, documented limitations, ⚡ Limit Break when the API caps out
