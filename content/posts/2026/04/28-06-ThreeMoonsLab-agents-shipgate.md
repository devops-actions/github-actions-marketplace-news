## Action Summary

Agents Shipgate is a GitHub Action and CLI tool that scans AI agent tool surfaces for release-readiness, ensuring compliance with explicit approval policies, scope coverage, idempotency evidence, and review readiness before deployment. It automates the generation of deterministic reports in Markdown, JSON, and SARIF formats, identifying potential release blockers and risks, such as missing approval policies or inadequate permission scopes. This action addresses the gap between code review, evaluation suites, and runtime observability by providing a structured pre-release validation for AI-driven tools.
