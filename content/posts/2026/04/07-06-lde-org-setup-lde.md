## Action Summary

The `setup-lde` GitHub Action is designed to automatically install and configure the `lde` tool on GitHub Actions runners, adding it to the system PATH. It simplifies the process of obtaining and managing specific `lde` versions, including stable releases, nightly builds, or the latest version, ensuring a consistent and streamlined setup for workflows.
