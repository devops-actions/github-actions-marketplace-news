## Action Summary

Verselicious is a GitHub Action that automates semantic versioning and release management by leveraging labels on pull requests. It simplifies the process of versioning by automatically detecting `major`, `minor`, or `patch` labels, incrementing the version number, creating a git tag, and generating a GitHub release with release notes upon merging a pull request. This action eliminates the manual effort of versioning and ensures consistent release practices in software projects.
