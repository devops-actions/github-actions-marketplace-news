## Action Summary

The `fish-shop/run-fishtape-tests` GitHub Action automates the execution of [Fishtape](https://github.com/jorgebucaran/fishtape) tests for scripts written in the Fish shell. It streamlines the process of identifying and running test files using customizable patterns and provides enhanced test output readability with the option for raw or prettified TAP results. This action simplifies test management for Fish shell projects, improving efficiency and consistency in CI workflows.
