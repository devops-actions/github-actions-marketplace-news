## Action Summary

The PullGuard GitHub Action automates code quality, security, and compliance analysis for pull requests by running up to 27 different analyzers (depending on the subscription tier). It helps developers identify issues like complexity, dead code, vulnerabilities (e.g., SQL injection, XSS), and compliance violations while providing actionable insights like drift scores and cost estimations. The action is designed to streamline reviews, improve code standards, and ensure secure, maintainable code without transmitting data outside the GitHub environment.

## What's Changed

Code quality, security & compliance on every PR. 27 analyzers, N-hop taint tracking, SOC 2 compliance, cost estimation. Free tier included.
