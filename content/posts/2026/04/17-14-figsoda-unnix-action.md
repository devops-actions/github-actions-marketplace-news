## Action Summary

The `unnix-action` GitHub Action automates the installation of packages using the `unnix.kdl` configuration file. It ensures the required version of the `unnix` tool is downloaded if not already present, and supports specifying custom directories and pinning the tool to a specific version. This action simplifies dependency management and streamlines workflows involving `unnix`.
