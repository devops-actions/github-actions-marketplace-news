## Action Summary

The Delimit GitHub Action automates API governance by analyzing changes to OpenAPI specifications in pull requests. It detects breaking changes, classifies semantic versioning updates, enforces custom API policies, and generates detailed migration guides, all while providing rich comments directly on PRs. This action simplifies API change management, enhances code quality, and integrates seamlessly into CI pipelines with optional signed attestations for transparency.
