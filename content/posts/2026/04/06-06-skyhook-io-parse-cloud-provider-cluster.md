## Action Summary

The "Parse Cloud Provider Cluster" GitHub Action extracts structured details (e.g., provider, account, region, cluster, and registry) from a standardized cloud provider tuple format used in Skyhook workflows. It automates the parsing of cloud configuration strings, simplifying the process of retrieving and utilizing cloud-specific information in workflows. This action helps ensure consistency and reduces manual input errors when working with multi-cloud environments.
