## Action Summary

The `aqua-installer-cache` GitHub Action streamlines CI/CD workflows by caching the [aqua](https://aquaproj.github.io/) installation directory, reducing redundant downloads of tools and mitigating GitHub API rate limit issues. It automatically detects aqua configuration files, supports multiple configurations, and creates optimized cache keys based on OS, architecture, and config file changes. This action accelerates tool setup across Linux, macOS, and Windows runners, ensuring efficient and reliable builds.
