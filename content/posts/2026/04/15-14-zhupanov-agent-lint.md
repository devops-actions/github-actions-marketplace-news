## Action Summary

Agent Lint is a GitHub Action and CLI tool for validating and linting Claude Code configurations and plugins. It automates the detection of configuration issues across `.claude/` and `.claude-plugin/` directories using 104 customizable linting rules, ensuring code hygiene, compliance, and maintainability. Key features include two linting modes (basic and plugin), auto-fix capabilities for select issues, and seamless integration into CI workflows, helping developers streamline and standardize their Claude Code setups.
