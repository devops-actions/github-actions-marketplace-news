## Action Summary

ResourcePulse — Bicep IaC Reviewer is a GitHub Action designed to analyze Azure Bicep infrastructure-as-code (IaC) files in pull requests. It automatically provides cost impact estimates, best-practice findings, and policy compliance checks by reviewing changes to Azure resources, helping teams optimize deployments and identify potential issues. This action simplifies the evaluation process, improves cost awareness, and ensures adherence to guidelines without requiring manual review or exposing sensitive source code.
