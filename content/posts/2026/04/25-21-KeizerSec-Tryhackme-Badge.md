## Action Summary

The **TryHackMe Badge** GitHub Action automates the creation and daily updating of a dynamic SVG badge displaying a user's TryHackMe stats, such as rank, rooms, badges, level, and league, by fetching data directly from the official API. It eliminates the need for manual updates, third-party rendering tools, or HTML scraping by leveraging lightweight, pure SVG generation with customizable themes and rotating visuals. This action is ideal for users looking to showcase live TryHackMe achievements in their GitHub profile or repositories.

## What's Changed
