## Action Summary

The Decodie GitHub Action automates file analysis in pull requests by leveraging the Decodie skill and Claude Code functionality. It generates structured learning entries, including patterns, rationale, and code snippets, and commits them to the `.decodie/` directory while providing a summary comment in the pull request. This action streamlines documentation and knowledge management, enhancing collaboration and code review processes.
