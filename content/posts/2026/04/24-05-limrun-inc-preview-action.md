## Action Summary

The **Limrun Preview Action** is a GitHub Action designed to build iOS apps using a temporary Limrun Xcode instance, enabling the creation of live preview links that are posted on pull requests. It automates the process of building and testing iOS apps in a browser-based iOS Simulator, eliminating the need for access to a physical Mac. This action streamlines collaboration and app preview workflows, particularly for teams without macOS environments.
