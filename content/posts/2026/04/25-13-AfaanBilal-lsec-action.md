## Action Summary

The `lsec` GitHub Action automates the process of running a security audit on Laravel projects using the `lsec` CLI tool. It identifies potential vulnerabilities across 61 rules in categories like authentication, injection, and secrets, uploads the findings to GitHub Code Scanning in SARIF format, and optionally posts a summary comment on pull requests. This action streamlines security checks, ensuring code quality and reducing the risk of deploying insecure code.
