## Action Summary

Claude Lint is a GitHub Action and CLI tool designed to validate and lint configuration files and plugins for Claude Code projects. It automates the detection of issues in `.claude/` and `.claude-plugin/` directories using 88 customizable linting rules across various categories, ensuring code quality and adherence to best practices. The tool integrates seamlessly into CI workflows, supports cross-platform environments, and offers configurable rule suppression and severity adjustments via a `claude-lint.toml` file.
