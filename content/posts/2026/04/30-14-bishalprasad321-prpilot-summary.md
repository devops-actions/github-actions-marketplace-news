## Action Summary

The "PR Pilot Summary" GitHub Action automates the generation of detailed, context-aware pull request (PR) descriptions using AI. It analyzes code diffs, commit history, and file changes to create structured summaries, including sections like Summary, Developer Notes, and Checklists, while preserving existing PR content. This action streamlines the code review process by improving PR clarity, saving developers time, and ensuring consistent and comprehensive documentation.
