## Action Summary

This GitHub Action, *tfstate loves mongo*, automates the management of Terraform state files (`.tfstate`) by using MongoDB as a backend. It simplifies workflows by automatically restoring the latest state before a Terraform run and saving updated states afterward, while retaining only the two most recent snapshots to optimize storage. This action ensures seamless state management and version control, addressing the challenges of state file storage and consistency in Terraform workflows.
