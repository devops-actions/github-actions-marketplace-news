## Action Summary

This GitHub Action streamlines pull request management by automating the creation and editing of comments within a PR. It allows teams to provide updates, such as test results, in a single, editable comment to reduce clutter and maintain a clean PR thread. Additionally, it supports adding comments from an external text file, enabling efficient presentation of structured outputs like test summaries.
