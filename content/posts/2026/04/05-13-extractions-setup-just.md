## Action Summary

The `setup-just` GitHub Action automates the installation of the [just](https://github.com/casey/just) command runner in workflows, allowing users to specify the desired version to install. It simplifies setting up `just` by handling version resolution and authentication for GitHub API requests, reducing manual setup efforts. This action is particularly useful for automating task execution in continuous integration pipelines.
