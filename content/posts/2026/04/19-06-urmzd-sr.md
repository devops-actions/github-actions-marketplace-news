## Action Summary

The GitHub Action **`sr`** automates semantic versioning and release management for projects using conventional commits. It handles tasks such as version bumping, changelog generation, GitHub releases, and tag creation, all in a single, language-agnostic static binary without requiring additional runtime dependencies. This action simplifies and streamlines the release process with zero-configuration defaults, making it an efficient tool for continuous integration and deployment workflows.
