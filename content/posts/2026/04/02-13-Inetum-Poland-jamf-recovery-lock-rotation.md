## Action Summary

The **Jamf Recovery Lock Rotation** GitHub Action automates the process of rotating Recovery Lock passphrases on Jamf Pro–managed Apple Silicon Macs using the Jamf Pro API. It enhances security by regularly updating passphrases to minimize exposure risk in case of credential compromise. Key features include OAuth-based authentication, device scoping via Smart Computer Groups, dry-run validation, and actionable outputs for integration into broader workflows.
