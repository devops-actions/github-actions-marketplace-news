## Action Summary

The `fish-shop/indent-check` GitHub Action automates the process of verifying proper indentation in `.fish` shell script files within a repository. It scans specified files or directories using customizable patterns and generates a summary of files that passed or failed the indentation check. This action helps maintain consistent code formatting, simplifying code reviews and ensuring adherence to style guidelines.
