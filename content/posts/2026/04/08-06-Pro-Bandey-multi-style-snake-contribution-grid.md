This GitHub Action automates the creation of dynamic animations based on the GitHub contribution graph, offering five distinct styles, custom shapes, bold month labels, and high-quality SVG/GIF outputs. The action streamlines the process of enhancing profile visuals and enables easy sharing of personalized contribution grid animations.
