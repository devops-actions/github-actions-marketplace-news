## Action Summary

The `update-golangci-lint-action` GitHub Action automates the process of updating `golangci/golangci-lint-action` references in GitHub workflow files and optionally synchronizes a version file used elsewhere in the repository. It simplifies maintaining consistent versions of `golangci-lint` by updating workflow configurations and version files, reducing manual effort and ensuring alignment across the project. Key features include updating action versions, syncing version files, and providing outputs to track changes made.
