## Action Summary

This GitHub Action automates the process of compiling LaTeX documents and publishing the resulting output (e.g., PDFs) to GitHub Releases. It also supports generating differential documents to highlight changes between versions using `auto-latexdiff`. This action streamlines the workflow for managing and distributing LaTeX projects directly from a GitHub repository.
