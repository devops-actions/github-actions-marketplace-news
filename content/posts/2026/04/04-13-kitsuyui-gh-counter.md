## Action Summary

`gh-counter` is a GitHub Action designed to automate the tracking and comparison of configurable code markers, such as `TODO` or `FIXME`, in pull requests and the default branch of a repository. It simplifies setup while providing functionality to generate managed pull request comments and publish JSON or badge assets for tracking code signals. This helps maintain visibility into technical debt and code annotations, making it easier to monitor and manage such markers across a codebase.
