## Action Summary

The **`setup-arctl` GitHub Action** automates the installation and verification of the `arctl` CLI tool from official GitHub releases, ensuring it is added to the system `PATH` and properly secured via SHA256 checksum validation. It simplifies the setup process for workflows that involve managing and deploying agents, prompts, and skills using `arctl`, while automatically handling necessary dependencies on Linux runners. This action helps streamline development pipelines by eliminating manual installation steps and ensuring environment consistency.
