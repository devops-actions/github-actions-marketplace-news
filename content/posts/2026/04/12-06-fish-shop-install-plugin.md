## Action Summary

The `fish-shop/install-plugin` GitHub Action automates the installation of plugins for the fish shell by supporting various plugin managers like Fisher and Oh My Fish. It simplifies the process of managing fish plugins in CI workflows by automatically installing the specified plugin manager (if not already present) and the desired plugins in a secure manner. This action is particularly useful for streamlining shell environment setup in GitHub workflows.
