## Action Summary

The `fish-shop/install-plugin` GitHub Action automates the installation of plugins for the Fish shell using supported plugin managers like Fisher and Oh My Fish. It simplifies the setup process by handling the installation of the required plugin manager (if not already present) and specified plugins. This action is ideal for streamlining workflows that rely on Fish shell plugins, ensuring efficiency and security with safe handling of inputs to mitigate script injection risks.
