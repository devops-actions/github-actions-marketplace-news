## Action Summary

The **Setup-Cangjie** GitHub Action automates the installation and setup of the Cangjie development environment for different versions (LTS, STS, Nightly) and cross-compilation targets. It simplifies the process of configuring the required SDK and dependencies, supports specifying versions and build targets, and handles authentication with the Cangjie repository if needed. This action is ideal for streamlining CI/CD workflows involving Cangjie-based projects.
