## Action Summary

The TealTiger Security Scan GitHub Action automates security and policy compliance checks for textual data in your repository. It detects issues such as PII exposure, prompt injection vulnerabilities, and content moderation violations, enabling developers to enforce security guardrails and validate policies directly within their CI/CD workflows. Key capabilities include customizable guardrail selection, sensitivity adjustment, policy file integration, and detailed reporting in various formats.

## What's Changed

Run TealTiger AI security guardrails and policy tests in your GitHub Actions workflow. Supports PII detection, prompt injection prevention, and content moderation with JSON, JUnit, and SARIF output.
