## Action Summary

The **Email HTML Validator** GitHub Action is designed to validate HTML email templates by analyzing their compatibility with email client features using the "Can I Email" dataset. It automates the detection of supported, partially supported, and unsupported features in HTML, while optionally performing accessibility audits with Playwright and axe-core. This tool provides versatile interfaces, including CLI, Java API, Docker, and GitHub Actions, and generates detailed reports in JSON, XML, HTML, or Markdown formats to streamline email template validation workflows.
