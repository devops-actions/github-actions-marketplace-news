## Action Summary

The **OctoTS Metrics Processor** GitHub Action automates the collection, processing, and storage of code metrics as a time series in a dedicated branch within a repository. By appending timestamped metrics from supported input files (e.g., CSV, JSON, Excel) to a persistent dataset, it enables streamlined tracking and versioning of metrics over time. This action is ideal for maintaining historical records of code metrics in a standardized, automated manner.
