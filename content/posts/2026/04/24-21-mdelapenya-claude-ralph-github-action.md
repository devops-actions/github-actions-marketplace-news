## Action Summary

The **Claude Ralph GitHub Action** automates iterative work/review/ship cycles on GitHub issues by leveraging the Claude Code CLI. It creates a branch, writes code to address the issue, reviews the changes, and either revises or ships the code based on feedback. It also includes an optional security audit step to ensure code quality and prevent vulnerabilities, streamlining the issue resolution and pull request creation process.
