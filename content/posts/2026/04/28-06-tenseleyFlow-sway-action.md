## Action Summary

This GitHub Action, **sway-action**, automates the execution of the `sway` tool to evaluate a PEFT adapter on every pull request. It generates a detailed report, posts it as a PR comment (with edit-in-place behavior to prevent spam), and uploads the report as a workflow artifact. By simplifying the integration of `sway` into CI workflows, it reduces onboarding friction, standardizes PR feedback formats, and handles common boilerplate tasks such as rebase handling and comment updates.
