## Action Summary

The **ATR (Agent Threat Rules)** GitHub Action provides a set of open-source, community-driven detection rules designed to identify security threats targeting AI agents, such as ChatGPT, Copilot, and other similar systems. It functions as a real-time threat detection layer, akin to antivirus signatures, to identify and mitigate attacks like data leakage, malicious command execution, and bypassing safety protocols. ATR integrates with existing security platforms, maps to OWASP Agentic Top 10 categories, and helps automate AI agent security by detecting and addressing vulnerabilities and malicious behaviors.
