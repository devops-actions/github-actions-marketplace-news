## Action Summary

The **Newma Review Action** is a GitHub Action that automates AI-powered code reviews and security audits for repositories. It provides two modes: a "diff mode" for reviewing pull request changes and a "full mode" for conducting repository-wide security scans. By identifying critical security vulnerabilities such as URL/path injection, secret leakage, and default fail-open issues, it helps developers streamline code review processes and enhance code security.
