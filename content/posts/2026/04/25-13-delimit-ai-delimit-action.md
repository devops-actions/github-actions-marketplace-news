## Action Summary

The `Delimit GitHub Action` automates API governance for OpenAPI specifications by detecting breaking changes, classifying semantic version updates, and generating migration guides. It runs on pull requests to compare API changes against the base branch, flagging policy violations and providing actionable insights directly in PR comments. Key capabilities include breaking change detection, semver recommendations, customizable governance policies, and signed, verifiable attestations for compliance tracking.
