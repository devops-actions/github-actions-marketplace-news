## Action Summary

The **uncloud-action** GitHub Action facilitates the deployment of Docker images to servers using Uncloud, automating tasks such as building, pushing, and deploying containerized applications. It simplifies deployment workflows by integrating with Docker and securely handling server access credentials. Key capabilities include image deployment, server-side configuration management, and optional image-only pushing.
