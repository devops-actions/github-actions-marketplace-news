## Action Summary

The **Report Deployment Action** is a GitHub Action that integrates with [Ewake](https://www.ewake.ai) to report deployment events, enabling correlation of deployments with incidents and providing contextual insights during investigations. It automates the process of sending deployment metadata (e.g., repository, commit SHA, timestamp, artifact name) to Ewake without disrupting workflows, even in the event of network errors or API issues. This action helps teams improve incident response by surfacing deployment-related context in their observability tools.
