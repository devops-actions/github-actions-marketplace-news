## Action Summary

The "no-autopilot" GitHub Action enforces human oversight in pull requests by detecting AI-generated content and ensuring contributors follow project guidelines. It automates checks for AI attributions in commit messages and PR descriptions, incomplete PR templates, and unchecked review checklists, with optional enforcement of an "honesty trap" to ensure genuine compliance. This action helps maintain code quality and accountability by encouraging thoughtful, human-led contributions.
