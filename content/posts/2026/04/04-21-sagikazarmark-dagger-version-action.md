## Action Summary

This GitHub Action retrieves the `engineVersion` field from a `dagger.json` file in a repository, either by reading the file directly if the repository is checked out or via the GitHub Contents API if it is not. It automates the process of identifying the required Dagger engine version for workflows, streamlining dependency management and ensuring compatibility. This action is particularly useful for setting up CI/CD pipelines that rely on Dagger.
