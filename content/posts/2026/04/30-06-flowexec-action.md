## Action Summary

This GitHub Action integrates with the [Flow](https://github.com/flowexec/flow) CLI to automate the execution of Flow workflows within GitHub Actions. It simplifies tasks like building, testing, deploying, and managing multi-workspace projects by providing a streamlined way to execute Flow commands and manage dependencies directly in CI pipelines. Key features include support for workspace management, parameterization, environment variable and secret injection, and log artifact uploads to enhance workflow automation and debugging.
