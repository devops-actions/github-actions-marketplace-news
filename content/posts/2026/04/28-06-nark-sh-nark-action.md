## Action Summary

The `nark-action` GitHub Action automates the detection of missing error handling in TypeScript projects by scanning code against predefined profiles for over 169 npm packages. It integrates the `nark` tool into CI workflows, enabling teams to identify and address error-prone library function calls effectively. This action streamlines code quality assurance by simplifying the process of ensuring robust error management.
