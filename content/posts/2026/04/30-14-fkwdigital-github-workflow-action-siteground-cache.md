## Action Summary

The **SiteGround Cache Manager GitHub Action** automates the process of purging SiteGround's dynamic NGINX cache and Memcached object cache directly from CI/CD workflows. It supports two modes: SSH for shared hosting environments and API for agency-tier accounts, enabling seamless cache clearing for WordPress sites hosted on SiteGround. This action simplifies cache management, reduces manual intervention, and ensures up-to-date content delivery during deployment pipelines.
