## Action Summary

The Lineaje Scan Action automates vulnerability scanning for container images and source code, providing a summary of detected issues and optionally generating a fix plan. It streamlines security workflows by producing patched artifacts, such as updated Dockerfiles or dependency manifests, that can be used to address vulnerabilities. This action helps developers ensure secure and up-to-date software components while reducing manual remediation efforts.
