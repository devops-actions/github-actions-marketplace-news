## Action Summary

The Delimit GitHub Action automates API governance by analyzing changes in OpenAPI specifications during pull requests. It detects breaking changes, classifies semantic versioning updates, generates migration guides, and enforces custom API policies, all while providing detailed feedback via pull request comments. This action helps teams maintain API stability and consistency, ensuring compliance with defined standards and reducing risks associated with unintentional breaking changes.
