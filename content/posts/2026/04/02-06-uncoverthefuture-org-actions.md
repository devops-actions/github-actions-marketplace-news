## Action Summary

This GitHub Action collection automates the deployment of containerized applications to remote Linux hosts over SSH, leveraging Podman. It simplifies SSH-based container deployments, including tasks like building and pushing container images, extracting versions, and managing generic container deployments (e.g., using Traefik by default). The actions streamline complex deployment workflows, making them composable and reusable for various container deployment scenarios.
