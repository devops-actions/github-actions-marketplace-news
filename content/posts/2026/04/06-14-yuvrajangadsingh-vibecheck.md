## Action Summary

Vibecheck is a GitHub Action that scans codebases for common issues associated with AI-generated code, such as security vulnerabilities, poor error handling, and low-quality code patterns. It automates the detection of these problems using a set of 34 rules, helping developers maintain cleaner, safer, and more reliable code. Key capabilities include identifying hardcoded secrets, unsafe practices (e.g., `eval`, SQL concatenation), and AI-specific patterns like redundant comments or excessive use of `any` in TypeScript.
