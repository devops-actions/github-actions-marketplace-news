## Action Summary

This GitHub Action automates the process of exporting Notion pages into Markdown files compatible with Docusaurus and Jekyll, making it easier to integrate Notion content into static site generators. It recursively converts a Notion page hierarchy into Markdown files with proper YAML frontmatter, ensuring compatibility with Docusaurus' MDX parser while preserving formatting and metadata. This action simplifies content synchronization between Notion and static websites, saving time and effort in manual export and formatting.
