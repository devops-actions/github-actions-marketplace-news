## Action Summary

This GitHub Action automates the expansion of AWS IAM wildcard actions in pull request diffs, providing inline comments that detail the specific IAM actions matched by each wildcard. It helps reviewers quickly understand changes to security posture by linking expanded actions to AWS documentation and summarizing them directly within the PR. The action streamlines security reviews by automating the identification and contextualization of wildcard usage in IAM policies.
