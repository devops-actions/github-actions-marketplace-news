## Action Summary

The `unnix-action` GitHub Action automates the installation of packages defined in a `unnix.kdl` file. It ensures that the `unnix` tool is available in the environment and allows users to specify custom file paths or pin the tool to a specific version. This action simplifies dependency management and streamlines workflows by handling tool and package setup automatically.
