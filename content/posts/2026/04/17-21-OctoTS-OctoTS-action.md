## Action Summary

The OctoTS Metrics Processor GitHub Action automates the collection, processing, and storage of code metrics into a persistent time-series dataset within a dedicated branch, such as `metrics-data`. It supports various input formats (e.g., CSV, JSON, Excel) and generates outputs in selectable storage formats while automatically appending a timestamp column for historical tracking. This action simplifies metrics management by consolidating data in a structured and version-controlled manner.
