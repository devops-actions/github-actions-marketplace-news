## Action Summary

The `speq-github-runner` GitHub Action is designed to orchestrate `speq-cli` commands in CI workflows, providing a reproducible mechanism to install, validate, run, and report tests. It simplifies test automation by offering predefined modes (`setup`, `run`, `custom`) and supports artifact uploads (e.g., test summaries, logs, and reports) for streamlined integration into CI pipelines. This action eliminates the need to reimplement test execution logic, focusing solely on managing `speq-cli` operations.
