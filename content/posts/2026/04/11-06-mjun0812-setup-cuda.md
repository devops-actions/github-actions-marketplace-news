## Action Summary

The `setup-cuda` GitHub Action automates the installation and configuration of NVIDIA CUDA on GitHub-hosted runners, supporting both Linux and Windows environments. It simplifies the setup process by allowing users to specify the desired CUDA version dynamically, supports a range of distributions, and automatically configures necessary environment variables. This action is particularly useful for workflows requiring GPU acceleration, eliminating manual setup and ensuring compatibility across different operating systems and CUDA versions.
