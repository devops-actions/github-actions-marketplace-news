## Action Summary

Aptu is an AI-powered triage utility designed to automate and improve open-source project management tasks such as issue triage, pull request reviews, and identifying good-first-issues. By leveraging small, efficient language models with optimized prompts, it delivers accurate results at lower costs and faster speeds compared to larger models. Key features include automatic labeling and commenting on GitHub issues, customizable prompts, multi-format output, and local security scanning for pull request reviews.
