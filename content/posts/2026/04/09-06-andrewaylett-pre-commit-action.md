## Action Summary

The `andrewaylett/pre-commit-action` GitHub Action automates the execution of [pre-commit](https://pre-commit.com) hooks in CI workflows. It streamlines tasks such as code linting and formatting by automatically running pre-commit hooks on code changes during pull requests or pushes. This action simplifies repository maintenance by setting up the `pre-commit` environment and optionally allowing customization of hook execution through additional arguments.
