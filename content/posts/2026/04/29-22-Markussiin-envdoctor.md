## Action Summary

EnvDoctor is a static analysis and CI tool designed to identify and diagnose issues with environment variables in modern JavaScript projects, including Node.js, Next.js, Vite, and Turborepo. It automates the detection of missing, leaked, stale, and misconfigured environment variables by scanning code and repository configurations, comparing them against `.env` files, env schemas, and framework-specific rules. Key capabilities include providing actionable diagnostics, generating fixes for `.env` files, and integrating with CI pipelines to prevent environment-related errors.
