## Action Summary

The KOREXT Enforce Action is a GitHub Action designed to automatically enforce compliance policies on AI-generated code within CI/CD workflows. It scans codebases against pre-defined policy packs and rules for multiple languages, identifies violations, and reports them as GitHub Code Scanning annotations directly on pull requests. This action helps automate code quality and security checks, ensuring adherence to regulatory and best practice standards while providing actionable insights for developers.
