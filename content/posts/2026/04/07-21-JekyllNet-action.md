## Action Summary

The **JekyllNet GitHub Action** automates the process of building Jekyll-style static sites using the `JekyllNet` .NET tool. It simplifies tasks such as installing dependencies, building the site, and optionally uploading the generated site as a workflow artifact. Key capabilities include support for caching dependencies, customizable build configurations, and integration with both NuGet and GitHub Packages for tool installation.
