## Action Summary

The **gentesty-evaluation-action** GitHub Action automates the process of triggering evaluation runs on the Gentesty platform, streaming real-time logs to GitHub Actions, and retrieving the final test results (success or failure). It simplifies the integration of Gentesty's evaluation capabilities into CI/CD workflows, providing secure, token-based access and compatibility with multiple environments.
