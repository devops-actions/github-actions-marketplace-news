## Action Summary

**Summary:**  
The `agent-bom` GitHub Action is a security scanner designed for AI supply chains, including agents, MCP servers, packages, containers, cloud environments, GPUs, and runtimes. It automates the detection and analysis of vulnerabilities (CVEs), mapping their blast radius through the AI stack to identify impacted components, exposed credentials, and tools at risk. Key capabilities include end-to-end vulnerability tracking, remediation recommendations, and prioritization to enhance security posture and mitigate risks efficiently.
