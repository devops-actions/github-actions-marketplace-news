## Action Summary

The **CI Matrix Planner** GitHub Action automates the creation and management of CI job matrices by dynamically generating jobs and targets based on inputs such as workflow dispatch events, pull request metadata, labels, and commit messages. It streamlines CI workflows by enabling fine-grained control over which jobs to run, skip, or customize, thereby improving efficiency and maintainability across projects. Key features include support for directive parsing, matrix customization, profile-driven configurations, and outputs optimized for conditional logic and matrix definitions.
