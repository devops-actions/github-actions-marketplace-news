## Action Summary

**branch-watch** is a CLI tool that streamlines the management of GitHub branches, forks, and pull requests by providing real-time sync status comparisons. It automates the identification of outdated forks, drifted feature branches, and open pull requests, eliminating the need for manual tracking via the GitHub UI. Key capabilities include tracking commit differences between forks and upstream repositories, monitoring branch alignment with a base branch, and listing open pull requests with detailed metadata directly in the terminal.
