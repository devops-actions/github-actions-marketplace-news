## Action Summary

The `actions-usage` GitHub Action provides usage metrics for GitHub Actions, allowing users to analyze and report on the consumption of Actions resources per developer across repositories or organizations. It automates the generation of detailed reports, including CI costs and usage breakdowns by developer, repository, or organization, with options for exporting data in various formats like JSON, CSV, or Markdown. This tool is particularly useful for monitoring and optimizing GitHub Actions usage, facilitating cost tracking and team accountability.
