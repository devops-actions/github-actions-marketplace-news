## Action Summary

Vorsken is a GitHub Action designed to automatically enforce API security policies on pull requests by scanning code changes for vulnerabilities using Semgrep and analyzing findings with Claude AI. It prevents insecure code from being merged by providing clear explanations of detected issues, mapping them to OWASP API Security categories, and suggesting fixes directly in PR comments. This action streamlines security reviews and enhances developer understanding of vulnerabilities, ensuring compliance with best practices without leaving GitHub.
