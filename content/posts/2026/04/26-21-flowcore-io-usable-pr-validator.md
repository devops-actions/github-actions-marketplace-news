## Action Summary

The **Usable PR Validator** GitHub Action uses AI-powered validation to analyze pull requests against an organization's coding standards and architectural guidelines stored in the Usable knowledge base. It automates the process of reviewing code for compliance by leveraging AI models (such as OpenAI, Anthropic, OpenRouter, or Google Gemini), generating detailed reports, providing smart PR comments, and allowing revalidation via comments. This action helps streamline code review workflows, enforce consistent standards, and document approved deviations automatically.
