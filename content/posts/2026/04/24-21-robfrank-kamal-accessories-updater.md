## Action Summary

The **Kamal Accessories Updater** GitHub Action automates the process of checking for and updating [Kamal](https://kamal-deploy.org/) deployment accessories to their latest versions from Docker Hub, GitHub Container Registry, and other OCI-compliant registries. It solves the problem of manually tracking and updating accessory versions by providing features like automatic version detection, semantic version comparison, SHA256 digest retrieval for added security, and optional pull request creation with update details. Its key capabilities include multi-registry support, caching for improved performance, and configurable modes for checking, interactive updating, or fully automated updates.
