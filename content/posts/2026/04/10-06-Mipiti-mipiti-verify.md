## Action Summary

Mipiti-verify is a GitHub Action designed to automate the verification of security controls and threat model assertions in CI/CD pipelines. It ensures security controls remain up-to-date by validating assertions locally or via AI-powered semantic analysis (Tier 2) using providers like OpenAI or Anthropic. Key features include batch verification, drift detection, signed report auditing, and compatibility with offline and AI-enhanced workflows.
