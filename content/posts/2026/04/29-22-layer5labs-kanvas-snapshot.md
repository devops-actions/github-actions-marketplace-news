## Action Summary

The **Kanvas Snapshot GitHub Action** automates the generation of visual infrastructure diagrams from code, such as Kubernetes manifests, Helm charts, or Docker Compose files. It enables developers to preview and analyze infrastructure changes directly within GitHub pull requests, providing snapshots that can be delivered via PR comments or email. This action simplifies infrastructure visualization and improves collaboration by offering clear insights into deployment configurations before merging.
