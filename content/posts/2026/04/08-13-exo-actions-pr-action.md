## Action Summary

The **eXo PR Builds Action** is a GitHub Action designed to automate the process of building Maven projects within a CI/CD pipeline. It simplifies the configuration and execution of Maven builds by allowing users to specify JDK versions, Maven profiles, and additional build options. This action helps streamline pull request workflows by automating the compilation and testing of Java projects, ensuring consistent and efficient builds.
