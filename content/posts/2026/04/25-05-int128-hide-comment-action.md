## Action Summary

The **hide-comment-action** GitHub Action automates the process of hiding (minimizing) comments in a pull request, particularly those created by the `github-actions` bot or matching specific conditions such as author, content, or pattern-based criteria. It helps maintain cleaner and more focused pull request discussions by reducing comment clutter. Key capabilities include filtering comments based on customizable conditions like text patterns and author names.
