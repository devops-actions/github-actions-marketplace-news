## Action Summary

The **Gentesty Evaluation Action** is a GitHub Action designed to automate the process of triggering evaluation runs on the Gentesty platform. It streams real-time logs from the evaluation process directly into GitHub Actions workflow logs and provides a final success or failure result upon completion. This simplifies the integration of Gentesty evaluations into CI/CD workflows, enabling secure, efficient, and transparent test execution.
