## Action Summary

This GitHub Action, **SWEny E2E**, facilitates agent-driven end-to-end browser testing against a deployed application without requiring traditional test scripts, selectors, or fixtures. By leveraging the `agent-browser` tool, the action automates UI testing using natural language workflows and self-healing capabilities that adapt to dynamic UI changes. It also captures and uploads screenshots as artifacts after every run, providing a visual audit trail for debugging and analysis.
