## Action Summary

The `agent-bom` GitHub Action is an open-source security scanner designed to identify and assess vulnerabilities in the AI supply chain, including agents, MCP servers, packages, containers, cloud infrastructure, GPUs, and runtime environments. It automates tasks such as dependency analysis, blast radius mapping, CVE detection, cloud posture auditing, and remediation planning, providing actionable insights to mitigate risks. Key features include CLI-based scanning, graph-based impact visualization, and a dashboard for persistent state reviews and tracking.
