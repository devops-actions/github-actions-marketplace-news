## Action Summary

DockDesk is a local-first semantic documentation auditor that ensures alignment between code logic and documentation claims by identifying discrepancies and suggesting fixes. It runs entirely on local machines or CI runners, eliminating privacy risks and reducing infrastructure costs associated with cloud-based solutions. Key capabilities include semantic drift detection, natural-language CLI for audits, and export options for reports in various formats, making it a powerful tool for maintaining accurate and consistent documentation.
