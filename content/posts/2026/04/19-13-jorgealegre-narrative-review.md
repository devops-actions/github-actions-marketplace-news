## Action Summary

Narrative Review is a GitHub Action that uses AI to reorder pull request diffs into a logical, causal narrative, helping reviewers understand the reasoning behind changes rather than just viewing them as a file list. It automates the organization of code changes, identifies root causes, traces dependencies, and groups related changes into coherent chapters with safety annotations, inline PR comments, and progress tracking. This action simplifies code reviews by improving clarity, reducing cognitive load, and providing a structured, story-like walkthrough of changes.
