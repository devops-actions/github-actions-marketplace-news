## Action Summary

**vibecheck** is a GitHub Action and standalone tool designed to identify and flag potential issues in AI-generated or manually-written code, with a focus on improving security, error handling, code quality, and identifying AI-specific coding patterns. It automates the detection of common problems like hardcoded secrets, unsafe practices (e.g., SQL concatenation, `eval`), and poor coding habits (e.g., empty catch blocks, overly long functions), helping developers maintain clean, secure, and production-ready codebases. The tool requires no configuration, API keys, or installation, making it fast and easy to use locally.
