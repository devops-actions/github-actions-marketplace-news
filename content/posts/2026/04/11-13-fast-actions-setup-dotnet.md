## Action Summary

The `setup-dotnet` GitHub Action provides a fast and efficient way to install .NET SDKs, Runtimes, and ASP.NET Core Runtimes in CI/CD workflows. It automates tasks such as parallel downloads, version resolution (with support for wildcards, keywords, and `global.json`), and caching to optimize performance and minimize redundant installations. Its key features include intelligent deduplication, non-root installation, and conditional use of pre-installed .NET versions, making it a robust alternative to streamline .NET setup in GitHub Actions.
