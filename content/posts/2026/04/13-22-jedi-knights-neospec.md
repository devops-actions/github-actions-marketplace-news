## Action Summary

`neospec` is a self-contained test runner and coverage tool designed for Neovim plugins and distributions. It automates the setup, execution, and isolation of tests without requiring a system-wide Neovim installation or external frameworks, making it ideal for ephemeral CI environments. Key features include automatic Neovim version management, isolated test environments, Lua code coverage instrumentation, and support for generating reports in common formats like LCOV, Cobertura XML, and JUnit XML.
