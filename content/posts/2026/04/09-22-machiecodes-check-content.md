## Action Summary

**Check Content** is a GitHub Action designed to automate the moderation of GitHub issues using AI. By analyzing new issues against user-defined categories (e.g., spam or off-topic), it can automatically classify, comment on, and close issues that match predefined criteria. This action streamlines project management by reducing the manual effort required to handle unwanted or irrelevant issue submissions.
