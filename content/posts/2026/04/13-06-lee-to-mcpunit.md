## Action Summary

**mcpunit** is a GitHub Action designed to audit the quality of Model-Context Protocol (MCP) servers in CI pipelines. It identifies issues such as generic tool names, vague descriptions, weak input schemas, and unsafe capabilities, ensuring MCP servers are optimized for reliable AI agent interactions. This lightweight, fast, and deterministic tool automates server validation, preventing errors and improving agent behavior while maintaining minimal overhead in CI environments.
