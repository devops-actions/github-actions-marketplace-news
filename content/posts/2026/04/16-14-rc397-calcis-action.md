## Action Summary

The "Calcis LLM Cost Estimator" GitHub Action automatically estimates the cost of using Large Language Model (LLM) APIs for changed prompt files in pull requests. It scans specified files, calculates token usage and associated costs using the Calcis pre-flight estimation engine, and posts a detailed cost breakdown as a comment on the pull request, updating it with new commits. This action helps developers track and manage LLM-related expenses efficiently during code reviews.
