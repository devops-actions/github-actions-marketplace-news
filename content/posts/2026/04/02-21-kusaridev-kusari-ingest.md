## Action Summary

The **kusari-ingest GitHub Action** automates the ingestion of artifacts like SBOMs, SLSA attestations, and OpenVEX documents into the Kusari Platform as part of a CI/CD workflow. It streamlines the process of securely uploading and associating metadata with these artifacts, enabling enhanced traceability and compliance monitoring for software dependencies and changes. Additionally, the action supports features such as blocked package checks and automated repository metadata capture for improved dependency management.
