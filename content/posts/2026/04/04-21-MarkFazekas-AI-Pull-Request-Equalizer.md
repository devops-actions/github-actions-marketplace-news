## Action Summary

The **AI Pull Request Equalizer** GitHub Action automates the moderation of pull requests from forked repositories by closing them if the contributor lacks sufficient permissions or an assigned issue, ensuring only authorized or actively engaged contributors can submit changes. This action addresses challenges related to managing contributions from users leveraging Large Language Models (AI), redirecting such efforts to more welcoming projects while reducing potential misuse or spam. Its key functionality includes automated permission checks, issue assignment verification, and customizable closure messages for pull requests.
