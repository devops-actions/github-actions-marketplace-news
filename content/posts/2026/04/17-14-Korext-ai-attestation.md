## Action Summary

AI Attestation is a GitHub Action that generates and maintains a machine-readable YAML file in your repository to track the use of AI coding tools. It automates the identification of AI-generated code, the tools used, and whether the code has undergone governance scanning, addressing compliance and accountability needs for enterprises and maintainers. Key capabilities include automatic detection of AI contributions from various tools and continuous updates via a post-commit hook.
