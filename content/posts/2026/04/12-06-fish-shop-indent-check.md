## Action Summary

The `fish-shop/indent-check` GitHub Action is designed to automatically verify the indentation consistency of `.fish` shell files within a repository. It simplifies the process of ensuring proper formatting by scanning specified files or directories, supporting wildcards and brace expansions for flexible file matching. This action helps developers maintain clean and standardized code in projects using the fish shell.
