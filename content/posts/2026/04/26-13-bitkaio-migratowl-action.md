## Action Summary

Migratowl is a GitHub Action that automates dependency migration analysis using AI. It identifies breaking changes in dependencies, provides detailed explanations of failures, and suggests fixes, streamlining the review process for Dependabot or Renovate pull requests. The action operates in an isolated environment by creating a temporary Kubernetes cluster within the GitHub runner, ensuring secure and efficient scanning without external infrastructure.
