## Action Summary

The `codetrace-action` GitHub Action automates the generation of changelogs using AI-powered analysis from CodeTrace. It simplifies the process of creating detailed, formatted changelogs by analyzing changes between specified commits, branches, or tags, and outputs them in various formats (e.g., markdown, plaintext, HTML, or JSON). This action is ideal for streamlining release workflows and improving documentation consistency.
