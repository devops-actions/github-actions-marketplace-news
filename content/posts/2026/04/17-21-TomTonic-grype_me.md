## Action Summary

This GitHub Action, **grype_me**, automates vulnerability scanning of a project's supply chain using Anchore Grype. It identifies known security issues in source dependencies, container images, directories, or SBOMs, and generates dynamic shields.io badges with detailed Markdown reports for easy monitoring. The action is optimized for speed with a pre-downloaded vulnerability database and offers features like nightly release scans, configurable thresholds for failing builds, and reporting only on vulnerabilities with available fixes.
