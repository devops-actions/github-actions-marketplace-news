## Action Summary

This GitHub Action, **Get Selected**, automates the process of retrieving a list of selected checkboxes from a `workflow_dispatch` event's input. It streamlines workflows by outputting the selected options as a formatted list or JSON, with support for custom separators and the ability to exclude specified items. This action is ideal for dynamically capturing and processing user-defined selections in GitHub Actions workflows.
