## Action Summary

The **Auto Version Bump Action** is a reusable GitHub Action that automates the process of incrementing a project's semantic version (SemVer), committing the changes to a dedicated branch, and opening a draft pull request for review. It streamlines release preparation workflows by handling version updates, branch management, and PR creation, ensuring consistency and reducing manual effort. Key features include support for `patch`, `minor`, or `major` version bumps and validation against existing tags or releases to prevent conflicts.
