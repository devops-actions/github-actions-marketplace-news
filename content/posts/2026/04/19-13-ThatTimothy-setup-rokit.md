## Action Summary

The **setup-rokit** GitHub Action is designed to quickly and efficiently set up Rokit, a tool used for Roblox development workflows, on Linux-based GitHub Actions runners. It automates tasks such as downloading Rokit and its dependencies, handling authentication to avoid rate-limiting, and caching these resources to optimize runtime performance. This action is optimized for speed and reliability, addressing limitations like slower execution and incomplete caching found in other similar solutions.
