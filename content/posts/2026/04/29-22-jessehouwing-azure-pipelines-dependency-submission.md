## Action Summary

The **Azure Pipelines Dependency Submission** GitHub Action automates the process of analyzing Azure Pipelines configurations to identify task dependencies and submitting them to GitHub's Dependency Graph. This enables vulnerability scanning and Dependabot alerts for potential security issues in Azure Pipelines tasks. Key features include automatic discovery of pipeline files, integration with the Azure DevOps API for detailed dependency resolution, and support for custom file paths and templates.
