## Action Summary

This GitHub Action automates the installation of the `stackit-cli` binary on workflow runners, allowing users to specify the desired version or default to the latest stable release. It simplifies the setup process by caching the binary and adding its path to the environment, ensuring seamless integration into CI/CD pipelines. This action solves the problem of manually managing `stackit-cli` versions and paths, streamlining workflows that rely on `stackit-cli`.
