## Action Summary

The **`node-semantic-release`** GitHub Action automates the process of installing dependencies, building a Node.js project, and using `semantic-release` to manage versioning and changelogs. It supports two modes: "Release mode" for building, tagging, and releasing a project, and "Publish mode" for creating a GitHub release from an existing tag with release notes. This action simplifies release workflows by automating semantic versioning, release note generation, and GitHub release creation.
