## Action Summary

**TrustPack Action** is a GitHub Action designed to automate the generation of a comprehensive security due diligence pack from a repository. It collects repository metadata, local files, dependency manifests, and workflow YAML to produce buyer-facing Markdown, JSON, and HTML reports that summarize security evidence for customer reviews. This tool streamlines the process of preparing repository-level security documentation, helping teams demonstrate compliance and address security concerns during customer evaluations.

## What's Changed

TrustPack Action v0.1.0

Initial Marketplace-ready release.

- Generates repository-level security due diligence reports from GitHub repository evidence.
- Includes Markdown, JSON, and optional HTML outputs.
- Checks repository metadata, branch protection/rulesets, expected security files, dependency manifests, workflow YAML, and workflow risk patterns.
- Runs without a hosted backend, telemetry, or runtime OpenAI dependency.

TrustPack helps collect repository-level security evidence. It is not legal, audit, or compliance certification.
