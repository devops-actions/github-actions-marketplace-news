## Action Summary

The Vizb GitHub Action automates the process of running Go benchmarks, visualizing performance data, and generating interactive 4D charts. It transforms raw benchmark outputs into deployable HTML files with customizable metrics, grouping, and filtering options, enabling detailed analysis across dimensions like time, memory, and releases. This tool simplifies performance tracking and comparison, particularly in CI workflows, by allowing seamless integration and historical data merging.
