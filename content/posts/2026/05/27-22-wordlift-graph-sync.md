## Action Summary

The `graph-sync` GitHub Action automates the execution of the `worai` tool, specifically its `graph sync run` functionality, to synchronize graph data based on configurable input sources such as URLs, sitemaps, or Google Sheets. It handles dependencies by installing `worai`, Python Playwright, and browser binaries, while offering caching for faster builds. This action simplifies workflows for managing data synchronization tasks, ensuring flexibility and integration with various input configurations.
