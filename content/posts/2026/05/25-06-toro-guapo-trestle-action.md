## Action Summary

This GitHub Action integrates the Trestle Community secret scanner into your CI/CD pipeline to automatically detect and report leaked secrets, such as API keys, passwords, and private keys, on every push or pull request. It helps prevent sensitive information from being exposed in your repository by scanning code and optionally uploading results as code scanning alerts in GitHub's Security tab. The action ensures secure and verified execution by downloading and validating the Trestle binary against published checksums.
