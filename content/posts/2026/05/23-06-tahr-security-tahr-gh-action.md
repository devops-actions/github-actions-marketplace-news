## Action Summary

The Tahr GitHub Action facilitates automated CI/CD assessments by integrating with the Tahr application. It triggers security or code analysis scans and returns a queued assessment ID, streamlining the process of initiating and tracking assessments without waiting for completion. This action helps automate security and analysis workflows directly within GitHub Actions.
