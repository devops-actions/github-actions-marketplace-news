## Action Summary

The **Setup OrchStep** GitHub Action installs the OrchStep CLI on a runner, enabling YAML-based workflow orchestration in GitHub Actions. This action automates the installation, verification, and configuration of the CLI, making it immediately available on the system `PATH` for running tasks such as executing or linting workflows. It simplifies workflow management by providing version control, caching, and secure installation of the CLI.
