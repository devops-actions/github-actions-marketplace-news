## Action Summary

The `dotnet-mkdocs-material` GitHub Action automates the process of generating Markdown API documentation from XML doc comments in .NET projects using `MkDocsSharp.MDGen`, and then builds a static site using MkDocs Material. It supports multi-project setups, creating unified documentation with shared navigation and search functionality. This action simplifies and streamlines the creation of professional, static documentation sites directly from .NET project files.
