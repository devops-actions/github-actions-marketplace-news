## Action Summary

This GitHub Action, named "Thoth," provides automated instrumentation for capturing OpenTelemetry traces, metrics, and logs from shell scripts, GitHub workflows, and various commands (e.g., `curl`, `wget`, etc.). It automates tasks like context propagation, log collection, and span creation, enabling developers to monitor and troubleshoot workflows and scripts with minimal manual configuration. Key capabilities include auto-injection into child scripts and executables, seamless integration with GitHub Actions, and support for multiple shell environments.
