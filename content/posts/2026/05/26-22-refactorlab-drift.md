## Action Summary

Drift is an open-source performance and FinOps platform designed to detect, diagnose, and prevent performance regressions during the software development lifecycle. It combines static call-graph analysis, runtime profiling, and automated pull request (PR) verdicts to identify performance bottlenecks such as N+1 queries, expensive computations, and blocking I/O, enabling developers to address issues early, before they impact production. By providing tools for static analysis, runtime profiling, and PR annotations, Drift helps teams optimize application performance, reduce operational costs, and streamline incident response efforts.
