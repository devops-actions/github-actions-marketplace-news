## Action Summary

This GitHub Action generates a GitHub App installation access token by securely signing a JSON Web Token (JWT) using a private key stored in Google Cloud KMS, eliminating the need to handle private keys directly. It automates the process of obtaining repository-scoped tokens for specific access levels, supporting secure integration between GitHub Actions workflows and GitHub Apps. The action is tailored for use with Google Cloud authentication and provides a minimal variant for projects without pre-existing Google Cloud authentication setups.
