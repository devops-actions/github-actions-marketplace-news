## Action Summary

The `scalp-action` GitHub Action integrates the SCAL-P tool into CI workflows to enforce dependency policies, verify lockfile integrity, and audit JavaScript project dependencies. It automates tasks like evaluating custom policies, blocking policy violations, verifying dependency hashes, and generating structured reports (JSON or SARIF). This action addresses security risks by ensuring policy compliance and dependency integrity, reducing vulnerabilities associated with package manager behaviors.
