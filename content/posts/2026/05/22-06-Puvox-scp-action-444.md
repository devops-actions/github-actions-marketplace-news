## Action Summary

The SCP GitHub Action enables automated file and artifact transfers to remote servers via SSH within GitHub workflows. It supports both SSH key and password authentication, handles cross-platform path conversions, and offers advanced features like incremental file transfers and proxy (jump host) usage. This action simplifies deployment tasks by streamlining secure file transfers across multiple environments directly from CI/CD pipelines.
