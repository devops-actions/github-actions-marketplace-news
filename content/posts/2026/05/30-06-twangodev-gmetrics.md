## Action Summary

The `gmetrics` GitHub Action is a Go-based tool designed to generate SVG output for repository metrics, specifically focusing on language statistics. It automates the process of collecting and caching detailed per-repository language data across runs, ensuring efficient updates by processing only new commits since the last execution. This action simplifies tracking and visualizing language usage trends in repositories.
