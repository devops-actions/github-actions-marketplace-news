## Action Summary

The `heatcheck-action` is a GitHub Action designed to perform static taint analysis on Python and Go projects, identifying security vulnerabilities such as SQL injection, command injection, and path traversal. It automates the detection of unsanitized, attacker-controlled inputs reaching sensitive sinks, with a focus on vulnerabilities commonly introduced by AI-assisted code. Key capabilities include automated scanning of project files, inline annotations for detected issues, workflow summaries, and optional job failure on violations, streamlining secure coding practices in CI/CD pipelines.
