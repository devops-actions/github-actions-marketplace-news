## Action Summary

The **Tahr GitHub Action** automates the process of triggering a Tahr CI/CD assessment directly from a GitHub Actions workflow. It queues an application assessment via the Tahr API and returns a unique assessment ID without waiting for the scan to complete. This action streamlines security and code quality checks by integrating Tahr's assessment capabilities into CI/CD pipelines.
