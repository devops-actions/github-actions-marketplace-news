## Action Summary

The `speq-github-runner` GitHub Action serves as a wrapper for orchestrating the `speq-cli` tool within CI workflows. It automates tasks such as installing the CLI, validating test configurations, executing tests, generating reports, and uploading artifacts, providing a reproducible and streamlined process for managing test execution. By handling setup, execution, and custom command sequences without reimplementing the test logic, it simplifies CI integration and ensures consistent results.
