## Action Summary

The **Email HTML Validator** is a versatile tool designed to validate email newsletter templates by analyzing inline HTML, files, stdin, or URLs against the Can I Email dataset to identify supported, partially supported, or rejected features. It automates the detection of email compatibility issues and provides optional accessibility audits using Playwright and axe-core. The tool supports multiple interfaces, including a GitHub Action for CI/CD pipelines, and generates detailed reports in JSON, XML, HTML, and Markdown formats for streamlined validation workflows.
