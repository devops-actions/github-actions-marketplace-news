## Action Summary

The `gh-agent` GitHub Action enables solo developers to interact with an AI-powered assistant directly within GitHub Issues and Pull Requests to discuss requirements and automate code modifications. Running entirely on GitHub Actions, it eliminates the need for deploying a server and supports custom AI model integrations. Key features include automated context management, selective tool usage, and the ability to create branches and pull requests for code updates while adhering to repository rules.
