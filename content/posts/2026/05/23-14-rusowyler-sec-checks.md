## Action Summary

This GitHub Action, **SEC Checks - SAST & Supply Chain Scanner**, automates security analysis by combining Semgrep-based static application security testing (SAST) with supply chain audits of `package.json` lifecycle scripts. It detects vulnerabilities across various programming languages, flags potentially malicious scripts, and generates actionable findings in a structured summary. The action helps developers identify and mitigate security risks in their codebase and dependencies, streamlining the process of securing software projects.
