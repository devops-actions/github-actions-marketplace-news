## Action Summary

This GitHub Action automates the installation of the `go-task/task` build tool on Linux and macOS GitHub Actions runners by downloading the specified version, verifying it against official checksums, and caching it for reuse in future workflow runs. It simplifies the setup process for workflows requiring `go-task/task`, ensuring secure and efficient installations while reducing redundant downloads through caching.
