## Action Summary

Skylos is an open-source, local-first static analysis tool designed to identify dead code, security vulnerabilities, secrets, quality issues, and common errors in AI-generated code across multiple programming languages. It automates code reviews by providing comprehensive checks for unused code, dangerous data flows, configuration issues, and other potential risks, making it a valuable tool for enhancing code quality and security in CI/CD pipelines. Skylos also supports customizable rules, local scanning, and integration with GitHub Actions and IDEs for flexible workflows.
