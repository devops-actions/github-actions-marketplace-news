## Action Summary

The `upm-config` GitHub Action automates the configuration of Unity Package Manager (UPM) private scoped registry credentials in CI/CD workflows. It simplifies the process of authenticating with private UPM registries by securely setting up the necessary credentials, such as registry URL, authentication token, or username and password. This action streamlines workflows by eliminating manual configuration, ensuring seamless access to private Unity package repositories.
