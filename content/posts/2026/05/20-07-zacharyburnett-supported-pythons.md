## Action Summary

The `supported-pythons` GitHub Action automates the retrieval of active Python minor versions supported by a specified Python package based on its `pyproject.toml` file. It simplifies compatibility checks by filtering out end-of-life or unsupported versions and optionally provides the latest supported version, streamlining dependency management and integration workflows.
