## Action Summary

The *Complimetric IaC Compliance Scan* GitHub Action automates the process of scanning Infrastructure-as-Code (IaC) templates, including Terraform, CloudFormation, and ARM, for compliance issues within a CI/CD pipeline. It helps ensure adherence to security and compliance standards by identifying issues, assigning severity levels, and optionally failing workflows based on criticality thresholds. Key features include generating compliance scores, detailed issue reports, and integrations with frameworks like SOC2, PCI-DSS, and ISO27001.
