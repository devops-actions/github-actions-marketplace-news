## Action Summary

The **CICD Stats Watcher** GitHub Action is designed to monitor and report resource usage (e.g., storage, Docker, CPU, memory, and inodes) on self-hosted runners during CI/CD workflows. It automates the tracking of key metrics for specific workflow steps, generating downloadable reports and enabling teams to identify and address resource bottlenecks. This action simplifies integration by bundling all necessary scripts, eliminating the need for manual setup in each repository.
