## Action Summary

This GitHub Action automates the cleanup of outdated releases and associated tags in a repository, helping maintain a clean and manageable release history. It allows users to retain a specified number of recent releases, delete specific tags/releases based on patterns, remove pre-releases, and clean up releases based on download counts or expiration dates. Designed with enterprise-level code quality and security, it simplifies repository maintenance by eliminating manual release management tasks.
