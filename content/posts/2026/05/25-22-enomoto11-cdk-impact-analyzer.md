## Action Summary

The `cdk-impact-analyzer` GitHub Action analyzes pull request diffs in TypeScript-based AWS CDK projects to identify which CDK stacks are impacted by code changes. It automates the process of tracing changes in the codebase to their associated CDK stack definitions and outputs a list of affected stacks, helping teams focus their `cdk diff` and `cdk deploy` operations only on relevant stacks for faster and more efficient CI/CD workflows.
