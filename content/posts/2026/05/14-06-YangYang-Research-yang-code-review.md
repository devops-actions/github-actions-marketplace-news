## Action Summary

Yang Code Review (YCR) is a GitHub Action that automates the process of reviewing code changes in repositories using AI models. It evaluates code against multiple OWASP security standards across various domains, such as web, mobile, APIs, machine learning, and infrastructure-as-code, identifying vulnerabilities and providing actionable recommendations. This tool streamlines secure code reviews, ensuring comprehensive compliance with industry-leading security practices.
