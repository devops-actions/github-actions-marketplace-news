## Action Summary

Sverklo is an open-source tool designed to enhance AI coding agents by providing local-first code intelligence, including a real symbol graph, blast-radius analysis, and git-pinned memory. It eliminates issues such as function name hallucinations in large codebases, enabling more accurate and efficient code understanding without relying on external cloud uploads or embedding-based approaches. Key capabilities include ranked, traceable, and call-graph-aware results, significantly reducing input tokens compared to traditional methods like grep.
