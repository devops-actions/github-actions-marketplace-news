## Action Summary

The `fish-shop/syntax-check` GitHub Action automates syntax validation for Fish shell script files within a repository. It scans `.fish` files, supports customizable file patterns using wildcards and brace expansions, and provides detailed results, including the number of files checked, passed, and failed. This action streamlines syntax checks, identifies issues, and optionally generates annotations for improved CI/CD workflows.
