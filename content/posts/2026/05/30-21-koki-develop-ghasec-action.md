## Action Summary

The **ghasec-action** is a GitHub Action designed to automate the execution of [ghasec](https://github.com/koki-develop/ghasec), a security linter for GitHub Actions workflows. It helps identify and address security vulnerabilities or misconfigurations in workflows, ensuring compliance with best practices. Key features include support for customizable rule sets, optional network-enabled checks, and integration with tools like ShellCheck for enhanced validation.
