## Action Summary

The `assign-reviewers-after-resolve` GitHub Action automates the assignment of human reviewers to a pull request only after all AI-generated review comments have been resolved. It addresses the issue of premature reviewer notifications by delaying assignments until the PR is ready for human review, ensuring a sequential "AI review first, human review after" workflow. Key features include support for CODEOWNERS-style reviewer files, handling unresolved review threads, excluding specific accounts, and providing configurable behavior for unresolved comments.
