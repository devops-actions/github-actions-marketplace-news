## Action Summary

The **igor-setup** GitHub Action automates the installation and configuration of the Igor executable to enable building GameMaker projects in CI/CD workflows. It simplifies the setup process by handling runtime management, platform preferences, and SDK configurations for supported build targets (Windows, Android, iOS). This action is designed to streamline GameMaker project builds, reducing manual configuration and ensuring consistent environments across workflows.
