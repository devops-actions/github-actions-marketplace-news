## Action Summary

The **AI Repo Digest** GitHub Action automates the creation of a concise, contributor-focused summary of recent repository activity by analyzing git history using a local AI model. It provides an easy-to-read digest that highlights key changes, contributions, and areas of expertise, enabling team members to stay informed about codebase updates without directly reviewing code. This action eliminates the need for external services, protects data privacy, and helps streamline team collaboration by identifying whom to consult for specific areas of the project.
