## Action Summary

Big Brother Intel is a GitHub Action designed to enhance pull request (PR) reviews by providing contextual understanding of what a developer is building, not just the code changes. It reads the PR description, uses an AI (Claude) to generate a summary of the intended feature or functionality, and passes this context to PR-Agent for a more informed review. This helps ensure reviews focus on solving the actual problem rather than just catching syntax or compilation issues.

## What's Changed

Reads your PR description, summarizes it with Claude, and passes that context to PR-Agent — so your code review is about whether your code actually solves the problem, not just whether it compiles. Big brother intel also assesses whether the architecture and design of the feature is the right approach, not just whether the implementation is correct.
