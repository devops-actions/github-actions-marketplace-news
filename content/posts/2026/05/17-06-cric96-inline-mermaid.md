## Action Summary

This GitHub Action automates the process of converting Mermaid.js charts into inline SVGs within static HTML pages using the Mermaid CLI. It scans specified HTML files, identifies embedded Mermaid.js code, and replaces it with rendered SVG graphics, ensuring charts are fully integrated and styled within the web page. This action streamlines workflows for developers creating static websites or documentation with embedded, visually enhanced diagrams.
