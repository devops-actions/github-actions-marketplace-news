## Action Summary

The "Artifact TTL Purge" GitHub Action automates the deletion of workflow artifacts that exceed a specified age, helping to manage and limit artifact storage usage. It supports configurable time-to-live durations and optional filtering by artifact name prefixes, ensuring precise control over what gets deleted. The action operates within GitHub's rate limits and outputs details of deleted artifacts, addressing the lack of a native GitHub feature for automated artifact expiration.
