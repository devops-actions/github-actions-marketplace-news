## Action Summary

The Pedagogical SAST Profile (PSP) GitHub Action is designed to perform application-level static analysis tailored for educational contexts. It focuses on beginner-friendly security concepts from the OWASP Top 10, analyzing only tutorial-authored code and providing framework-specific remediation guidance. By reducing cognitive load and excluding irrelevant findings, this action helps learners identify and address security issues in their own code without overwhelming them with extraneous or overly complex alerts.
