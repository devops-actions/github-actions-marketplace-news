## Action Summary

The `fish-shop/indent-check` GitHub Action is designed to automatically verify the indentation of `.fish` shell script files within a repository. It scans specified files or directories, ensuring compliance with expected indentation standards and outputs a summary of files that pass or fail the checks. This action helps developers maintain consistent code formatting and streamline the review process in projects using the Fish shell.
