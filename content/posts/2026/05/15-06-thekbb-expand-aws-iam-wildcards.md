## Action Summary

This GitHub Action, **Expand AWS IAM Wildcards**, automates the process of expanding wildcard actions in AWS IAM policies found in pull request (PR) diffs. It posts inline comments to PRs, detailing the specific actions matched by each wildcard, along with links to relevant AWS documentation. This helps reviewers quickly understand the security implications of policy changes, streamlining code reviews and improving visibility into IAM permissions.
