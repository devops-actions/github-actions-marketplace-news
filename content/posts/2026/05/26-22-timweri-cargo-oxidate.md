## Action Summary

`cargo-oxidate` is a tool and GitHub Action designed to analyze the `Cargo.lock` file in Rust projects to identify dependencies that are either too new (posing potential supply chain risks) or too old (susceptible to staleness or known vulnerabilities). It automates the process of flagging and managing package versions based on customizable age thresholds, helping maintainers ensure dependencies are neither outdated nor unvetted. Key features include exempting specific packages, excluding undated dependencies, and suggesting downgrade commands for overly recent packages.
