## Action Summary

The `cursor-issue-triage` GitHub Action automates issue triage by triggering a Cursor Cloud Agent when an issue is labeled. It processes issue text (including embedded HTTPS images), works in a dedicated branch, and can open a pull request to close the issue. This action streamlines issue resolution by reducing manual effort and enabling automated pull requests based on AI-driven analysis.
