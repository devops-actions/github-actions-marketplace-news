## Action Summary

The **heatcheck-action** is a GitHub Action designed to perform static taint analysis on Python projects, identifying and reporting vulnerabilities where untrusted input reaches sensitive operations (sinks) without proper sanitization. It automates the detection of common security issues like SQL injection, command injection, XXE, and path traversal, with a focus on patterns prevalent in AI-assisted Python development. Key capabilities include scanning Python codebases for vulnerabilities, generating detailed reports, and optionally failing workflows to enforce security hygiene.
