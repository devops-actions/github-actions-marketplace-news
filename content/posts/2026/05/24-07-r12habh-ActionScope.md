## Action Summary

ActionScope is a GitHub Action and CLI tool designed to analyze CI/CD workflows for security vulnerabilities and assess the potential impact of compromised AWS credentials. It automates the detection of risks such as privilege escalation, misconfigured OIDC trust policies, unpinned actions, and known-compromised dependencies, providing plain-English reports on the blast radius of workflows without requiring AWS credentials. This tool helps developers identify and mitigate security risks in their GitHub Actions workflows, ensuring safer integration with AWS resources.
