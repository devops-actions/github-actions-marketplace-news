## Action Summary

The Fortify GitHub Action integrates OpenText Fortify Application Security Testing (AST) into GitHub workflows, automating application security scans such as Static Application Security Testing (SAST) and Software Composition Analysis (SCA). It streamlines DevSecOps practices by enabling developers to identify vulnerabilities and secure their software supply chain within CI/CD pipelines. The action supports customizable workflows and ensures compatibility with new features and updates by leveraging the Fortify CLI.
