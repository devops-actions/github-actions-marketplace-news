## Action Summary

The **Unentropy Track Metrics** GitHub Action enables users to collect, store, and visualize custom code metrics directly within their CI/CD pipelines. It automates the tracking of trends and performance metrics in a codebase over time, providing insights into development progress and areas for improvement. This action supports multiple storage backends, such as local SQLite, artifact-based storage, and S3-compatible storage, and generates reports for analysis.
