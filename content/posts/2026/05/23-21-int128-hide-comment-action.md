## Action Summary

The **hide-comment-action** GitHub Action automates the process of hiding (minimizing) specific comments in pull requests, typically those generated by automated workflows or the `github-actions` user. It provides filtering capabilities based on comment authors or text patterns (e.g., starts with, ends with, or contains specific strings), allowing users to keep discussions in pull requests clean and focused. This action is particularly useful for managing and decluttering comment sections in active pull requests by automatically minimizing outdated or irrelevant comments.
