## Action Summary

Coldstep is a GitHub Action paired with a lightweight eBPF agent for GitHub-hosted Ubuntu runners that monitors and logs egress network activity and process behavior during CI/CD workflows. It automates network security by offering two modes: "detect" for logging activity without blocking, and "defend" to enforce allowlist-based network access policies, preventing unauthorized outbound connections. This helps users enhance security and visibility in their CI pipelines without restricting other build tools or processes.
