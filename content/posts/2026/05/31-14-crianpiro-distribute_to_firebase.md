## Action Summary

This GitHub Action automates the process of uploading Android and iOS app artifacts (`.apk`, `.aab`, `.ipa`) to Firebase App Distribution, simplifying app testing and deployment workflows. It leverages the Firebase CLI to handle authentication, optional inputs, and provides useful outputs like tester links, console links, and direct binary download links. This action streamlines artifact distribution, reducing manual steps and ensuring efficient app delivery to testers.
