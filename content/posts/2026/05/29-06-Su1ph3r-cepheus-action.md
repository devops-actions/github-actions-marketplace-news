## Action Summary

The `cepheus-action` GitHub Action integrates the Cepheus container escape scanner into CI/CD pipelines, enabling automated security scans of container images. It detects escape chains, generates SARIF reports, and uploads them to GitHub Code Scanning, failing builds when security thresholds are exceeded or regressions are identified. This action simplifies container security testing, ensuring vulnerabilities are flagged and addressed during development.
