## Action Summary

The `supply-chain-guard` GitHub Action is an open-source security scanner designed to protect software supply chains across platforms like npm, PyPI, Docker, Terraform, GitHub repositories, and more. It automates the detection of malware campaigns, code-level threats, supply chain attacks, and infrastructure vulnerabilities, while also verifying dependency integrity and repository trust signals. Key capabilities include threat analysis, CycloneDX SBOM generation, SLSA provenance verification, and attack-chain correlation to enhance security and safeguard against malicious activities.
