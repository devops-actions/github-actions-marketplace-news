## Action Summary

OpenAPI Forge is a WebAssembly (WASM)-based tool designed to replace `openapi-generator` by enabling users to generate code from OpenAPI specifications through customizable and sandboxed plugins. It addresses the limitations of the traditional approach by decoupling code generation from the core project, allowing anyone to create, publish, and use plugins without requiring trust or upstream changes, while ensuring security and deterministic outputs through a tightly controlled WASM sandbox. The tool provides capabilities such as parsing OpenAPI specs into a normalized intermediate representation (IR), applying transformations, and generating code via configurable plugins sourced from local files or OCI registries.
