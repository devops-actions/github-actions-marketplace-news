## Action Summary

**alint** is a high-performance, language-agnostic linter designed to enforce repository structure, file conventions, and content standards as defined in a single `.alint.yml` configuration file. It automates tasks such as verifying required files, checking naming/content patterns, validating ecosystem metadata (e.g., `package.json` or `Cargo.toml`), and ensuring compliance with cross-file relationships or project hygiene rules. Key features include support for 83 rule types, ecosystem-specific rulesets, auto-fix capabilities, and compatibility with any repository or programming language through a standalone Rust-based binary.
