## Action Summary

Scanbox is a Docker-based GitHub Action designed for automated security auditing of codebases using AI-powered analysis. It identifies critical vulnerabilities such as RCE, SQL Injection, and XSS, and generates structured JSON reports for easy integration into CI/CD pipelines or manual reviews. This tool simplifies security workflows by providing comprehensive vulnerability detection, configurable AI options, and seamless deployment in isolated environments.
