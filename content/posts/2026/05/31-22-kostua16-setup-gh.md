## Action Summary

This GitHub Action enables the installation and configuration of the GitHub CLI (`gh`) on GitHub Actions runners, including self-hosted runners and container-based workflows that do not have `gh` pre-installed. It automates the process of setting up the desired version of `gh`, managing authentication, and configuring isolated environments for CLI usage. This action is particularly useful for workflows requiring `gh` functionality in custom environments.
