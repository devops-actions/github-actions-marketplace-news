## Action Summary

The `node-semantic-release` GitHub Action automates the process of installing dependencies, building a Node.js project, and releasing new versions of the project using `semantic-release`. It generates and base64-encodes release notes for safe downstream transport, simplifying workflows for automated versioning and CI/CD pipelines. This action helps streamline release management, ensuring consistent semantic versioning and efficient handling of release notes.
