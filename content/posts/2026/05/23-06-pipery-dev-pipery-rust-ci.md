## Action Summary

The **Pipery Rust CI** GitHub Action provides a comprehensive continuous integration (CI) pipeline for Rust projects. It automates key development tasks, including static application security testing (SAST), software composition analysis (SCA), linting, building, testing, versioning, packaging, releasing, and reintegration. The action is designed to streamline the Rust development lifecycle, ensuring code quality, security, and efficient release management.
