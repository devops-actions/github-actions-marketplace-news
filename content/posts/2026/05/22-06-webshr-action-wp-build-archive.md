## Action Summary

The **WordPress Build Archive GitHub Action** automates the process of creating a zip archive of a WordPress project using WP-CLI, with optional support for installing Composer dependencies, running npm build scripts, and customizing archive settings. It simplifies packaging WordPress plugins or projects for distribution by handling build steps and file exclusions efficiently. Additionally, it provides the option to upload the generated archive as an artifact.
