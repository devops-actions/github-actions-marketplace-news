## Action Summary

The OSS Security Policy as Code Starter Kit is a GitHub Action and CLI tool designed to evaluate open-source repositories against security and governance policies using bundled profiles and optional evidence files. It automates repository governance, CI/CD hardening, release posture evaluation, and framework-oriented reporting by generating human-readable (Markdown) and machine-readable (JSON, SARIF) reports for compliance and assurance purposes. This action simplifies security policy enforcement in CI pipelines by combining multiple signals, waivers, and evidence into an integrated assessment workflow.
