## Action Summary

Find the Gaps is a CLI tool designed to analyze a codebase and its associated documentation to identify outdated or missing documentation. It automates the detection of discrepancies between code (e.g., function signatures, public APIs) and documentation, solving the common problem of "documentation drift" that link or spell checkers cannot address. Key capabilities include symbol extraction from multiple programming languages and generating a browsable report to pinpoint areas needing documentation updates.
