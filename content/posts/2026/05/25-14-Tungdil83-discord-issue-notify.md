## Action Summary

This GitHub Action automates the process of sending notifications to Discord channels whenever a new issue is opened in a repository. It generates a message with an embed containing the issue title, URL, and body, and optionally allows mentioning specific users or roles. This helps streamline communication and ensures team members are promptly informed about new issues directly in Discord.
