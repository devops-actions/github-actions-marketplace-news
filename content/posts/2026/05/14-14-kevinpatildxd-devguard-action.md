## Action Summary

The **devguard-action** is a GitHub Action designed to audit Node.js and React projects during CI workflows. It automates the detection of issues such as dependency problems, environmental misconfigurations, dead imports, accessibility violations, security risks, license concerns, and supply chain vulnerabilities. By catching these issues in one step and failing CI automatically when errors are detected, it helps ensure code quality, security, and compliance throughout the development process.
