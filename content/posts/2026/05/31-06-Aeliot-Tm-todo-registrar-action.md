## Action Summary

The TODO Registrar GitHub Action automates the process of scanning your codebase for TODO, FIXME, and similar comments and creates corresponding issues in supported issue trackers (e.g., GitHub, GitLab, JIRA). It prevents duplicate issues by injecting issue IDs back into the code comments and commits the changes, streamlining issue tracking and eliminating the need for external databases. This action is ideal for maintaining organized and up-to-date task tracking directly within your repository.
