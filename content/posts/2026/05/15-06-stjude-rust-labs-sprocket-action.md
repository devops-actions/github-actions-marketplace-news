## Action Summary

The Sprocket GitHub Action integrates the Sprocket CLI tool into CI/CD pipelines to automate the validation, linting, and formatting of WDL (Workflow Description Language) documents. It performs static analysis to ensure documents are error-free, adherent to linting rules, and correctly formatted, and can also validate input JSON files against WDL schemas. This action streamlines workflow quality checks, enforces coding standards, and ensures consistency across WDL files in repository-based projects.
