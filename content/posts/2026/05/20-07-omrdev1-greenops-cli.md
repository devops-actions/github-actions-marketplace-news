## Action Summary

The **GreenOps CLI** GitHub Action provides automated carbon footprint analysis for AWS, Azure, and GCP Terraform plans within CI/CD pipelines. It evaluates the environmental impact of cloud infrastructure, including Scope 2 operational emissions, Scope 3 embodied emissions, and water consumption, and generates actionable optimization recommendations directly on GitHub pull requests. This tool helps developers reduce carbon emissions, water usage, and costs by suggesting resource configuration changes and enforces sustainability policies through customizable budget thresholds.
