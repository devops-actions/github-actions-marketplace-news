## Action Summary

**action-vnext** is a GitHub Action that automates semantic versioning by analyzing Git commit messages based on Conventional Commit standards. It calculates the next semantic version (major, minor, or patch) without requiring Node.js, making it lightweight and language-agnostic. This action streamlines version management in CI/CD pipelines and can optionally generate changelogs to accompany version updates.
