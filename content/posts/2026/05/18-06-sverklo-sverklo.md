## Action Summary

Sverklo is an open-source, local-first code intelligence tool that enhances AI coding agents by providing accurate symbol graphs, call-graph-aware blast-radius analysis, and git-pinned memory. It eliminates issues like hallucinated function names and irrelevant search results in large codebases by delivering precise, ranked, and traceable outputs with significantly fewer input tokens compared to traditional methods like grep or embedding-based approaches. Sverklo operates entirely locally, ensuring no code leaves the machine, and offers reproducible benchmarks to validate its performance.
