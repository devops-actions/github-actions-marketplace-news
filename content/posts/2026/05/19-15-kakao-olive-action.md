## Action Summary

**OLIVE Action** integrates with GitHub Actions to automate open-source license compliance and vulnerability management using the OLIVE CLI. It performs dependency analysis on source code during pull requests, provides results as comments on PRs, and integrates with the OLIVE Platform for centralized license tracking and issue management. Additionally, it stores detailed analysis results as artifacts for further review.
