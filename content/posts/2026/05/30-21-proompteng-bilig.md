## Action Summary

Bilig is a GitHub Action and accompanying library designed to identify and fix stale formula values in `.xlsx` files that occur when Node.js-based libraries like SheetJS, ExcelJS, or `xlsx-populate` modify workbook data without recalculating dependent formulas. It automates the detection and recalculation of outdated formula values, ensuring accurate and up-to-date results in workflows that rely on Excel files for tasks such as pricing, validation, and financial modeling. This tool is particularly valuable for maintaining data integrity in CI/CD pipelines and automated processes that involve `.xlsx` files.
