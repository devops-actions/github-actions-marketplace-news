## Action Summary

The `@nyaomaru/changelog-bot` GitHub Action automates the generation of polished changelog entries by analyzing Git history, pull request titles, and release notes. It solves the problem of manual, error-prone changelog creation by producing human-readable outputs and optionally creating pull requests with updated changelogs and release details. With support for CI workflows, optional AI-powered summarization, and fallback heuristics, it streamlines the release process and ensures consistent, professional documentation.
