## Action Summary

The `skill-warden-action` GitHub Action integrates the `skill-warden` security scanner into workflows to automatically analyze AI skill repositories for vulnerabilities such as prompt injection, jailbreak attempts, and token smuggling. It ensures compliance by detecting security issues and advisory violations, uploading results to the GitHub Security tab via SARIF, and providing actionable outputs to block unsafe merges or enforce security standards. This action streamlines AI security scanning, reduces manual effort, and enhances repository safety.
