## Action Summary

This GitHub Action enables users to run CI workflows in an OpenIndiana virtual machine environment, addressing the limitation of GitHub Actions' default support for Ubuntu, Windows, and macOS only. It automates the setup, synchronization, and execution of CI tasks in the VM, ensuring consistent directory structures and environment variable mappings between the host and VM. Key capabilities include configurable VM memory, CPU allocation, file synchronization methods, and network port mapping, providing flexibility for tailored CI scenarios.
