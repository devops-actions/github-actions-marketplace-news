## Action Summary

The **`node-semantic-release`** GitHub Action automates the process of installing dependencies, building a Node.js project, and publishing releases using **`semantic-release`**. It simplifies CI/CD workflows by generating versioned releases, release notes, and changelogs based on commit history, while also providing base64-encoded release notes for safe downstream use. This action streamlines release management, reducing manual effort and ensuring consistent, automated versioning.
