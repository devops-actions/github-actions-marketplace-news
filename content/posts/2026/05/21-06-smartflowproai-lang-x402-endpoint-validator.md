## Action Summary

The **x402 Endpoint Validator** is a GitHub Action designed to automate the validation of x402-compliant API endpoints (paid APIs that quote pricing in `402 Payment Required` responses and accept micropayments). It performs checks on endpoint reachability, manifest format, response body compliance, latency, and payment-required behavior, ensuring adherence to the x402 specification and preventing regressions. The tool eliminates manual testing, streamlining CI workflows and providing enhanced insights like reputation scoring and behavioral analysis with its paid tier.
