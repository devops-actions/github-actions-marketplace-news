## Action Summary

The `agent-bom` GitHub Action is a security scanner tailored for AI supply chains and infrastructure components, including agents, MCP servers, packages, containers, cloud platforms, GPUs, and runtimes. It automates the identification and analysis of vulnerabilities (CVEs) within the AI stack, mapping their blast radius to show the impact on associated packages, servers, agents, credentials, and tools. By providing actionable remediation guidance, it helps users mitigate risks and prevent security breaches.
