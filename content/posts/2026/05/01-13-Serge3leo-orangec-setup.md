## Action Summary

The `orangec-setup` GitHub Action automates the installation and configuration of the OrangeC compiler, ensuring compatibility with CMake. It simplifies the setup process by managing paths, environment variables, and optional patches for OrangeC's CMake support modules. This action streamlines workflows for projects utilizing the OrangeC compiler, particularly in multi-platform environments.
