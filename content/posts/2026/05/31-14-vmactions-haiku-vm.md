## Action Summary

This GitHub Action enables running CI workflows on the Haiku operating system, which is not natively supported by GitHub Actions (limited to Ubuntu, Windows, and macOS). It facilitates tasks such as environment setup, code execution, and file synchronization by provisioning a Haiku virtual machine (VM) and mirroring the host environment within the VM. Key features include support for environment variable passing, customizable VM configurations (e.g., memory, CPU, architecture), and flexible file synchronization methods.
