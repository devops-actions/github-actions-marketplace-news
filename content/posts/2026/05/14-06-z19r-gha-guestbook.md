## Action Summary

The `gha-guestbook` GitHub Action automates the management of a guestbook within a repository's README file by appending guestbook entries from issues labeled with a specific tag (default: `guestbook`). It updates the guestbook entry count in the README and can optionally comment on and close the issue after processing. This action simplifies the process of maintaining an interactive guestbook directly in a repository, enabling user-generated entries to be seamlessly added and tracked.
