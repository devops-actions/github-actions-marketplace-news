## Action Summary

`submark` is a GitHub Action and CLI tool designed to extract specific sections from CommonMark/Markdown documents based on heading levels, titles, or patterns. It automates tasks like retrieving changelog entries for release notes during CI/CD workflows. Key capabilities include precise section extraction, customizable filtering, and output formatting for seamless integration into automated processes.
