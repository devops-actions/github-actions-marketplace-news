## Action Summary

`agent-bom` is an open-source security scanner designed for AI and MCP (Machine Control Plane) infrastructures. It automates the inventorying and analysis of agents, servers, tools, packages, credentials, and runtime environments to identify vulnerabilities, compliance evidence, and exposure paths. Its key capabilities include generating graph-based insights, supporting CLI, REST API, and dashboard interfaces, and providing optional runtime controls for improved security management.
