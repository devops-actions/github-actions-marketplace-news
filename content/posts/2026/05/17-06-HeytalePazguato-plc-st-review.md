## Action Summary

The `plc-st-review` GitHub Action is a semantic linter, code reviewer, and team-style enforcer designed for IEC 61131-3 Structured Text (ST) codebases used in PLC programming. It automates the detection of semantic issues, enforces naming conventions, and performs static analysis, pull request reviews, and team-style enforcement. By identifying bugs and inconsistencies that are often missed during manual reviews, such as changes to timer values, unused variables, or outdated call sites, it streamlines the CI workflow for PLC developers and improves code quality and consistency.
