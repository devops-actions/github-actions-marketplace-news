## Action Summary

The TestivAI GitHub Action enables automated visual regression testing for modern web applications, operating entirely locally or with an optional cloud-based service. It facilitates tasks such as capturing full-page screenshots, performing pixel-based diffs, generating self-contained HTML reports, and managing baselines for streamlined version control.
