## Action Summary

DocLoop is a GitHub Action that uses AI to automatically update documentation whenever code changes are merged in a pull request. It ensures that documentation remains synchronized with the codebase, reducing drift and maintaining trust in the docs. Key capabilities include intelligent doc updates, customizable prompts, automated review routing, and support for multiple delivery modes to seamlessly integrate into existing workflows.
