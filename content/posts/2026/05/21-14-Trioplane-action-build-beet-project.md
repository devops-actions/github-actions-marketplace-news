The `action-build-beet-project` GitHub Action automates the process of building a Beet project by generating data packs, resource packs, and other related files from the specified configuration. It simplifies project workflows by providing structured outputs, such as JSON arrays of file paths, the project name, version, and build output directory, which can be used in subsequent steps like packaging or deployment. This action is ideal for streamlining the build and release process for Beet-based projects.
