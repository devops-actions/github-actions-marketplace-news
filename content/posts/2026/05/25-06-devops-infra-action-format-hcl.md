## Action Summary

This GitHub Action automatically formats HashiCorp Configuration Language (HCL) and Terraform (TF) files to ensure code consistency and adherence to standard formatting practices. It helps teams working with Terraform or Terragrunt by automating the formatting process, reducing manual effort, and minimizing errors caused by improperly formatted files. Key capabilities include recursive file processing, selective file ignoring, and options for checking, displaying diffs, or overwriting files with corrected formatting.
