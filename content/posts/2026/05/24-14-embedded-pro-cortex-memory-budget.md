## Action Summary

The `cortex-memory-budget` GitHub Action performs static analysis of ARM Cortex-M ELF binaries to measure flash and RAM usage, providing detailed breakdowns by memory region, section, symbol, and source file. It automates resource usage reporting, detects pre-allocated stack and heap, and supports diffing against baseline binaries, making it ideal for monitoring and optimizing memory usage in resource-constrained MCU projects. The action is CI-friendly, framework-agnostic, and generates human-readable reports, machine-readable metrics, and optional PR comments for streamlined collaboration.
