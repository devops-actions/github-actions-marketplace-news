## Action Summary

The **`node-semantic-release`** GitHub Action automates the process of installing dependencies, building a Node.js project, and performing a release using `semantic-release`. It streamlines versioning, changelog generation, and publishing by leveraging semantic versioning practices, while also providing base64-encoded release notes for safe downstream usage. This action simplifies release workflows, reduces manual effort, and ensures consistent release management in CI/CD pipelines.
