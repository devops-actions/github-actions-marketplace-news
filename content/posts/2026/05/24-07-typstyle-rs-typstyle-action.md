## Action Summary

The **typstyle GitHub Action** automates the formatting of `.typ` files in a repository using the Typst formatter, ensuring consistent code style across the project. It is designed to streamline code formatting tasks by automatically formatting files and optionally committing the changes, reducing manual effort and improving workflow efficiency. This action is particularly useful for maintaining clean and standardized Typst files in collaborative projects.
