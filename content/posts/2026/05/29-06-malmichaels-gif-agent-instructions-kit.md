## Action Summary

The **agent-instructions-kit** GitHub Action automates the creation, validation, and maintenance of agent instruction files (e.g., AGENTS.md, CLAUDE.md, GEMINI.md) for repositories. It ensures these files are consistent, up-to-date, and secure by generating templates tailored to your project stack, enforcing structure and quality standards, and detecting safety risks like prompt injection or security vulnerabilities. This solution helps streamline agent-related workflows, reduce file drift, and prevent unintended behavior in AI coding tools.
