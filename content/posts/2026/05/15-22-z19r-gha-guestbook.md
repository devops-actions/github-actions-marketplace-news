## Action Summary

The **gha-guestbook** GitHub Action automates the process of maintaining a guestbook within a project's README file by appending entries from issues labeled with a specified tag (default: `guestbook`). It updates the guestbook entry count in the README, optionally comments on and closes the issue, and provides metadata such as the author and entry count. This action streamlines guestbook management and eliminates the need for manual updates.
