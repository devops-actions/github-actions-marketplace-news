## Action Summary

The **HF-MS Sync** GitHub Action automates the bidirectional synchronization of AI models and datasets between HuggingFace and ModelScope repositories. It streamlines the process of keeping repositories up-to-date by detecting and transferring only changed files, resolving conflicts with configurable strategies, and supporting large files via Git LFS. The action is ideal for managing multi-platform model and dataset workflows, offering features like dry-run mode, state persistence, and detailed job summaries.
