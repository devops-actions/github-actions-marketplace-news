## Action Summary

The `bilig` GitHub Action identifies and resolves stale formula values in `.xlsx` files edited via Node.js libraries such as SheetJS, ExcelJS, or `xlsx-populate`, which do not automatically recalculate formulas. It automates the detection and recalculation of outdated formula results, ensuring accurate and up-to-date values in Excel workbooks used for tasks like data validation, financial modeling, and CI workflows. Key capabilities include inspecting workbooks for stale formulas, generating JSON reports, and integrating seamlessly into GitHub-based workflows.
