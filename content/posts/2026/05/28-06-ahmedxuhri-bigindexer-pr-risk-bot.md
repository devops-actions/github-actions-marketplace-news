## Action Summary

The **Big Indexer PR Risk Bot** is a GitHub Action designed to automate architecture-aware pull request (PR) reviews by analyzing code changes and identifying potential risks such as blast radius, high-coupling seams, and impacted clusters. It helps teams detect architectural issues early, improving code quality and reducing technical debt. Key capabilities include generating risk insights, optional twin-based hints, and automated PR commenting for streamlined feedback.
