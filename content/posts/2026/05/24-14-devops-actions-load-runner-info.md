## Action Summary

This GitHub Action, `load-runner-info`, retrieves detailed information about self-hosted runners for a specified organization or repository, including their statuses and label groupings. It automates the process of monitoring runner availability and status, helping users verify that sufficient runners are online and properly labeled for workflows. Key features include the ability to output runner data as JSON or save it to files for large datasets, enabling efficient runner management and status checks.
