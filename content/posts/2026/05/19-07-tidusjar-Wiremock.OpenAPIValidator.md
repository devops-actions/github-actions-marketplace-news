## Action Summary

The **WireMock OpenAPI Validator GitHub Action** automates the validation of WireMock stub mappings against OpenAPI v3 specifications within GitHub Actions workflows. It ensures that mock APIs remain aligned with API contracts by providing detailed validation results, including summaries, pass rates, and failure details, and can post updates directly to pull requests. This action streamlines API contract testing in CI/CD pipelines, reducing manual effort and improving API reliability.
