## Action Summary

The **Email HTML Validator** GitHub Action provides a streamlined solution for validating email newsletter templates by analyzing inline HTML, files, URLs, or input streams. It identifies supported, partially supported, and unsupported email features based on the "Can I Email" dataset, while also offering optional accessibility audits using Playwright and axe-core. The action automates validation in CI/CD workflows, generates detailed reports in multiple formats (JSON, XML, HTML, Markdown), and can be used via CLI, Java DSL, Docker, or as a GitHub Action.
