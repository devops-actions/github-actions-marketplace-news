## Action Summary

The `fish-shop/indent-check` GitHub Action automates the process of verifying proper indentation in `.fish` shell scripts. It scans specified files or directories, supports patterns with wildcards and brace expansions, and provides detailed outputs summarizing the number of files checked, passed, and failed. This action helps ensure consistent and correct formatting in `fish` shell scripts, streamlining code review and quality assurance processes.
