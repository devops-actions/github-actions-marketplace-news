## Action Summary

The `@zeropress/build-pages` GitHub Action automates the process of converting a directory of Markdown files and public assets into a fully static ZeroPress website. It streamlines tasks such as discovering Markdown pages, preparing site data, staging public files, and generating deployable static files compatible with platforms like GitHub Pages, Netlify, and Vercel. This action is designed to simplify static site generation and deployment workflows for documentation or content-rich websites.
