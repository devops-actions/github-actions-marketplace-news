## Action Summary

The **OpenCart 4 Setup Action** provides a fast and streamlined way to set up a pre-configured OpenCart 4 environment for continuous integration and deployment (CI/CD) workflows. By leveraging pre-built container images from the GitHub Container Registry (GHCR), it eliminates repetitive tasks such as PHP extension installation and source file extraction, significantly reducing setup time to under 15 seconds. This action automates the provisioning, database configuration, and health checks needed to quickly prepare an environment for testing or deployment.
