## Action Summary

The "skeptic" GitHub Action is a local repository trust auditor designed to detect structural vulnerabilities within CI/CD pipelines, ecosystem dependencies, and low-review attack surfaces. It addresses gaps left by traditional security tools like CVE scanners, SAST, and secret scanners by identifying attack-enabling conditions and correlating risks across files, directories, and repositories. Key capabilities include behavior chain analysis, cross-finding correlation, payload decoding, identity graph traversal, and entropy-based anomaly detection to enhance supply chain security and prevent cascading compromises.
