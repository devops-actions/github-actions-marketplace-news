## Action Summary

The `iac-cartographer` GitHub Action automates the discovery, documentation, and publishing of fleet-level insights for Terraform and IaC codebases across various repository platforms. It extracts structural metadata, generates purpose summaries using LLMs, and publishes organized documentation in formats like Confluence, Notion, Markdown, or GitHub Wiki. This action addresses challenges like maintaining up-to-date infrastructure documentation, improving onboarding for engineers, and surfacing actionable signals for IaC repositories, all while allowing flexible integration with different platforms and tools.
