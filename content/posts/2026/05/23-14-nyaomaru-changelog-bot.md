## Action Summary

The `@nyaomaru/changelog-bot` GitHub Action automates the generation of polished changelog entries by leveraging Git history, PR titles, and release notes. It eliminates manual effort and errors in creating changelogs by integrating with CI/CD pipelines or running locally, while optionally using AI (OpenAI or Anthropic) for enhanced, tone-aware summaries. Key capabilities include generating PR-ready changelogs, handling duplicate versions, maintaining compare links, and providing fallback mechanisms when AI services are unavailable.
