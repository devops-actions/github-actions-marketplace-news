## Action Summary

The `mdsf` GitHub Action automates the formatting and linting of code snippets embedded in markdown files using user-specified tools. It helps ensure consistent code formatting and quality across documentation and markdown files, reducing manual effort and errors. Key capabilities include formatting, verifying formatting, and integrating with various tools and workflows, such as GitHub Actions, editors, and pre-commit hooks.
