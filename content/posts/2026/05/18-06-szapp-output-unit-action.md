## Action Summary

The **Output Unit GitHub Action** automates the process of parsing Daedalus script files to extract `AI_Output` commands and generate a valid output unit cutscene library (CSL) file in ASCII format. It simplifies the creation of these files, providing a streamlined way to integrate their generation into CI/CD workflows, where they can be committed back to the repository or used as artifacts. This action is particularly useful for automating tasks in projects that rely on Daedalus scripting for dialogue or cutscene management.
