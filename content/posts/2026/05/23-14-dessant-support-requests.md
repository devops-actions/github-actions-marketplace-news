## Action Summary

Support Requests is a GitHub Action designed to automate issue management for repositories by handling issues labeled as support requests. It streamlines workflows by automatically commenting on, closing, and optionally locking these issues, while also reopening and unlocking issues if the support label is removed. This action helps maintain cleaner issue trackers by directing support-related queries to appropriate channels.
