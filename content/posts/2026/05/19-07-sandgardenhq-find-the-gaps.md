## Action Summary

"Find the Gaps" is a CLI tool that analyzes a codebase and its associated documentation to identify discrepancies such as outdated or missing documentation. By extracting symbols (e.g., functions, types, exports) from source code in multiple programming languages, it cross-references them with existing documentation to ensure alignment. This tool automates the process of maintaining accurate and up-to-date documentation, saving developers time and reducing the risk of inconsistencies.
