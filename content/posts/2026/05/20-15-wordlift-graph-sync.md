## Action Summary

The `graph-sync` GitHub Action automates the setup and execution of `worai` for graph synchronization tasks, including installing dependencies like Python Playwright and Chromium. It simplifies workflows by managing configuration resolution, caching, and dependency installation, while supporting various input sources like URLs, sitemaps, and Google Sheets. This action is particularly useful for users needing streamlined and repeatable graph sync operations with integrated debugging and logging capabilities.
