## Action Summary

The `@zeropress/build-pages` GitHub Action automates the process of converting a directory of Markdown files and public assets into a static ZeroPress site optimized for deployment on modern hosting platforms like GitHub Pages, Netlify, or Vercel. It simplifies website generation by discovering Markdown content, preparing site data, managing public assets, and producing static HTML files, assets, and supplemental files (e.g., sitemap.xml) ready for hosting. This action streamlines the workflow for creating and deploying documentation or static websites.
