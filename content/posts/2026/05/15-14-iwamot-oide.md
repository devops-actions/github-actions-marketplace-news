## Action Summary

Oide is a GitHub Action that automates the process of pulling specific files from a source repository, as defined in an `Oidefile`, into a target repository. It is designed to facilitate the synchronization of shared files (e.g., licenses, security policies, CI configurations) across multiple repositories, streamlining updates and ensuring consistency. Key features include support for both public and private source repositories, self-skipping to avoid conflicts, and optional integration with Renovate for automated version management.
