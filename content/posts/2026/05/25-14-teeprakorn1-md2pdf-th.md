## Action Summary

The `md2pdf-th` GitHub Action is a versatile tool that converts Markdown files into professionally styled PDFs with robust support for both Thai and English text. It automates tasks such as font detection, metadata management, watermarking, batch processing, and PDF merging, while also offering features like themes, templates, table of contents generation, and web preview. Designed for CLI, Node.js, and NestJS environments, it streamlines document generation workflows for developers needing localized or customized PDF outputs.
