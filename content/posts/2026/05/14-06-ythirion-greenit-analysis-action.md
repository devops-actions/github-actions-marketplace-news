## Action Summary

The **GreenIT Analysis GitHub Action** automates the evaluation of web applications' environmental impact using the GreenIT-Analysis CLI. It analyzes specified URLs, generates EcoIndex scores, creates a detailed HTML report, and enforces a configurable quality gate that fails the CI pipeline if any URL score falls below the set threshold. This action streamlines the integration of environmental impact assessments into CI pipelines, promoting sustainable web development practices.
