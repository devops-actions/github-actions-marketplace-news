## Action Summary

The `go-rpcatlas` GitHub Action automates the generation of static RPC/API call maps for Go-based services. It leverages AST-based static analysis to create deterministic Markdown or JSON summaries of RPC handlers, including call trees, decision points, and categorized function layers, without executing the target service. This action simplifies code reviews, onboarding, and AI-assisted code analysis by providing structured insights into RPC behavior and architecture.
