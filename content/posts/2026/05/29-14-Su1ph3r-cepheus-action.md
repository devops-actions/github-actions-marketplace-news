## Action Summary

The `cepheus-action` GitHub Action integrates the Cepheus container escape scanner into CI/CD workflows to automatically detect security vulnerabilities in container images. It scans a specified Docker or Podman image, generates a SARIF report, and uploads it to GitHub Code Scanning for review. The action helps automate security checks by failing builds if vulnerabilities meet a specified severity threshold or if regressions are detected compared to a baseline report.
