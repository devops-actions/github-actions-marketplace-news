## Action Summary

The **Walrus Sites Provenance** GitHub Action automates the deployment of decentralized static websites to the Walrus storage protocol on the Sui blockchain. It supports reproducible CI deployment, integrates with provenance workflows for verifiable builds, and enables secure remote signing to keep private keys off GitHub runners. This action streamlines the process of publishing verifiable on-chain assets while ensuring enhanced security and traceability for website deployments.
