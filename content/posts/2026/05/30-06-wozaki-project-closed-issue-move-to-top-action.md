## Action Summary

This GitHub Action automates the process of managing closed issues in GitHub Project V2 boards. When an issue is closed, it updates the issue's status to a specified column (e.g., "Done") and moves it to the top of that column, ensuring recently closed issues are prominently displayed. This streamlines project board organization and improves visibility of completed tasks.
