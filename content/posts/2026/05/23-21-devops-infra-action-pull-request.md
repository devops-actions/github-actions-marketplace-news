## Action Summary

This GitHub Action automates the creation of pull requests from a specified source branch to a target branch, allowing customization of titles, descriptions, assignees, reviewers, labels, milestones, and draft status. It streamlines the pull request process by optionally integrating commit details, modified files, and automated template updates, making it ideal for CI/CD workflows. Additionally, it supports execution even when branches have no differences and is compatible with both `amd64` and `arm64` architectures.
