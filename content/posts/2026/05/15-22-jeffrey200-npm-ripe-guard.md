## Action Summary

The **npm-ripe-guard** GitHub Action automatically blocks the installation of npm packages released within the last 24 hours, mitigating supply-chain attacks that exploit the critical window before malicious packages are detected. It acts as a transparent proxy between npm/pnpm and the npm registry, enforcing a 24-hour quarantine, while also attempting to enable package-manager-level release-age filtering for additional protection. Key capabilities include handling scoped packages and dist-tags, caching metadata, and seamlessly integrating with GitHub Actions workflows.
