## Action Summary

The `pr-title-check` GitHub Action automates the validation of pull request titles by enforcing structural rules such as emoji prefixes, minimum length, maximum character limits, and banned phrases. It helps maintain consistency, professionalism, and clarity in PR titles, and optionally leverages AI (Gemini Flash) to assess semantic quality for more meaningful titles. The action is easy to implement with default configurations and supports customization via YAML for tailored rules.
