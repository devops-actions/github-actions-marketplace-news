## Action Summary

The `release-actions` GitHub Action automates the release process for GameMaker library authors by building a `.yymps` package from a specified project folder, attaching it to a GitHub release, and auto-generating a detailed release description with closed issues and recent commits. It simplifies the packaging and versioning workflow, ensuring consistent and efficient releases directly from tagged pushes.
