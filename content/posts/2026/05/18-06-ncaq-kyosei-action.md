## Action Summary

The `kyosei-action` GitHub Action wraps the `kyosei` AI code review plugin, providing an automated solution for reviewing pull requests in terms of code quality, performance, security, test coverage, and documentation accuracy. It addresses challenges like redundant or inappropriate comments in re-reviews and supports project-specific coding conventions by excluding defaults that may not apply. This action enables seamless CI integration for iterative, context-aware code reviews.
