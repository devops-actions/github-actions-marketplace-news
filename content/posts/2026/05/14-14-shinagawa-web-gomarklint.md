## Action Summary

The **gomarklint** GitHub Action is a Markdown linter designed to catch issues like broken links, validate internal and external anchors, and enforce consistent document structure. It automates the process of identifying formatting errors and link integrity problems in Markdown files, making it suitable for continuous integration workflows. Key capabilities include high performance (processing large files quickly), customizable rules, and outputs that are human- and machine-readable.
