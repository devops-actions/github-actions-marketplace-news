## Action Summary

The `fish-shop/install-plugin` GitHub Action automates the installation of plugins for the Fish shell using supported plugin managers like Fisher or Oh My Fish. It simplifies the process of managing Fish shell plugins in CI workflows by ensuring the specified plugin manager is installed (if not already present) and then installing the required plugins. This action is particularly useful for streamlining shell environment setup and ensuring consistent plugin configurations across development environments.
