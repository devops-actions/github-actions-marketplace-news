## Action Summary

The `node-semantic-release` GitHub Action automates the process of installing dependencies, building a Node.js project, and managing releases via the `semantic-release` tool. It simplifies release workflows by handling tasks such as creating git tags, generating release notes (with optional base64 encoding for downstream use), and publishing GitHub Releases. The action supports two modes: a default release mode for end-to-end automation and a publish-only mode for creating releases from existing tags.
