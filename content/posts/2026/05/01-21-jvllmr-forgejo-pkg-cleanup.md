## Action Summary

The `forgejo-pkg-cleanup` GitHub Action automates the cleanup of Forgejo packages by applying customizable rules, such as version retention patterns and retention periods. It helps streamline package management by removing outdated or unnecessary package versions, ensuring repositories remain organized and optimized. This action is particularly useful for maintaining clean and efficient container or package repositories in Forgejo environments.
