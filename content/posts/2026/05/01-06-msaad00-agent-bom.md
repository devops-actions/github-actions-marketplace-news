## Action Summary

The `agent-bom` GitHub Action is an open security scanner designed to analyze the AI supply chain and infrastructure, including agents, MCP servers, packages, containers, cloud services, GPUs, and runtimes. It identifies vulnerabilities (CVEs) and maps their impact across components, revealing potential credential leaks and security risks while recommending fixes to mitigate these issues. The tool automates end-to-end vulnerability tracking and remediation, providing detailed insights into the blast radius of each identified CVE.
