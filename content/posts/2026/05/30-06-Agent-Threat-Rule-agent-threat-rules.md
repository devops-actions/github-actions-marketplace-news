## Action Summary

The ATR (Agent Threat Rules) GitHub Action provides an open, vendor-neutral detection rule format for identifying security threats in AI agents, similar to how Sigma and YARA operate for SIEM and malware detection. It automates threat detection by leveraging machine-readable YAML rules that conform to a standardized schema, enabling consistent, peer-reviewed, and scalable security practices. This action includes reference engines in TypeScript and Python, supporting integration into workflows for enhanced AI agent security monitoring and compliance.
