## Action Summary

The *Setup cosign* GitHub Action automates the installation and verification of the `sigstore/cosign` binary on Linux and macOS GitHub Actions runners, adding it to the system PATH. It solves the problem of securely and efficiently setting up `cosign` by verifying the binary's authenticity using a pinned public key and leveraging caching to optimize subsequent runs. Key capabilities include automatic version resolution, integrity verification, and outputting the installation path for seamless integration into workflows.
