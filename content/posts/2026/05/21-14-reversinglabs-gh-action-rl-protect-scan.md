## Action Summary

The ReversingLabs GitHub Action `rl-protect` automates the scanning of open-source dependencies in CI/CD workflows to identify and block vulnerabilities and security risks before build processes. It leverages the `rl-protect` CLI tool to analyze package manifest files or specific software package URLs, using the Spectra Assure API for up-to-date threat intelligence. This action provides a proactive "shift-left" solution for securing software supply chains by ensuring only safe and compliant dependencies are used.
