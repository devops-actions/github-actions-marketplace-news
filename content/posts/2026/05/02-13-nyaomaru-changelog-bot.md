## Action Summary

The `@nyaomaru/changelog-bot` GitHub Action automates the generation of professional changelog entries by analyzing Git history, PR titles, and release notes, with optional integration of AI-generated summaries for enhanced readability. It eliminates manual tasks like copy-pasting and ensures accurate changelog updates, including the ability to create pull requests with pre-filled compare links and release notes. This action is designed for seamless integration into CI pipelines or local workflows, offering robust defaults and fallback mechanisms to streamline release management.
