## Action Summary

Knowledge Diff is a GitHub Action designed to identify and address discrepancies between code changes and project documentation. By analyzing pull request diffs, it detects contradictions between updated code and related documentation, leveraging AI to suggest corrections and optionally automating documentation updates. This action helps teams maintain accurate and up-to-date documentation, reducing confusion and improving onboarding efficiency.
