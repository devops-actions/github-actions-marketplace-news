## Action Summary

The **github-screenshot-action** automates the process of capturing, monitoring, and versioning website screenshots based on a JSON-defined list of URLs. It supports parallel execution, retry logic, and cron scheduling to streamline monitoring workflows, and it can optionally create pull requests with updated screenshots. This action is ideal for tracking visual changes in websites over time, leveraging a Puppeteer and Chromium-based stack for precise rendering.
