## Action Summary

The `zeropress-build-pages` GitHub Action automates the process of building static websites from Markdown files and public assets using the ZeroPress framework. It streamlines tasks such as discovering markdown pages, preparing site data, staging public files, and generating fully static outputs (HTML, assets, and metadata) ready for deployment to platforms like GitHub Pages, Netlify, or Vercel. This action simplifies the workflow for creating and deploying modern, static websites.
