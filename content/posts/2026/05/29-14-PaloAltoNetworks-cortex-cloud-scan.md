## Action Summary

The **Cortex Cloud Scan GitHub Action** integrates with the Cortex CLI to automate security scanning for code, container images, and APIs. It detects vulnerabilities, malware, secrets, and misconfigurations, streamlining the process of identifying and addressing security risks in software projects. Key capabilities include code security scanning (SCA, IaC), container workload protection, and API security testing, with automatic retrieval of the appropriate Cortex CLI version for seamless integration.
