## Action Summary

The `fish-shop/run-fishtape-tests` GitHub Action automates the execution of [Fishtape](https://github.com/jorgebucaran/fishtape) test suites for `.fish` scripts within a repository. It simplifies test management by supporting customizable file pattern matching, wildcards, and brace expansion, while optionally formatting test output for improved readability. This action is particularly useful for developers working with the Fish shell to ensure their scripts are properly tested in CI/CD workflows.
