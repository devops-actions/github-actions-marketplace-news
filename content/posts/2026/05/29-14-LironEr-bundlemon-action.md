## Action Summary

The **BundleMon GitHub Action** integrates BundleMon with GitHub Actions to monitor and track your project's bundle size on every commit or pull request. It automates the process of analyzing bundle size changes, providing visibility into size impacts of code changes to help maintain performance and prevent regressions. Key capabilities include support for multi-project configurations and the ability to fail workflows on detected issues, ensuring consistent bundle size monitoring across projects.
