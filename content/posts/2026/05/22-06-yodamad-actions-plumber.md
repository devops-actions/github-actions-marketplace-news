## Action Summary

The Plumber GitHub Action automates the analysis of GitHub Actions workflows and repository settings to ensure CI/CD compliance and identify supply chain risks. It evaluates factors such as branch protection rules, workflow permissions, secret management, and use of pinned actions, generating a compliance score, a SARIF report for GitHub Code Scanning, and a Pipeline Bill of Materials (PBOM). This action helps streamline security reviews and improve repository governance by providing actionable insights and reports.
