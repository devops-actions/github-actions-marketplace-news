## Action Summary

ActionScope is a GitHub Action and CLI tool that analyzes GitHub Actions workflows, Terraform IAM resources, and inline JSON IAM policies to map the potential security impact ("blast radius") of your CI/CD pipelines on your AWS environment. It helps identify what an attacker could do in AWS if a workflow is compromised, providing a plain-English report of risks and permissions. Key features include AWS credential source detection, workflow-to-IAM role correlation, and risk classification of IAM actions to enhance workflow security and mitigate potential vulnerabilities.
