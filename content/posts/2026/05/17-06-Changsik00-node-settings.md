## Action Summary

This GitHub Action, **@env-kit/node-settings**, streamlines environment configuration management for Node.js applications using a schema-first approach. By leveraging a single Zod schema, it automates the generation of typed runtime configurations, `.env.example` files, Markdown documentation, Kubernetes manifests, and a CLI for deployment validation in CI pipelines. This action is designed to reduce manual maintenance and ensure consistency across multiple environments, simplifying tasks like managing environment variables, secrets, and dynamic configuration overrides.
