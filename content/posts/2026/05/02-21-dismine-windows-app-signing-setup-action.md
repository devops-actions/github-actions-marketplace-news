## Action Summary

The **Windows App Signing Setup Action** automates the preparation of a Windows GitHub Actions environment for secure and unattended code signing using Certum's SimplySign cloud signing service. It installs the required SimplySign software, configures system settings, handles TOTP-based authentication, and verifies the signing certificate. This action simplifies and streamlines the process of integrating automated Windows application code signing into CI workflows.
