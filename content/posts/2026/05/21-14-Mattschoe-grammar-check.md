## Action Summary

The **`grammar-check` GitHub Action** automates grammar and spelling corrections in prose files (e.g., `.md`, `.tex`) within a repository by leveraging a language model (LLM). On every push, it identifies modified files, applies corrections while preserving code, math, and markup, and creates a pull request with the proposed changes for review. This action streamlines the editing process, ensures content quality, and keeps users in control of what gets merged.
