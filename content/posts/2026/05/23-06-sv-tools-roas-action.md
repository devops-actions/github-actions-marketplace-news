## Action Summary

The **roas-action** GitHub Action automates the validation, conversion, and manipulation of OpenAPI specifications (Swagger 2.0, OpenAPI 3.0.x/3.1.x/3.2.x) and OpenAPI Overlay documents (Overlay 1.0/1.1). It simplifies tasks like validating API specifications, converting them between versions, merging multiple specs, and applying overlays to modify or enhance specifications. This action streamlines API documentation workflows, ensuring compliance with standards and supporting advanced transformations in a CI/CD pipeline.
