## Action Summary

AssertIQ is a GitHub Action and CLI tool that provides static analysis and reporting for JavaScript and TypeScript test suites without executing tests or requiring project-specific configuration. It automates the evaluation of test quality by scoring dimensions like assertion quality, flakiness risk, naming clarity, coverage balance, and dead test risk, and generates actionable insights through reports and badges. This helps developers identify and address potential weaknesses in their test suites to improve code reliability and maintainability.
