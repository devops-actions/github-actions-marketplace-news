## Action Summary

ActionScope is a GitHub Action designed to analyze CI/CD workflows and associated IAM policies to identify potential security risks in AWS environments. It automates the mapping of AWS permissions and capabilities derived from GitHub Actions workflows, providing a plain-English "blast radius" report to highlight what an attacker could do if a workflow is compromised. Key features include correlation of IAM roles with workflows, risk classification of AWS actions, and detection of security vulnerabilities such as privilege escalation paths.
