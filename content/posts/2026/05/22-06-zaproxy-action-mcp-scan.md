## Action Summary

This GitHub Action, **Action MCP Scan**, automates security testing for MCP (Model Context Protocol) servers using OWASP ZAP's Automation Framework and MCP add-on. It performs active and passive scans, generates reports in multiple formats (JSON, HTML, Markdown), and optionally creates a GitHub issue with the scan results. This action helps identify vulnerabilities in MCP servers while streamlining the scanning process, ensuring proper authorization and compatibility with test environments.
