## Action Summary

This GitHub Action enables users to run continuous integration (CI) workflows on OpenIndiana, an operating system not natively supported by GitHub Actions. It automates the setup and execution of CI tasks within an OpenIndiana virtual machine, synchronizing files and environment variables between the host and VM. Key features include support for different OpenIndiana releases, customizable VM resources (CPU and memory), and multiple file synchronization methods.
