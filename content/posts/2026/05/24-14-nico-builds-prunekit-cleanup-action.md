## Action Summary

PruneKit Codebase Cleanup is a GitHub Action designed to automate the process of cleaning up repositories by identifying and optionally removing unused code, redundant files, stale configurations, and outdated documentation. It generates reports with prioritized action plans and supports optional application of cleanups, such as file removals, documentation merges, and link fixes. This action streamlines repository maintenance and improves codebase organization, leveraging AI-powered insights for efficient cleanup.
