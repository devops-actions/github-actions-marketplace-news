## Action Summary

The **SEC Checks - SAST & Supply Chain Scanner** GitHub Action automates static application security testing (SAST) using Semgrep and audits `package.json` lifecycle scripts to detect potential supply chain attacks. It helps developers identify security vulnerabilities in code across multiple languages and flags suspicious behaviors in build scripts, such as dynamic code execution or downloading remote content. Key capabilities include customizable scanning rules, severity filtering, artifact uploads, and a detailed job summary in the GitHub Actions UI.
