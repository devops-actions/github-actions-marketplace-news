## Action Summary

This GitHub Action automates the creation of self-hosted GitHub Actions runners on Hetzner Cloud servers, launching a server just before a workflow begins and terminating it upon completion. It enables users to reduce CI/CD costs, bypass GitHub Actions usage limits, and customize the hardware and software environment of their runners. The action offers an efficient and cost-effective alternative to GitHub-managed runners, particularly for organizations with high or specialized CI/CD needs.
