## Action Summary

The Blackout Secure Sitemap Generator is a GitHub Action designed for automated creation and validation of sitemaps (XML, TXT, GZIP) to enhance SEO and website indexing. It supports static sites, SSG frameworks (e.g., Next.js, Gatsby), and dynamic applications, offering capabilities like URL discovery, canonical parsing, last modification tracking, and auto-splitting for large sites. This action streamlines sitemap generation, ensures compliance with sitemaps.org protocols, and eliminates the need for manual updates or builds.
