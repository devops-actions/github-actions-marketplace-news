## Action Summary

The **Azure Pipelines Dependency Submission** GitHub Action automates the process of analyzing Azure Pipelines configurations and submitting task dependencies to GitHub's Dependency Graph for vulnerability scanning and Dependabot alerts. It identifies and resolves pipeline definitions, including templates, and integrates with Azure DevOps APIs to retrieve detailed task metadata. This action enables proactive security monitoring by detecting vulnerable dependencies in Azure Pipelines workflows.
