## Action Summary

The **GHCR Untag Action** is a GitHub Action designed to remove specific tags from a container package in the GitHub Container Registry (GHCR), addressing the lack of a native "remove tag only" feature. It automates the process by creating a temporary copy of the manifest with the specified tag before deleting the tag, ensuring the rest of the package remains intact. This action simplifies tag management in GHCR, making it easier to clean up or update container tags without affecting the entire package.
