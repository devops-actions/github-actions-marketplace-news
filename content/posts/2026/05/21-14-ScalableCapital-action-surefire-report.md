## Action Summary

This GitHub Action processes JUnit-compatible XML test reports (e.g., Maven Surefire, Gradle, Pytest, Go) and publishes the results as GitHub check runs, workflow-native annotations, or job summaries. It automates the reporting of test results directly in GitHub workflows, improving visibility into test outcomes and aiding debugging. Key capabilities include flexible reporting modes, compatibility with forked pull requests, and support for custom summary annotations.
