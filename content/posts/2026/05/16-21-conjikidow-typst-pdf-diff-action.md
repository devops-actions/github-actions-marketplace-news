## Action Summary

The **Typst PDF Diff Action** is a GitHub Action designed to automate the generation of PDF diffs for Typst documents by comparing changes between a base and head revision in pull requests or other Git events. It builds Typst documents, creates visual diff PDFs using `diff-pdf`, uploads the resulting files as workflow artifacts, and optionally updates pull request comments with the diff results. This action streamlines the process of reviewing document changes and ensures efficient collaboration in projects involving Typst documents.
