## Action Summary

Anno is a GitHub Action that utilizes large language models (LLMs) to automate the creation of concise summaries for code releases and pull requests. It generates release summaries by analyzing code changes between workflow runs and can post them to Slack, with optional Jira integration for linking relevant issues. Additionally, it provides brief pull request summaries, streamlining communication and improving collaboration in development workflows.
