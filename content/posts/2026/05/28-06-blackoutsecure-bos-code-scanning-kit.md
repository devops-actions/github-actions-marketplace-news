## Action Summary

The **Blackout Secure Code Scanning Kit** is a GitHub Action designed to automate comprehensive code and repository security audits. It detects the technologies present in a repository, runs context-appropriate scanners (e.g., for secrets, workflows, and shell scripts), performs posture audits for repository settings (e.g., branch protection, permissions, and reviews), and consolidates all findings into a single SARIF file uploaded to GitHub Advanced Security. This action simplifies security workflows by providing an all-in-one solution for identifying vulnerabilities, enforcing best practices, and enhancing repository compliance.
