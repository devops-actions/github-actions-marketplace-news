## Action Summary

The `setup-tombi` GitHub Action streamlines the installation and configuration of the Tombi CLI tool in workflows. It automates the setup process by allowing users to specify or dynamically resolve Tombi versions, verify downloads via checksum, and optimize performance with caching options. This action simplifies TOML file validation tasks and ensures consistent tool setup across CI/CD pipelines.
