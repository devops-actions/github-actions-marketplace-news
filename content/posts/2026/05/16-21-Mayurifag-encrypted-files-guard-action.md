## Action Summary

The **Encrypted Files Guard Action** is a GitHub Action designed to prevent encrypted files from being accidentally committed in plaintext. It automatically scans tracked files for encryption compliance, focusing on files specified in `.gitattributes` with `filter=git-crypt` and `.ejson` files. This action enhances repository security by identifying potential misconfigurations without exposing sensitive data.
