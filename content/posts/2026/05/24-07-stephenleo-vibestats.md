## Action Summary

VibeStats is a GitHub Action designed to track and preserve usage metrics for Claude Code and Codex sessions, providing long-term analytics that extend beyond the default 30-day session retention limit. It automates the synchronization of aggregated daily statistics (e.g., tokens, sessions, and usage breakdowns) to a private GitHub repository, ensuring data persistence, privacy, and accessibility across devices. The action enables users to visualize their activity with a GitHub heatmap and a comprehensive analytics dashboard at `vibestats.dev/<username>`.
