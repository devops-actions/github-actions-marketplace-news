## Action Summary

The **SQLi Security Scan** GitHub Action is a tool designed to detect SQL Injection vulnerabilities in REST APIs by analyzing their Swagger/OpenAPI specifications. It automates the process of generating context-aware SQL Injection payloads using a GAN (Generative Adversarial Network) model, testing endpoints and parameters, and classifying the results into detailed reports. This action streamlines vulnerability scanning, enhances security testing, and helps identify potential SQLi risks in APIs.
