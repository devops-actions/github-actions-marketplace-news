## Action Summary

Heatcheck Action is a GitHub Action designed for static taint analysis in Python and Go projects, primarily targeting security vulnerabilities like SQL injection, command injection, SSRF, and other common attack vectors. It automates the detection of untrusted inputs, including outputs from language models (LLMs), flowing into unsafe code sinks, with a focus on issues prevalent in AI-assisted coding. The action provides inline PR annotations, code scanning alerts, workflow summaries, and customizable job failure handling to enhance security and streamline the code review process.
