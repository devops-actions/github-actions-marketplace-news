## Action Summary

The **ctxloom PR review GitHub Action** automates pull request reviews by analyzing code changes directly within your CI pipeline, without relying on external services or costs. It uses a local dependency graph and git history to identify files with high blast radius, suggest reviewers based on historical ownership, and uncover co-change patterns. This action streamlines the review process, prioritizes risk areas, and enhances collaboration while maintaining full privacy by performing all analysis locally.
