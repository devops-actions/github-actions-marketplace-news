## Action Summary

This GitHub Action, **SEC Checks - SAST & Supply Chain Scanner**, performs static application security testing (SAST) using Semgrep and audits `package.json` lifecycle scripts to detect potential supply chain attacks. It automates the identification of code vulnerabilities across multiple languages and malicious script patterns, providing a customizable solution for improving software security. Key capabilities include generating detailed findings reports, configurable thresholds, artifact uploads, and the ability to suppress or trust specific actions or workflows.
