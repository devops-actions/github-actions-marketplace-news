## Action Summary

The **bulk-github-repo-sync-action** automates the process of synchronizing repositories between source and target GitHub organizations using mirror cloning. It simplifies tasks such as creating target repositories if they don't exist, managing repository visibility, disabling GitHub Actions, and archiving repositories post-sync. This action is particularly useful for managing large-scale repository migrations or keeping repositories in sync across GitHub instances, including GitHub Enterprise Server.
