## Action Summary

ActionScope is a security analysis tool for GitHub Actions workflows that evaluates AWS permissions and identifies potential vulnerabilities in CI/CD pipelines. It automates the detection of risks such as compromised actions, misconfigured OIDC trust policies, privilege escalation paths, unpinned actions, and other attack surfaces, providing actionable insights in plain English without requiring AWS credentials. By mapping the "blast radius" of workflows, it helps users mitigate security threats and improve the overall resilience of their pipelines.
