## Action Summary

Plumber is a compliance scanner designed for GitLab CI and GitHub Actions pipelines, enabling automated validation of CI/CD workflows against customizable Rego policies. It streamlines compliance by normalizing pipeline configurations into a provider-agnostic format, analyzing them locally or via API, and reporting policy violations. By supporting both GitLab and GitHub providers with a unified configuration file and policy engine, Plumber helps ensure consistency, security, and best practices across CI/CD pipelines.
