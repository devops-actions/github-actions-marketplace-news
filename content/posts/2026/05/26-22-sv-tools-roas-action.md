## Action Summary

The `roas-action` GitHub Action automates the validation, conversion, and manipulation of OpenAPI specifications (Swagger 2.0, OpenAPI 3.x), OpenAPI Overlays, and OpenAPI Arazzo workflow descriptions. It simplifies workflows by enabling tasks such as validating API specs, upgrading versions, merging overlays, and applying transformations directly within CI/CD pipelines. This action is Docker-based and leverages the `roas` tool for seamless integration and enhanced API specification management.
