## Action Summary

`docker-hash` is a GitHub Action and CLI tool that generates a deterministic SHA-256 hash for Docker image builds by analyzing the Dockerfile, build arguments, and files specified in `COPY`/`ADD` instructions. It helps automate tasks such as cache-busting, detecting changes in image builds, and ensuring deterministic outputs in CI pipelines. This tool is useful for optimizing build processes and maintaining consistency across builds.
