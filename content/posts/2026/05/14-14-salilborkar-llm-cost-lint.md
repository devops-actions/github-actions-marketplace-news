## Action Summary

The `llm-cost-lint` GitHub Action automates cost analysis for AWS Bedrock and Azure OpenAI API calls in Python code by scanning pull requests and estimating their monthly impact on expenses. It helps teams identify potentially expensive LLM operations during code review, enabling adjustments to models, token limits, or call frequency before deployment. Key capabilities include generating detailed cost breakdowns, enforcing cost thresholds, and posting actionable insights as PR comments.
