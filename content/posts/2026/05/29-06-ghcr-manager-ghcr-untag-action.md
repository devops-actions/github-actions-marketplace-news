## Action Summary

The **GHCR Untag Action** is a GitHub Action designed to remove specific tags from a container image in the GitHub Container Registry (GHCR) without deleting the entire package. It automates a workaround for the lack of a direct "remove tag only" operation in GHCR by creating and deleting a temporary package version. This action simplifies tag management and ensures efficient cleanup of outdated or incorrect tags.
