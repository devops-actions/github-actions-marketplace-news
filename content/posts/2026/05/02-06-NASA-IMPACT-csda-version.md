## Action Summary

The `csda-version` GitHub Action automates the calculation of the next CSDA-compliant version number (formatted as `vYY.PI.SP-X`) for a repository based on the year, program increment, sprint, and release. It integrates with the `release-please` action to streamline semantic versioning and release management workflows. This action simplifies version tracking and ensures consistency while generating release versions and changelogs.
