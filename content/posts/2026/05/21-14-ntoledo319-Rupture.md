## Action Summary

**Summary:**  
The Rupture GitHub Action and CLI tools help organizations address AWS platform deprecations by automating the migration and remediation process for upcoming runtime end-of-life deadlines, such as Amazon Linux 2, Lambda Python versions, and Lambda Node.js 20. These tools scan for affected resources, update source code and infrastructure configurations, audit dependencies, generate deployment plans with rollback options, and enable staged rollouts, ensuring seamless transitions and reducing the risk of production outages.
