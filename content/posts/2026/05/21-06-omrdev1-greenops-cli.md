## Action Summary

The **GreenOps CLI GitHub Action** provides automated carbon footprint analysis and optimization recommendations for Terraform plans in AWS, Azure, and GCP CI/CD pipelines. It calculates Scope 2 operational, Scope 3 embodied emissions, and water consumption impacts, posting actionable insights directly on GitHub pull requests to help developers reduce their environmental footprint. Key features include detailed resource impact breakdowns, region-specific optimization suggestions, inline Terraform code recommendations, and configurable policy budgets to enforce sustainability goals.
