## Action Summary

The `action-stars-next` GitHub Action automates the creation of a curated README file that lists and organizes your starred repositories by month. It simplifies the task of categorizing and maintaining an up-to-date overview of your starred repositories, presenting them in a structured format with expandable sections for older years. Key features include automatic grouping by date, customizable output options, and optional attribution for transparency.

## What's Changed

Initial release. Generates a monthly grouped README from your GitHub stars, with current-year sections expanded, older years collapsed, optional attribution footer, and optional CC0 license footer. Use `erbanku/action-stars@v1`.
