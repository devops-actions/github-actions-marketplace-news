## Action Summary

This GitHub Action, "SEC Checks - SAST & Supply Chain Scanner," integrates Semgrep for static application security testing (SAST) and audits `package.json` lifecycle scripts to identify potential supply chain vulnerabilities, such as malicious `postinstall` or `preinstall` scripts. It automates security scanning across multiple languages, provides detailed findings in the Actions UI, and supports customizable configurations for severity thresholds, rulesets, and trusted actions. This action streamlines vulnerability detection and enhances supply chain security in CI/CD pipelines.
