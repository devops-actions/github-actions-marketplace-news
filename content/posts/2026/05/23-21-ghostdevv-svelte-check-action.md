## Action Summary

The **Svelte Check Action** is a GitHub Action designed to automate the process of running `svelte-check` in pull requests. It identifies errors and warnings in Svelte projects, providing clear feedback by adding comments and file annotations directly to pull requests. This action streamlines code reviews by highlighting issues in changed files, helping developers maintain code quality and catch problems early.
