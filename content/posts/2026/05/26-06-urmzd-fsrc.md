## Action Summary

The `fsrc` GitHub Action automates the process of embedding external source file contents into text files using customizable comment markers. It simplifies the task of maintaining up-to-date code snippets or configuration examples directly within documentation or code files. Key features include support for multiple file types, raw or code-fenced insertion, idempotent updates, and seamless integration into GitHub workflows.
