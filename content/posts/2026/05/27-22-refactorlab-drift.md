## Action Summary

Drift is an open-source performance and FinOps platform designed to detect and prevent performance regressions during development by combining static call-graph analysis, runtime profiling, and per-pull request (PR) performance verdicts. It helps teams identify and fix inefficiencies like N+1 queries, expensive computations, and blocking I/O before they reach production, reducing cloud costs, improving performance, and minimizing incident response efforts. Key capabilities include profiling every PR, analyzing hot paths, and providing actionable feedback on potential regressions directly in code reviews.
