## Action Summary

The `tftidy` GitHub Action automates the installation of the `tftidy` CLI tool, which scans Terraform files to clean up transient blocks (`moved`, `removed`, `import`) left after refactors or state operations. It simplifies Terraform configuration management by removing unnecessary blocks, ensuring clean and maintainable `.tf` files. The action allows users to easily integrate `tftidy` into CI workflows for automated cleanup and formatting of Terraform code.
