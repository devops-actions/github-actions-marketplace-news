## Action Summary

The `rbx-convert-action` is a GitHub Action that automates the conversion of Roblox place and model files between XML (`.rbxlx`, `.rbxmx`) and binary (`.rbxl`, `.rbxm`) formats using the `rbx-util` tool. It streamlines file format transformations for developers working with Roblox projects, eliminating manual conversion steps and ensuring compatibility across different formats. This action provides efficient and reliable file handling for workflows involving Roblox assets.
