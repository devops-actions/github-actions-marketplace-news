## Action Summary

The `cortex-memory-budget` GitHub Action performs static analysis of ARM Cortex-M ELF binaries to provide detailed flash and RAM usage reports, broken down by memory regions, sections, symbols, and source files. Designed for resource-constrained microcontrollers, it automates memory footprint assessment, detects stack and heap allocations, and supports regression analysis with diff reports and CI gating based on configurable memory usage thresholds. The action is framework-agnostic, requires no runtime dependencies, and outputs results in multiple formats, including Markdown reports, PR comments, and machine-readable JSON.
