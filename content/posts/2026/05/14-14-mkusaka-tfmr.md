## Action Summary

The `tfmr` GitHub Action is a CLI tool designed to analyze Terraform modules and their dependencies, providing a comprehensive list of related files, including `.tf` and `.tf.json` files. It automates tasks such as identifying affected modules and filtering outputs based on changes in file sets, which is particularly useful for optimizing CI/CD workflows. Key capabilities include detecting local and remote module references, determining if modules are impacted by file changes, and generating outputs suitable for integration with tools like Terraform or static analysis utilities.
