## Action Summary

Release Pilot is a GitHub Action designed to automate the process of versioning, tagging, changelog generation, and release creation by leveraging the Conventional Commits standard. It simplifies release workflows by combining multiple functionalities—such as semantic version calculation, breaking change detection, GitHub release creation (including drafts and pre-releases), and floating tag management—into a single, zero-configuration action. This tool eliminates the need for additional dependencies or scripts, streamlining the release process for developers.
