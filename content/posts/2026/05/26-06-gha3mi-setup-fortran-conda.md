## Action Summary

This GitHub Action sets up a comprehensive Fortran development environment using Conda or vendor-specific packages, supporting multiple Fortran compilers across Linux, macOS, and Windows. It automates the installation of compilers (Fortran, C, C++) and essential development tools like `fpm`, `cmake`, `ninja`, `meson`, and `pkg-config`, while also allowing additional package installations. By streamlining the environment setup, it simplifies CI/CD workflows for projects involving Fortran development.
