## Action Summary

Vizb is a CLI tool and GitHub Action that converts raw Go benchmark output into interactive 4D visualizations, enabling in-depth performance analysis through a single deployable HTML file. It automates tasks such as merging multiple benchmark datasets, applying advanced grouping logic, and generating visual comparisons of time, memory, and other metrics. Key features include support for various chart types, logarithmic scaling, flexible input formats, smart grouping, and seamless CI pipeline integration for benchmarking and visualization workflows.
