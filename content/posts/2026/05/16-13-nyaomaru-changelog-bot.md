## Action Summary

The `@nyaomaru/changelog-bot` GitHub Action automates the creation of polished changelog entries by transforming Git history, PR titles, and release notes into human-readable sections. It eliminates the manual effort of writing changelogs by integrating AI-powered summaries (optional) or using heuristic methods for fallback, and can optionally create a pull request with the updated changelog and release notes. Designed for CI/CD workflows, it ensures safe defaults, version control, and seamless integration without requiring custom scripting.
