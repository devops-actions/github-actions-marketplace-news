## Action Summary

The **SEC Checks - SAST & Supply Chain Scanner** GitHub Action automates static application security testing (SAST) using Semgrep to analyze code for vulnerabilities and audits `package.json` lifecycle scripts to detect potential supply chain attacks. It helps developers identify security issues across multiple languages and safeguard against malicious scripts in dependency workflows. Key features include customizable rule sets, severity-based failure thresholds, detailed findings summaries in the Actions UI, and artifact uploads for results storage.
