## Action Summary

The `setup-unirtm` GitHub Action automates the installation and configuration of UniRTM, a runtime and tools manager, in CI/CD workflows. It provides features like smart auto-detection of the best installation method, support for multiple install methods (e.g., npm, pip, GitHub Release), caching for faster runs, and GitHub proxy support for restricted networks. This action simplifies and accelerates the setup process for UniRTM across Linux, macOS, and Windows environments.
