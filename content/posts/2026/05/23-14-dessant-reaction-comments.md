## Action Summary

**Reaction Comments** is a GitHub Action designed to automatically manage reaction-only comments (e.g., "+1" or emoji responses) on issues and pull requests by replacing them with a customizable message and deleting them after a set period. This action promotes the use of GitHub reaction features instead of cluttering discussions with redundant comments. It helps maintain cleaner and more focused communication in repositories by automating the detection, replacement, and removal of such comments.
