## Action Summary

The **`node-semantic-release`** GitHub Action automates the process of managing and publishing releases for Node.js projects. It handles tasks such as installing dependencies, building the project, running `semantic-release` to generate versioned releases, and optionally creating GitHub Releases with base64-encoded release notes for downstream workflows. This action simplifies release management by automating versioning, changelog generation, and release asset handling, while supporting flexible configurations for release and publish workflows.
