## Action Summary

The `ghaups` GitHub Action is a CLI tool that automates updating, pinning, and scanning GitHub Actions in workflow files. It ensures workflows use the latest versions of actions by updating them with the latest release and corresponding commit SHA, while also scanning for high/critical vulnerabilities using Trivy. This tool streamlines dependency management, enhances security, and provides detailed reports on updates and potential issues.
