## Action Summary

`go-rpcatlas` is a tool that analyzes Go source code to generate static summaries of RPC/API call flows, including call trees, handler details, and execution summaries, in Markdown or JSON formats. It automates the process of understanding and documenting RPC flows, aiding tasks like code review, onboarding, and AI-assisted code analysis. Its lightweight, no-runtime approach makes it suitable for CI integration, enabling comprehensive insights into code execution paths without requiring service bootstrapping or runtime tracing.
