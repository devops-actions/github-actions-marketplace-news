## Action Summary

`zero-release` is a GitHub Action designed for automating semantic versioning and release workflows using Conventional Commits, without requiring runtime dependencies like Node.js, Python, or external tools. It calculates the next semantic version, generates release notes, creates Git tags, and supports customizable plugins for publishing and notifications. This action simplifies CI/CD pipelines by enabling lightweight, dependency-free release automation directly within GitHub Actions.
