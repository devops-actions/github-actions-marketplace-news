## Action Summary

This GitHub Action allows users to run continuous integration (CI) workflows on the OpenIndiana operating system, which is not natively supported by GitHub Actions. It automates the setup of an OpenIndiana virtual machine (VM), synchronizes the source code and environment variables from the host machine to the VM, and enables running custom scripts or commands within the VM. Key features include support for environment variable injection, file synchronization, network port forwarding, and customizable VM resources such as memory, CPU, and OpenIndiana release versions.
