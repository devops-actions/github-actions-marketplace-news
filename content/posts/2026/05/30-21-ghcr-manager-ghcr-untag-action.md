## Action Summary

The **GHCR Untag Action** automates the removal of specific tags from a GitHub Container Registry (GHCR) package while preserving the rest of the package's contents. It addresses the limitation of GHCR, which does not natively support direct tag removal, by employing a workaround that temporarily duplicates and deletes the associated manifest. This action streamlines container management by enabling precise tag cleanup in CI/CD workflows.
