## Action Summary

This GitHub Action automates the validation of PowerOn scripts on the Jack Henry™ Symitar credit union core platform. It ensures that PowerOn files comply with server-side requirements by connecting to the Symitar system and performing checks on modified or all files in a repository. This action streamlines compliance processes, reduces manual validation effort, and supports advanced capabilities like file exclusions and preserving server-managed files.
