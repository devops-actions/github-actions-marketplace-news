## Action Summary

The **Veriva Code Scan GitHub Action** analyzes pull request diffs to detect AI-generated code patterns, security vulnerabilities, dependency issues (e.g., typosquatting), and code quality concerns. It automates the generation and optional upload of SARIF reports to GitHub Code Scanning, enabling detailed security and quality insights directly within the repository's security dashboard. Additionally, it provides actionable outputs, such as trust scores and severity counts, to help enforce code quality and security standards in CI/CD workflows.
