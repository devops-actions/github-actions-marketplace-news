## Action Summary

This GitHub Action, *Get Supported GHC Version*, automates the process of identifying the latest and oldest GHC (Glasgow Haskell Compiler) versions compatible with a Haskell project's `base` dependency constraints specified in `package.yaml`. It simplifies CI/CD workflows by determining the appropriate GHC version to install and optionally validates that the lower bound of the `base` dependency aligns with the tested GHC versions, helping to ensure compatibility and prevent build issues.
