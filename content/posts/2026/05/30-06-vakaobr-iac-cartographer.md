## Action Summary

`iac-cartographer` is a GitHub Action designed to automate the discovery, documentation, and publishing of detailed, up-to-date information about an organization's Terraform or Infrastructure-as-Code (IaC) repositories. It scans source control systems, extracts structural and metadata insights, generates summaries with AI, and publishes hierarchical documentation to platforms like Confluence, Notion, or GitHub Wiki. The action streamlines onboarding, ensures documentation consistency, and highlights issues across an IaC estate, offering customizable integrations for discovery, publishing, and notifications.
