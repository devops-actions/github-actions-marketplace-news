## Action Summary

The **scalp-action** GitHub Action automates dependency security and compliance checks for JavaScript projects by running the SCAL-P tool in CI workflows. It enforces policies, verifies dependency hashes, audits lockfiles, and generates structured reports (JSON/SARIF) to identify and prevent vulnerabilities or policy violations. This action streamlines secure dependency management, ensuring integrity and reducing risks in package installations.
