## Action Summary

**Summary:**  
Find the Gaps is a CLI tool that analyzes a codebase and its associated documentation to identify outdated or missing documentation, ensuring consistency between code and documentation. It automates the detection of issues such as mismatched function signatures or undocumented public APIs, which are often overlooked by traditional link or spell checkers. The tool supports multiple programming languages and generates a browsable report to streamline documentation maintenance.
