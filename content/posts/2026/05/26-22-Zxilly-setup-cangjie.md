## Action Summary

The `setup-cangjie` GitHub Action automates the installation and configuration of the Cangjie compilation environment within GitHub Actions workflows. It supports different version channels (LTS, STS, Nightly), enables cross-compilation for various target platforms, and facilitates the installation of additional components with automatic environment variable injection. This action simplifies the setup of Cangjie development environments, reducing manual configuration efforts.
