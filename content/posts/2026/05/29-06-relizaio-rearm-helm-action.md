## Action Summary

The `rearm-helm-action` GitHub Action automates the versioning, packaging, and publishing of Helm charts to OCI-compliant registries, AWS ECR, or ChartMuseum, while integrating with ReARM to submit release metadata. It streamlines the process of managing Helm chart versions by updating the `Chart.yaml` with a ReARM-generated version, committing and pushing changes back to the repository, and enabling additional features like SBOM generation and signing for enhanced security. This action simplifies Helm chart lifecycle management and ensures consistent version tracking and publication.
