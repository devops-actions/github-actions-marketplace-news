## Action Summary

This GitHub Action automates the process of updating IBM DataStage export assets by applying one or more overlay directories in a specified order. It enables users to modify, replace, or add assets and values to an existing DataStage export using overlays, simplifying asset customization and management. Key capabilities include support for sequential overlays, output to zip or directory formats, and optional property-based substitutions.
