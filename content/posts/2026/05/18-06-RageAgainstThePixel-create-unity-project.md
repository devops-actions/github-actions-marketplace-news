## Action Summary

The `create-unity-project` GitHub Action automates the creation of a new Unity project using a specified template package. It simplifies project setup by generating a Unity project in a defined directory with a customizable name and template, reducing manual steps in initializing projects. Key capabilities include support for template matching via regex and outputting the absolute path of the created project.
