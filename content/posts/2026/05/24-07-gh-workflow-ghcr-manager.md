## Action Summary

The `ghcr-manager` GitHub Action is designed to inspect, manage, and clean up packages in the GitHub Container Registry (GHCR). It automates tasks such as scanning packages, removing unused or outdated tags, and previewing cleanup actions with a `dry-run` mode, providing clear summaries and optional database artifacts for auditing. This action simplifies registry maintenance, ensuring efficient package management and storage hygiene.
