## Action Summary

The **GitHub Actions Example Checker** is a GitHub Action designed to validate the usage examples in documentation against the action's `action.yml` schema. It automates the process of checking that inputs, outputs, and YAML examples in markdown files are correctly defined and conform to the schema, ensuring consistency and accuracy in documentation. Key features include automatic discovery of `action.yml` files, input/output validation, type checking, pattern matching, and precise error reporting, streamlining the maintenance of reliable and error-free documentation.
