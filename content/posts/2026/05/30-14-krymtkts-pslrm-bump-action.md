## Action Summary

The `pslrm-bump-action` GitHub Action automates the process of updating `psreq.lock.psd1` lockfiles for projects using the `pslrm` dependency manager. It streamlines dependency management by running `pslrm`, updating the lockfile, and automatically handling branch creation, commits, pushes, and pull requests when changes occur. This action simplifies scheduled or manual dependency update workflows, ensuring projects stay up-to-date with minimal manual intervention.
