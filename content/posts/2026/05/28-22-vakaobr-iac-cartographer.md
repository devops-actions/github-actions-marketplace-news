## Action Summary

`iac-cartographer` is a GitHub Action designed to automate the discovery, documentation, and publishing of fleet-level insights for Terraform and Infrastructure-as-Code (IaC) repositories. It identifies repositories across various source control platforms, extracts key structural details, generates purpose summaries with an LLM, and publishes organized documentation to platforms like Confluence, Notion, or Markdown. This action streamlines onboarding, ensures up-to-date documentation, and provides visibility into IaC estates, reducing manual effort and improving operational clarity.
