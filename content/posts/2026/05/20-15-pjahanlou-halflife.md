## Action Summary

Halflife is a GitHub Action that evaluates the health and maintenance status of npm project dependencies listed in a `package.json` file. It automates the process of assessing dependencies by analyzing signals such as commit activity, issue-to-star ratio, and archival status, assigning a health score (0–100) to each package. This action helps identify outdated or poorly maintained dependencies, failing the build if critical issues are found, and generates a comprehensive health report in the GitHub Actions summary.
