## Action Summary

The **SEC Checks - SAST & Supply Chain Scanner** GitHub Action automates security analysis by performing Semgrep static application security testing (SAST) across multiple programming languages and auditing `package.json` lifecycle scripts to detect potential supply chain attacks. It solves problems related to code vulnerabilities and malicious script detection, providing actionable findings in a structured summary and enabling configurable thresholds, custom rules, and artifact uploads for security audits and CI/CD workflows.
