## Action Summary

The `upload_build_to_testflight` GitHub Action automates the process of uploading a pre-built iOS `.ipa` file to App Store Connect / TestFlight. It handles tasks such as selecting the appropriate Xcode version, performing the upload using App Store Connect API credentials, monitoring the build processing status, and optionally setting release notes or encryption compliance flags. This action streamlines the post-build distribution workflow, providing developers with a seamless solution for deploying iOS apps to TestFlight.
