## Action Summary

This GitHub Action, **lint-gha-reviewdog**, automates the linting of GitHub Actions workflows and custom actions using `actionlint`, `zizmor`, and `ghalint`. It provides pull request feedback through `reviewdog`, generates CI failure alerts on push, and ensures consistent tool versions, simplifying the detection and reporting of issues in workflow files. The action supports centralized configuration, optional per-tool execution, and outputs combined linting results for enhanced workflow quality and maintainability.
