## Action Summary

The **sonarqube-multi-project-comment** GitHub Action consolidates SonarQube quality gate results from multiple projects into a single, aggregated pull request (PR) comment. It simplifies workflows for monorepos by replacing multiple project-specific comments with a unified summary, providing clear visibility into key metrics like quality gate status, coverage, and duplications across all projects. This action automates the process of aggregating data, minimizes comment clutter, and ensures each re-run updates the PR with fresh, streamlined results.
