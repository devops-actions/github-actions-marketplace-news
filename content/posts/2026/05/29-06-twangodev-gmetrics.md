## Action Summary

The `gmetrics` GitHub Action is a Go-based tool for generating SVG metrics, specifically focused on language statistics for repositories. It automates the collection and caching of detailed per-repo language data, ensuring efficient processing by analyzing only new commits since the last run. This action simplifies tracking and visualizing repository language usage over time.
