## Action Summary

The `setup-ardent` GitHub Action automates the installation of the `ardent` NSIS formatter CLI, allowing users to easily integrate it into their workflows. It simplifies the setup process by managing the installation of `ardent` and any necessary Rust toolchain dependencies, enabling developers to format NSIS scripts efficiently. This action is particularly useful for streamlining code formatting tasks in CI/CD pipelines.
