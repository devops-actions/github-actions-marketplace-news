## Release notes

## Summary
This release introduces a major version of Bump Version by Labels because the internally used actions/checkout action was upgraded from v4 to v6.  
As a result，the minimum required GitHub Actions Runner version is now v2.327.1 when you use this action.

If you are using GitHub hosted runners，you are usually already on a recent runner version and no change is required.  
If you are using self hosted runners，please check the runner version before upgrading to v2.x of this action.
