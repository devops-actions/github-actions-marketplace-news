```sh

```

```sh
powershell -ExecutionPolicy Bypass -c "irm https://github.com/hougesen/kdlfmt/releases/download/v0.1.2/kdlfmt-installer.ps1 | iex"
```

### Install prebuilt binaries via Homebrew
