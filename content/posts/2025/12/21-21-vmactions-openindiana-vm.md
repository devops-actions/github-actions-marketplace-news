## Action Summary

This GitHub Action enables running CI workflows on OpenIndiana, addressing the limitation of native GitHub Actions support for only Ubuntu, Windows, and macOS. It automates the setup and synchronization of the CI environment between the host and an OpenIndiana virtual machine, ensuring seamless code sharing, environment replication, and execution of scripts. Key features include customizable resource allocation, file synchronization methods, environment variable handling, and network configuration for flexibility in CI execution.
