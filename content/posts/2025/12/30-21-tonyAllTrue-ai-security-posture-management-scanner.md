## Action Summary

This GitHub Action provides automated security testing for large language model (LLM) endpoints and AI models by integrating with the AllTrue platform. It helps identify vulnerabilities such as prompt injection, data leakage, and malicious code, while offering capabilities like flexible scoping, parallel execution, and detailed reporting. The action streamlines security assessment processes, enabling organizations to proactively safeguard their AI infrastructure.
