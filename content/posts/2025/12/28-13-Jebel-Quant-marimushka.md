## Action Summary

Marimushka is a tool designed to export [marimo](https://marimo.io) notebooks and applications into HTML or WebAssembly formats, enabling the creation of interactive and shareable web-based versions without requiring Python installation for viewers. It automates tasks such as customizing output with Jinja2 templates, generating index pages, and integrating with GitHub Actions for seamless deployment. Key capabilities include support for interactive and standalone exports, recursive directory scanning, and flexible configuration through a command-line interface or Python API.
