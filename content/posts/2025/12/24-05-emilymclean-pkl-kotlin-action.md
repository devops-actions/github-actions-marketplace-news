## Action Summary

The **pkl-kotlin-action** GitHub Action automates the generation of Kotlin source code from Pkl model files. It streamlines the process of converting Pkl files into Kotlin bindings, solving the need for manual code generation and ensuring consistent and efficient output. This action is particularly useful for developers working with Pkl models in Kotlin projects.
