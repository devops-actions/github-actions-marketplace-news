## Action Summary

The Content Reviewer Action is a GitHub Action designed to automate the review of content in pull requests using LLM-powered analysis. It identifies issues in changed files, provides inline feedback as PR comments, and categorizes results by severity (error, warning, suggestion). This action streamlines content quality assurance, supports multilingual reviews, and outputs structured results for improved collaboration.
