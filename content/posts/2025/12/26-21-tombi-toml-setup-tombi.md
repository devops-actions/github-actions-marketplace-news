## Action Summary

The `setup-tombi` GitHub Action automates the installation and setup of the Tombi tool within GitHub Actions workflows. It simplifies the process of configuring Tombi for tasks such as validating and linting TOML files, ensuring a consistent and efficient workflow. Key features include support for specifying specific Tombi versions and optional checksum verification for enhanced security.
