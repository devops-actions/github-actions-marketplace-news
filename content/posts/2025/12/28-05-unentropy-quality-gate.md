## Action Summary

The **Unentropy Quality Gate** GitHub Action evaluates pull request metrics against predefined baseline thresholds to identify potential regressions. It automates quality assurance by analyzing metrics, posting results as pull request comments, and optionally failing workflows to enforce quality standards. Key capabilities include configurable storage options, support for different enforcement modes (off, soft, or hard), and detailed reporting for metric-based decision-making.
