## Action Summary

The **Gentesty Evaluation Action** is a GitHub Action designed to automate the process of triggering evaluation runs on the Gentesty platform. It streams live logs from the evaluation process directly into your GitHub Actions workflow and provides the final test result (success or failure) upon completion. This action simplifies the integration of Gentesty evaluations into CI/CD pipelines, offering real-time feedback and secure, token-based authentication.
