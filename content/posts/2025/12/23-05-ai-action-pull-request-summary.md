## Action Summary

The **pull-request-summary** GitHub Action utilizes AI (large language models) to automatically generate concise summaries of pull requests by analyzing code changes. It simplifies the review process by providing a quick and clear overview of what each pull request entails, saving time for developers and reviewers. Key capabilities include customizable prompts, support for different AI models, and seamless integration into GitHub workflows.
