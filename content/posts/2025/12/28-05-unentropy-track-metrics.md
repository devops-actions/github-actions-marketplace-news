## Action Summary

The **Unentropy Track Metrics** GitHub Action automates the collection, storage, and visualization of custom code metrics within CI/CD pipelines. It helps developers monitor and analyze trends in their codebase over time by generating reports and storing metrics in configurable backends like SQLite or S3. This action simplifies tracking key insights about code health and build performance, aiding in data-driven decision-making for software projects.
