## Action Summary

SkillLens is a GitHub Action that transforms pull request (PR) review feedback into personalized learning opportunities by analyzing review comments using AI to extract relevant topics and suggesting curated Hyperskill resources. It automates the process of identifying skill gaps and providing actionable educational recommendations, fostering continuous learning and developer growth. This action is privacy-focused, requires no configuration, and integrates seamlessly into existing workflows.
