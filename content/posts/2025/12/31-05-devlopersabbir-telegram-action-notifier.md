## Action Summary

The **Telegram Failure Notifier** GitHub Action automates the process of sending Telegram messages to notify your team instantly when a GitHub Actions job fails. It provides detailed failure notifications by including contextual information such as the repository, branch, commit, and actor, and allows for customizable message formatting. This action helps streamline incident response and ensures timely communication of CI/CD pipeline failures.
