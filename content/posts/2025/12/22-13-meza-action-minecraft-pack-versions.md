## Action Summary

The Minecraft Pack Versions Action is a GitHub Action designed to automate the creation and maintenance of a JSON file that maps Minecraft versions to their corresponding datapack and resourcepack format numbers. It fetches official Minecraft version data, analyzes client JAR files to extract necessary format information, and can optionally automate commits and pull requests when updates are available. This action simplifies version tracking and ensures up-to-date mappings for developers working with Minecraft packs.
