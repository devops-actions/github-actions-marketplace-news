## Action Summary

The Contrast Verify GitHub Action automates the process of verifying an application onboarded to Contrast Security by assessing compliance with a predefined Job Outcome Policy or checking for open vulnerabilities against specified thresholds. This action helps streamline security checks in CI/CD pipelines, ensuring that applications meet organizational security standards before deployment. Key capabilities include filtering vulnerabilities by severity, build version, or discovery time, and integrating seamlessly with GitHub Actions or other environments via a Docker image.
