## Action Summary

The `setup-probitas` GitHub Action simplifies the integration of the Probitas scenario-based testing framework into CI/CD workflows by automating the installation and setup of the Probitas CLI. It ensures that the specified version of Probitas is installed, verified, and available in the execution environment, allowing users to seamlessly run tests across different environments and configurations. This action streamlines testing processes and supports tasks like version management, matrix testing, and cross-platform compatibility.
