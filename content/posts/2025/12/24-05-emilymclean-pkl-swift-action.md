## Action Summary

The **pkl-swift-action** GitHub Action automates the generation of Swift source code from Pkl model files. It simplifies the process of converting Pkl data into reusable Swift bindings, saving time and reducing manual coding efforts. Key capabilities include processing multiple Pkl files and specifying output locations for the generated code.
