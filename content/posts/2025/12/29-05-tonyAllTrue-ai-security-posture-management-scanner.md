## Action Summary

This GitHub Action enables automated security testing for LLM endpoints and AI models by integrating with the AllTrue platform. It identifies and tests vulnerabilities such as prompt injection, data leakage, harmful content generation, and malicious code, while providing options for detailed reporting and GitHub issue creation. The action streamlines security testing across various scopes, supports parallel execution, and automates tasks like model onboarding from platforms like HuggingFace.
